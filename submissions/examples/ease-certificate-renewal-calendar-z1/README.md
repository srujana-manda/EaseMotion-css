# Ease Certificate Renewal Calendar

## Overview
A CSS-only component for displaying SSL/TLS certificate renewal schedules.

## Features
- Renewal status badges
- Responsive card layout
- CSS variables
- Hover animations
- Pure HTML & CSS

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` directly in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari

## Responsive
Optimized for desktop, tablet, and mobile devices.

## Why EaseMotion CSS?
Provides a reusable animation-first component for certificate monitoring, hosting dashboards, cloud platforms, and DevOps administration panels.
