# Ripple-Wave Badge Component

A lightweight, pure CSS badge module featuring continuous multi-stage ripple wave propagation rings focused around active signal indicators.

## Features
- **Pure CSS/HTML**: Animated using stacked delay keyframes with zero external JavaScript dependencies.
- **Accessible**: Built-in support for `prefers-reduced-motion` media queries.
- **Customizable**: Exposes CSS custom properties to tweak ripple propagation speed, maximum scaling factors, and wave color hues.

## Customization Parameters
The following CSS custom properties are available in `style.css`:
- `--ripple-speed`: Controls the wave propagation cycle speed.
- `--ripple-scale-max`: Adjusts the maximum outward scale factor for expanding rings.
- `--ripple-color`: Sets the background color and transparency for the expanding waves.
