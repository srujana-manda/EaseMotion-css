# Animated Gradient Background

A smooth color-shifting animated gradient background utility — pure CSS.

## Features
- Multi-color gradient animation
- Smooth color transitions via background-position
- Reusable utility class

## Usage
```css
.gradient-box { background: linear-gradient(-45deg, ...); background-size: 400% 400%; animation: gradient-shift 6s ease infinite; }
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
