# Ease Kafka Topic Throughput Monitor

## Overview

A responsive HTML and CSS component for monitoring Kafka topics, message throughput, consumer lag, and broker health.

## Features

- Topic throughput metrics
- Consumer lag monitoring
- Partition health status
- Broker synchronization alerts
- Responsive layout
- Pure HTML & CSS
- CSS animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable monitoring component for event streaming platforms, Kafka clusters, distributed messaging systems, and observability dashboards.
