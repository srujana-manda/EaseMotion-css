# Vinyl Records Shop Demo

A responsive **Vinyl Records Shop** landing page built using EaseMotion CSS.

## Features

- Hero section with retro vinyl theme
- Genre categories
- New arrivals
- Record of the Week
- Artist spotlights
- Vinyl grading guide
- Customer reviews
- Responsive layout
- Hover effects
- EaseMotion CSS animations

## Files

```
demo.html
style.css
README.md
```

- ease-slide-left
- ease-slide-right
- ease-zoom-in
