# Fade In Scroll

Elements fade in as they scroll into viewport using Intersection Observer.

## Features

- Fade in on scroll reveal
- Configurable stagger delay via data-delay
- Smooth CSS transition
- Unobserve after reveal for performance

## Usage

Add `.fis-card` with optional `data-delay` attribute. JS handles IntersectionObserver; adding `.fis-visible` triggers the transition.
