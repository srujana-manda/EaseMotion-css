# Ease Certificate Authority Trust Dashboard

## Overview

A responsive HTML and CSS component for monitoring certificate authority trust, certificate issuance, revocation status, and renewal planning.

## Features

- Root CA trust overview
- Issued certificate monitoring
- Certificate revocation tracking
- Renewal readiness alerts
- Responsive layout
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable dashboard component for PKI infrastructure, certificate lifecycle management, enterprise security operations, and cloud identity services.
