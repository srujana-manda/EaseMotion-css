# Ease Container Registry Gallery

## Overview
A CSS-only component for displaying container images stored in a registry.

## Features
- Version and image size display
- Animated status badges
- Responsive layout
- CSS variables
- Pure HTML & CSS

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` directly in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari

## Responsive
Works across desktop, tablet, and mobile devices.

## Why EaseMotion CSS?
Provides a reusable animation-first component for container registry dashboards, DevOps platforms, CI/CD tools, and cloud-native infrastructure.
