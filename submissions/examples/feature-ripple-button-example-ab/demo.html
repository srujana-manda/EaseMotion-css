Ripple Button Example

A polished button with a smooth ripple attention effect — click to cast a material-style ripple, or enable the soft looping ring for passive attention. Built for issue #50778.

Live playground

Click any button to spawn a ripple from the pointer. Use the controls to change color, enable the attention-seeker ring, or adjust ripple speed.

Speed maps to --ripple-button-speed-ab.

Variant gallery

Each button supports click ripples. The last card shows the attention-seeker loop used to gently pull focus without requiring a click.

How to use

<button type="button" class="ripple-button-ab ripple-button-primary-ab">
  Hover me
</button>

<!-- Optional attention-seeker ring -->
<button
  type="button"
  class="ripple-button-ab ripple-button-primary-ab is-attention-ab"
>
  Notice me
</button>
  1. Add ripple-button-ab plus a color variant class.
  2. Clicks automatically spawn a ripple from the pointer position.
  3. Add is-attention-ab for a soft looping attention ring.
  4. Keep real <button> elements for keyboard access.

Accessibility notes

  • Uses native <button> elements with visible focus rings.
  • Ripple layers are decorative (aria-hidden) and do not trap focus.
  • Under prefers-reduced-motion: reduce, click ripples and attention rings are disabled; buttons keep a simple press feedback.
  • Attention rings pause when the user focuses the button via keyboard.
