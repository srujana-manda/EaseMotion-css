# Zoom-In Tabs for E-Commerce Checkout Layout

A modern **CSS-only** animated checkout tabs component built for the **EaseMotion CSS** examples collection.

This component demonstrates a smooth **Zoom-In Tabs** interaction using only **HTML5** and **CSS3**, without any JavaScript or external libraries.

---

## ✨ Features

- Pure HTML & CSS
- Smooth Zoom-In active tab animation
- CSS-only tab switching using radio inputs
- Responsive layout
- Keyboard accessible
- E-Commerce checkout workflow
- Modern UI design
- CSS Custom Properties for easy customization
- Soft shadows and micro-interactions
- No external libraries
- No JavaScript required

## 🎨 Customization

The component uses CSS variables for easy customization.

```css
:root{
    --primary-color;
    --secondary-color;
    --background-color;
    --surface-color;
    --text-color;
    --border-radius;
    --transition-duration;
    --transition-easing;
    --zoom-scale;
    --shadow;
    --spacing;
}
```

You can easily customize:

- Colors
- Animation speed
- Zoom scale
- Border radius
- Shadows
- Typography
- Spacing
