# Loading Animations Module

A collection of lightweight, CSS-only loading indicators for modern web applications.

## Included Animations

| Class | Type | Description |
| :--- | :--- | :--- |
| `.em-loader-spinner` | Spinner | A smooth, rotating circular loader. |
| `.em-loader-dots` | Dots | Three pulsing dots, ideal for chat or text transitions. |
| `.em-loader-progress-bar` | Progress | An indeterminate bar for loading states. |
