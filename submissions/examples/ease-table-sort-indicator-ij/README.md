# Table Sort Indicator

Animated sort arrow indicator for table column headers. Arrow rotates 180 degrees on click to toggle ascending/descending state.

## Features

- Arrow rotates smoothly on sort direction change
- Active column highlighted with background color transition
- Fully customizable duration, easing, and colors via CSS custom properties
- Lightweight — no external dependencies

## Usage

Toggle `.tsi-asc` or `.tsi-desc` class on `.tsi-header`. JavaScript handles click toggling. CSS handles arrow rotation and active background transition.
