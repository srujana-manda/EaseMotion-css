# Animated Hero Banner (`ease-hero-banner-cv`)

A responsive animated hero banner built using **HTML5** and **Pure CSS**.

## Features

- Gradient animated heading
- Floating background shapes
- Call-to-action buttons
- Fade/slide entrance animation
- Responsive layout
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-hero-banner-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

Works in all modern browsers.
