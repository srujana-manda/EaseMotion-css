# Breadcrumb Arrow Separator

Breadcrumb navigation with animated arrow separators. Each arrow fades in with the crumb text. Hover effect on each crumb.

## Features

- Breadcrumb with arrow separators
- Arrow fade-in animation
- Hover effect on crumbs
- Staggered entrance animation

## Usage

Structure `<ol>` with `.bas-list` containing `.bas-item` list items. CSS handles separators via `::before`.
