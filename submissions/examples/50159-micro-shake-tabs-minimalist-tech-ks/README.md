**Tech Stack:** Pure HTML5 & CSS3 (No JavaScript)

---

## Overview

CSS Micro Shake Tabs is a lightweight, fully responsive tab component built using only HTML and CSS. It features a subtle micro shake interaction for the active tab, a clean minimalist interface, semantic HTML, and CSS custom properties for easy customization.

Designed for modern dashboards, developer tools, SaaS products, admin panels, and technology-focused interfaces, this component delivers smooth interactions while maintaining excellent accessibility and performance.

## Customization

All important design values are controlled through CSS Custom Properties.

| Variable | Description |
|-----------|-------------|
| `--accent` | Primary accent color |
| `--accent-light` | Light accent background |
| `--background` | Page background |
| `--surface` | Card background |
| `--text` | Primary text color |
| `--text-light` | Secondary text color |
| `--border` | Border color |
| `--radius` | Card border radius |
| `--tab-radius` | Tab border radius |
| `--animation-duration` | Micro Shake animation duration |
| `--animation-easing` | Animation timing function |
| `--micro-shake-distance` | Shake movement distance |
| `--shadow-sm` | Small shadow |
| `--shadow-md` | Medium shadow |
| `--shadow-lg` | Large shadow |

Example:

```css
:root{

    --accent:#0ea5e9;

    --background:#ffffff;

    --radius:20px;

    --animation-duration:.45s;

}
```
