# Ease Network Firewall Rule Board

## Overview
A responsive HTML and CSS component for displaying firewall rules, ports, protocols, and access status in security dashboards.

## Features
- Firewall rule cards
- Protocol and port details
- Animated status badges
- Responsive layout
- CSS variables
- Pure HTML & CSS

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` directly in any modern browser.

## Browser Support
- Chrome
- Firefox
- Edge
- Safari

## Responsive
Optimized for desktop, tablet, and mobile devices.

## Why EaseMotion CSS?
Provides a reusable animation-first component for enterprise firewall management, cloud security dashboards, and DevSecOps monitoring interfaces.
