# CSS-Only Morphing Search Bar

A clean and modern search bar that smoothly expands from a circular search icon into a full-width search input using only HTML and CSS.

## Features

- 🎨 Pure HTML & CSS
- 🔍 Smooth expanding animation
- 📱 Responsive layout
- ⚡ No JavaScript required
- 🎯 Uses `:hover` and `:focus-within`
- 💡 Easy to customize

## Folder Structure

```
morphing-searchbar/
├── demo.html
├── style.css
└── README.md
```

## How to Use

1. Open `demo.html` in your browser.
2. Hover over the search bar or click inside it.
3. The search field expands smoothly.

## Customization

You can easily customize:

- Border color
- Background color
- Width
- Animation duration
- Font
- Border radius
- Shadow intensity

## Browser Support

Works in all modern browsers supporting:

- CSS Flexbox
- CSS Transitions
- `:focus-within`

## Preview

- Compact circular search button by default.
- Expands into a full search input on hover or focus.
- Smooth transition without JavaScript.
