# Staggered Glass Cards

A self-contained example demonstrating the power of **EaseMotion CSS** combined with modern glassmorphism design.

## ✨ Features Highlighted
- **Staggered Entrances**: Uses `ease-slide-up` with `ease-delay-100`, `ease-delay-200`, and `ease-delay-300` for a cascading load effect.
- **Interactive Hover States**: Combines `ease-hover-lift` and `ease-hover-glow` for engaging micro-interactions.
- **Zero JavaScript**: All animations and transitions are handled purely by CSS.
- **Responsive Layout**: Utilizes `ease-flex`, `ease-flex-wrap`, and `ease-justify-center` for seamless adaptation across screen sizes.

## 🚀 How to Run
1. Clone the EaseMotion CSS repository.
2. Navigate to this folder: `cd submissions/examples/staggered-glass-cards`
3. Open `demo.html` in any modern web browser.
4. No build step or local server is required!

## 📂 Files
- `demo.html`: The main markup utilizing EaseMotion utility classes.
- `style.css`: Minimal custom CSS for the glassmorphism background and card styling.
