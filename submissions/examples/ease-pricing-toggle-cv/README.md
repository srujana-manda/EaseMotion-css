# Animated Pricing Toggle (`ease-pricing-toggle-cv`)

A responsive pricing toggle component built using **HTML5** and **Pure CSS**.

## Features

- CSS-only pricing toggle
- Animated switch
- Monthly & yearly pricing
- Highlighted featured plan
- Hover animations
- Responsive layout
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-pricing-toggle-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

Works in all modern browsers.
