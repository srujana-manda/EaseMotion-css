# ease-loading-spinner-overlay

A full-screen loading overlay with a spinning ring indicator.

## Features

- Fixed full-screen overlay with semi-transparent background
- Centered spinning ring with accent color
- Loading text label
- Controlled via `--loading` CSS custom property
- Pointer events disabled when hidden

## Usage

Set `--loading: 1` on the `.lso-overlay` element to show the loading overlay. Set to `0` to hide.
