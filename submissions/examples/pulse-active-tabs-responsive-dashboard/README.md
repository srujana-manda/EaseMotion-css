# Pulse Active Tabs - Responsive Dashboard Layouts


A pure CSS animated Pulse-Active Tabs component
created for responsive dashboard interfaces.


## Features

- CSS-only animation
- No JavaScript dependency
- Responsive dashboard layout
- Smooth pulse active state
- Keyboard focus support
- Custom CSS variables
- Lightweight reusable component


## Usage

```html
<div class="tabs">

<button class="tab active">
Overview
</button>

<button class="tab">
Analytics
</button>

</div>
