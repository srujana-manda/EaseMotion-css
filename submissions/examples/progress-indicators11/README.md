# Animated Progress Indicators

A lightweight, CSS‑only module for animated progress bars with various styles.

## Features
- Smooth, GPU-accelerated animations
- Multiple styles: solid, striped, gradient
- Responsive and easy to customize

## Demo
Open `demo.html` in your browser to see animated progress bars.

## Usage
```html
<link rel="stylesheet" href="style.css" />

<!-- Simple -->
<div class="progress-container">
  <div class="progress-bar" data-progress="70%">70%</div>
</div>

<!-- Striped -->
<div class="progress-container striped">
  <div class="progress-bar striped" data-progress="85%">85%</div>
</div>

<!-- Gradient -->
<div class="progress-container multicolor">
  <div class="progress-bar multi" data-progress="60%">60%</div>
</div>
