# Ease Virtual Machine Resource Pool

## Overview
A responsive HTML and CSS component for monitoring virtual machine resource pools and infrastructure utilization.

## Features
- VM pool overview
- CPU and memory utilization
- Instance count display
- Animated status badges
- Responsive layout
- Pure HTML & CSS

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` directly in any modern browser.

## Browser Support
- Chrome
- Firefox
- Edge
- Safari

## Responsive
Optimized for desktop, tablet, and mobile devices.

## Why EaseMotion CSS?
Provides an animation-first reusable component for virtualization dashboards, cloud infrastructure portals, and private cloud management systems.
