# Magnetic Button Hover

Button with magnetic hover pull effect. Button shifts toward cursor using --mbh-x and --mbh-y.

## Features

- Magnetic pull toward cursor
- Button shifts with transform via --mbh-x/--mbh-y
- Smooth bounce-back on leave
- JS tracks cursor position

## Usage

Set `--mbh-x` and `--mbh-y` (in pixels) on `.mbh-btn`. CSS handles transform transition.
