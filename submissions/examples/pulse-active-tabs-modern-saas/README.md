# Pulse-Active Tabs – Modern SaaS Layout

**Issue:** #50166

## Overview

A pure CSS Pulse-Active Tabs component designed for Modern SaaS layouts with smooth pulse animation, responsive behavior, and keyboard accessibility.

## Features

- Pure CSS
- Pulse-active animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript
- Lightweight and reusable

## Files

```
demo.html
style.css
README.md
```

## Usage

```html
<link rel="stylesheet" href="style.css">

<div class="tabs">
    <button class="tab active">Dashboard</button>
    <button class="tab">Features</button>
    <button class="tab">Pricing</button>
    <button class="tab">Support</button>
</div>
```

## Customization

```css
:root{
  --primary:#4f46e5;
  --duration:.35s;
  --radius:12px;
}
```

## Why EaseMotion CSS?

- Pure CSS implementation
- Human-readable class names
- Animation-first design
- Easy customization
- Responsive and reusable
