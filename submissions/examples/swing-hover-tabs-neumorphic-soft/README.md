# Swing Hover Tabs - Neumorphic Soft

## Overview

A pure CSS Swing Hover Tabs component inspired by Neumorphic Soft UI.

## Features

- Pure HTML & CSS
- No JavaScript
- Responsive Design
- Keyboard Accessible
- Swing Hover Animation
- Uses CSS Custom Properties
- Easy to Customize

## Files

- demo.html
- style.css

## Custom Properties

```css
--duration
--ease
--rotate
--scale
--primary
```

## Usage

```html
<link rel="stylesheet" href="style.css">
```

Open `demo.html` directly in any browser.

Designed for EaseMotion CSS.
