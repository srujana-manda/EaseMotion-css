# FAB Speed Dial

Floating action button with expandable speed dial. Menu items fade and slide in with staggered delays via --fsd-i.

## Features

- FAB with expandable speed dial
- Menu items with staggered entrance
- Smooth open/close via --fsd-open
- Bounce cubic-bezier animation

## Usage

Set `--fsd-open` to `1` on `.fsd-menu` to show items. Set `--fsd-i` (0-based) per item for stagger delay.
