# ease-price-filter-range

A dual-handle price range slider with animated fill track.

## Features
- Two thumb handles for min/max price selection
- Filled track between thumbs animates on slide
- Thumbs have hover scale and active glow on box-shadow
- Values swap automatically when handles cross
- CSS custom properties for track color, fill color, and thumb size

## CSS Custom Property Control
Set `--pfr-min` and `--pfr-width` on `.em-pfr-slider` to control the filled track position. The `:root` variables `--pfr-track-color`, `--pfr-fill-color`, `--pfr-thumb-size`, `--pfr-thumb-border`, and `--pfr-thumb-glow` allow full theme customization.
