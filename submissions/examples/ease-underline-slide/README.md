# 📏 Underline Slide – Underline Slides Between Tabs

> A smooth, polished tab navigation where a gradient underline slides between tabs on click or hover, creating a modern, engaging interface.

---

## 📖 Description

The **Underline Slide** is a sleek tab navigation component where a gradient underline smoothly slides between tabs on click or hover. The underline moves with a spring-like animation, creating a polished, modern feel that enhances the user experience. The component features four tabs with icons and labels, and each tab displays corresponding content panels with smooth fade-in transitions.

The underline uses a beautiful blue-to-cyan gradient and slides with a `cubic-bezier` easing that creates a satisfying bounce effect. Tabs are fully accessible with keyboard navigation (arrow keys, Home, End), and the component includes responsive design that adapts to smaller screens by hiding tab labels and showing only icons.

Perfect for navigation menus, content sections, dashboards, and any interface where you need to switch between views with a polished, modern touch. The Underline Slide transforms a simple tab bar into an engaging, interactive experience.
