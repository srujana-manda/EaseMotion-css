# CSS Zoom-Out Modal

A pure CSS Zoom-Out Modal designed for minimalist tech interfaces.

## Features

- Pure CSS (No JavaScript)
- Smooth Zoom-Out animation
- Responsive layout
- Keyboard accessible
- CSS Custom Properties
- Easy to customize

## Files

```
demo.html
style.css
README.md
```

## CSS Variables

```css
:root{
  --duration:.45s;
  --scale:.9;
  --easing:cubic-bezier(.2,.8,.2,1);
}
```
