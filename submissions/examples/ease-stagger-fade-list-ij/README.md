# Stagger Fade List

List with items that fade and slide in one by one with staggered delays via `--sfl-i`.

## Features

- List items fade and slide in
- Staggered delay between items via `--sfl-i`
- Each item animates in sequence
- Smooth cubic-bezier transition

## Usage

Add `.sfl-visible` class on `.sfl-list` to trigger animation. Set `--sfl-i` (0-based index) on each item for stagger delay.
