# 🧊 GlassMorph Todo List

A beautiful, modern task manager showcasing the **EaseMotion CSS** utility framework and Glassmorphism design principles. It features a soft, transparent UI overlaid on a dynamic, animated mesh-gradient background.

## ✨ Features

- **Glassmorphism UI:** Built with `backdrop-filter`, semi-transparent backgrounds, and soft border glows to mimic frosted glass.
- **EaseMotion Animations:** Integrates custom, smooth keyframe animations:
  - `ease-fade-up` for task entry.
  - `ease-fade-out` for task deletion.
  - `ease-scale` for interactive buttons and checkboxes.
  - `ease-shake` for the "Clear All" alert sequence.
- **Task Management:** Add, complete, and delete tasks dynamically.
- **Smart Filtering:** Switch between "All", "Active", and "Completed" views.
- **Persistent Storage:** Tasks are automatically saved to the browser's `localStorage`.
- **Dark Mode:** A seamless, instant toggle that adjusts both the glass panels and the background gradient.

## 📁 File Structure

- `demo.html` - The markup and Vanilla JS engine powering the Todo list.
- `style.css` - CSS variables, Glassmorphism classes, and the EaseMotion animation system.
- `README.md` - Project documentation.

## 🚀 How to Run

This project is completely dependency-free.

1. Clone or download the repository folder.
2. Ensure `demo.html` and `style.css` are in the same directory.
3. Open `demo.html` in any modern web browser (Chrome, Safari, Firefox).
4. Add a task and enjoy the smooth motion!
