# Tag Removable Slide

A removable tag with slide-out animation — pure CSS.

## Features
- Pill-shaped tag with close button
- Hover reveals close button with color transition
- Slide effect on hover
- Reusable tag component

## Usage
```html
<span class="removable-tag">Label
  <span class="tag-close">&times;</span>
</span>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
