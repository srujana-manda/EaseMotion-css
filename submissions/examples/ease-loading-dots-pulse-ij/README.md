# Loading Dots Pulse

Loading indicator with pulsing dots. Dots scale in and out with staggered delays via `--ldp-i`.

## Features

- Pulsing dots with scale animation
- Staggered animation delays via `--ldp-i`
- Multicolor dots
- Pure CSS, no JS required

## Usage

Add `.ldp-dot` elements inside `.ldp-container`. Set `--ldp-i` (0-based index) for stagger delay.
