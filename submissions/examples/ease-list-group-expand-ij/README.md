# ease-list-group-expand

A vertical accordion list with expandable items. Each item reveals its content on click with a smooth max-height transition and rotating arrow indicator.

## Features
- Expandable list items with smooth max-height transition
- Rotating arrow indicator on expand
- Independent per-item state
- Dark theme card styling

## Usage
Set `--expanded` to `0` (collapsed) or `1` (expanded) on each `.lge-item` element via `style.setProperty()`. The body slides open and the arrow rotates.
