# Timeline Card Rise

This example adds timeline cards that rise into place with a short stagger.

## Files

- `demo.html` contains the timeline card markup.
- `style.css` defines the staggered rise animation and reduced-motion fallback.

## Highlights

- Keeps card dimensions stable while animating with transforms.
- Uses per-card delays for a controlled stagger.
- Presents readable timeline content in compact cards.
- Shows static cards when reduced motion is enabled.
