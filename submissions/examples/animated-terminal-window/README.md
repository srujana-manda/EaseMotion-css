# Animated Terminal Window

A CSS-animated terminal window with typing text and blinking cursor — pure CSS.

## Features
- Staggered line reveal with animation-delay
- Blinking cursor effect
- Realistic terminal UI design

## Usage
```html
<div class="terminal">
  <div class="term-header">...</div>
  <div class="term-body">...</div>
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
