# Confetti Celebration

A CSS confetti burst animation for celebration effects — pure CSS.

## Features
- Falling confetti pieces with rotation
- Multi-color particle burst
- Staggered timing for natural look

## Usage
```html
<div class="confetti-wrap">
  <div class="confetti c1"></div>
  <!-- add more pieces -->
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
