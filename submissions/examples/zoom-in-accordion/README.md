# Zoom-In Accordion

Issue: #54331

## Overview

A responsive accordion component featuring a smooth zoom-in opening animation using only HTML and CSS.

## Features

- Pure HTML & CSS
- Native `<details>` and `<summary>`
- Responsive layout
- Smooth zoom animation
- Keyboard accessible
- No JavaScript required
- Supports `prefers-reduced-motion`

## Folder

## Files

## Usage

```html
<link rel="stylesheet" href="style.css">

<details class="zoom-accordion">
    <summary>Accordion Title</summary>
    <p>Your content here.</p>
</details>
```
