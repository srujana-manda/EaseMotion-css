# Flip Card 3D

3D flip card on hover with back face reveal. Perspective and rotateY create realistic 3D effect.

## Features

- 3D perspective on card container
- rotateY flip on hover
- Back face with hidden backface-visibility
- Smooth cubic-bezier timing

## Usage

Structure `.f3d-card > .f3d-inner > .f3d-front + .f3d-back`. CSS handles hover flip.
