# Ease Redis Cache Insights

## Overview

A responsive HTML and CSS component for displaying Redis cache statistics and health metrics.

## Features

- Cache hit rate
- Memory utilization
- Connected clients
- Evicted keys overview
- Responsive design
- CSS animations
- Pure HTML & CSS

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

A lightweight, reusable dashboard component for Redis monitoring, backend observability, and cloud infrastructure interfaces.
