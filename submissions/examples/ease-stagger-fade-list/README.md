# Stagger Fade List

A list with staggered fade-in items using animation-delay — pure CSS.

## Features
- Items fade and slide in one by one
- Staggered delay via `--i` custom property
- Smooth cubic-bezier transition

## Usage
```html
<ul class="stagger-list">
  <li style="--i:1">Item</li>
  <li style="--i:2">Item</li>
</ul>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
