# Ease Particle Click Explosion (V2)

## Overview
Advanced particle explosion with gravity and bounce physics.

## ✨ Features
- Gravity-based motion
- Bounce interaction
- Fade + scale animation
- Random colors and directions
- Multi-click support

## 💡 Usage
Click anywhere to trigger particle explosion.
