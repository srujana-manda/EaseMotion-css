# Autonomous Deep-Sea Exploration Intelligence Command — Phase #853

A deep-ocean command dashboard for autonomous underwater exploration. Built entirely with **HTML5**, **CSS3**, and **vanilla JavaScript**. Part of the EaseMotion CSS design system showcase.

---

## Features

- Animated ocean background with depth layers and bioluminescence
- Bubble rise effect
- Sonar grid overlay
- Depth gauge with ring dial and live stats
- 4 ROV fleet cards with battery, signal, heading metrics
- Animated ROV body hover, light pulse, propeller spin
- Interactive sonar display with rotating sweep and blips
- Sample analysis with animated spectra bars
- Environmental sensor grid
- Oxygen reserve ring gauge
- Toast notification system
- Responsive layout
- Reduced motion support
- Zero external dependencies
