# CSS Wobble-Focus Tabs for Product Catalog Layouts

**Issue:** #50216

## Description

A pure CSS animated Wobble-Focus Tabs component for Product Catalog layouts.

## Features

- Pure HTML & CSS
- Wobble focus animation
- Responsive design
- Keyboard accessible
- CSS custom properties
- No JavaScript

## Usage

```html
<link rel="stylesheet" href="style.css">
```

Use the HTML structure from `demo.html`.

## Customization

```css
:root{
  --primary:#2563eb;
  --duration:.5s;
}
```
