# Animated Statistics Card (`ease-stats-card-cv`)

A responsive statistics card component built using **HTML5** and **Pure CSS**.

## Features

- Animated statistic cards
- Floating icons
- Gradient number styling
- Hover shimmer effect
- Hover lift animation
- Responsive grid layout
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-stats-card-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

Works in all modern browsers.
