# Dropdown Fade Slide

Dropdown menu that fades in and slides down on click. Toggle arrow rotates on open.

## Features

- Dropdown toggle button
- Menu fades in and slides down
- Arrow rotation indicator
- Smooth transitions via `--dfs-open`

## Usage

Set `--dfs-open` to `1` on `.dfs-menu` to open. JS handles click toggling and outside close.
