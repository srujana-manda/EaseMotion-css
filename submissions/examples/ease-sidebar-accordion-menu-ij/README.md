# Sidebar Accordion Menu (`ease-sidebar-accordion-menu-ij`)

A dark-themed sidebar component with nested accordion menu items. Each parent item reveals its children with a smooth max-height transition. Chevron rotates 90deg on open.

## Features

- Dark sidebar theme with light main content area
- Accordion expand/collapse via `.sam-open` class
- Sub-menu slides down with max-height transition
- Chevron rotation on open (90deg) with spring-like cubic-bezier
- Hover states on triggers and links
- Responsive layout (sidebar shrinks on small screens)
- Google Fonts Inter loaded via CDN
- No external dependencies (pure CSS + vanilla JS)

## Usage

Open `demo.html` directly in any modern browser. Click any parent menu item to toggle its sub-menu.

## Browser Support

Works in all modern browsers.
