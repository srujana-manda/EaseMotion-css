# Form Submit Lift

This example adds a small form where the submit button lifts on hover and keyboard focus.

## Files

- `demo.html` contains the email field and submit button.
- `style.css` defines the form layout, button lift, focus states, and reduced-motion fallback.

## Highlights

- Uses native form controls.
- Keeps input and button heights stable across states.
- Provides visible keyboard focus treatment.
- Removes button movement when reduced motion is requested.
