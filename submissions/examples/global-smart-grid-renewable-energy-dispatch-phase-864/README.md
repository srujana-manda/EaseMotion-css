# Required Contribution Answers

## 1. What does this do?

This project demonstrates a responsive and animated smart-grid dashboard for monitoring renewable energy generation, battery storage, and power dispatch using HTML and CSS.

---

## 2. How is it used?

Open `demo.html` in any modern web browser.

Example:

```html
<section class="hero">
    ...
</section>

<div class="stat-card">
    ...
</div>

<div class="feature">
    ...
</div>
```

The page automatically loads:

- `easemotion.css`
- `style.css`

to display the complete interface.

---

## 3. Why is it useful?

This showcase demonstrates how modern dashboards can be created using reusable HTML structures, responsive CSS layouts, and smooth animations while remaining lightweight and requiring no JavaScript.
