# Skew 3D Card

Card skews with a 3D perspective transform on hover. Uses CSS `perspective` and `rotateX`/`rotateY` transforms for a realistic 3D tilt effect.

## Features

- 3D perspective transform on hover
- Skew and rotation via `rotateX`, `rotateY`, `skewX`
- Smooth ease-out transition
- Pure CSS, no JS required

## Usage

Add class `.stc-card` as a container with `.stc-inner` for the card face. Hover effect runs automatically.
