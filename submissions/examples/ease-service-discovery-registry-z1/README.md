# Ease Service Discovery Registry

## Overview

A responsive HTML and CSS component for visualizing service discovery registry health in distributed systems.

## Features

- Registered service cards
- Healthy instance count
- Heartbeat monitoring
- Status badges
- Responsive layout
- CSS animations
- Pure HTML & CSS

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

A reusable UI component for service discovery platforms, Kubernetes dashboards, and microservice monitoring interfaces.
