# CSS Rotate-Click Carousel – SaaS Showcase

A responsive **Rotate-Click Carousel** built using pure HTML and CSS for modern SaaS landing pages and product showcases.

## Features

- Pure HTML & CSS
- Rotate-click interaction
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript

## Usage

Open `demo.html` directly in any browser.

Customize animation using CSS variables.

```css
:root{
  --rotate:7deg;
  --duration:.35s;
}
```
