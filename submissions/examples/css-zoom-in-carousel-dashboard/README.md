# CSS Zoom-In Carousel – Responsive Dashboard

A pure CSS animated carousel featuring a smooth zoom-in interaction designed for responsive dashboard interfaces.

## ✨ Features

- Pure CSS
- Zoom-in hover animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required

## Usage

```html
<div class="carousel">
  <div class="card active">Dashboard</div>
  <div class="card">Analytics</div>
  <div class="card">Reports</div>
</div>
```

## Custom Properties

```css
:root{
  --zoom-scale:1.1;
  --duration:.35s;
  --easing:cubic-bezier(.22,1,.36,1);
}
```

## Technologies

- HTML5
- CSS3

## Why EaseMotion CSS?

This component provides a reusable, responsive zoom-in carousel using only HTML and CSS. It follows the EaseMotion CSS philosophy by being lightweight, accessible, customizable, and requiring no JavaScript.
