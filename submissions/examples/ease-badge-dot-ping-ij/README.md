# ease-badge-dot-ping

Small red dot badge with ping ring animation. Two expanding rings radiate outward with opacity fade to draw attention to the notification dot.

## Features

- Compact notification dot with red fill
- Two concentric ping rings with staggered delay
- Rings scale outward and fade simultaneously
- Clean dark badge container

## CSS Custom Property Control

- `--ping`: Set to `1` (default) to enable the ping ring animation, `0` to stop it. The checkbox toggles this property on the `.bdp-badge` element.
