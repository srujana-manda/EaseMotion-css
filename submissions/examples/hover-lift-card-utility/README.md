# Hover Lift Card Utility

A reusable CSS utility that smoothly lifts a card and increases its shadow on hover.

## Features

- Smooth hover lift animation
- Enhanced shadow effect
- Lightweight and reusable
- Pure HTML & CSS
- Suitable for cards, portfolios, pricing sections, and dashboards

## Usage

```html
<div class="hover-lift-card">
  <h2>Hover Me</h2>
  <p>This card lifts smoothly on hover.</p>
</div>
