# CSS 3D Flip Dropdown

A modern dropdown component built using pure HTML and CSS featuring a smooth **3D Flip** opening animation.

## Features

- Pure HTML & CSS
- 3D Flip Animation
- Responsive Layout
- Keyboard Accessible (`:focus-within`)
- CSS Custom Properties
- `prefers-reduced-motion` Support
- No JavaScript Required

Open `demo.html` directly in any modern browser.

- Chrome

## Accessibility

- Supports keyboard navigation using `:focus-within`.
- Respects the `prefers-reduced-motion` media query for users who prefer reduced animations.
- Fully responsive for desktop, tablet, and mobile devices.
