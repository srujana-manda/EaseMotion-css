# Ease Load Balancer Target Group Monitor

## Overview

A responsive HTML and CSS component for monitoring load balancer target groups, health checks, response times, and request distribution.

## Features

- Target group overview
- Healthy target count
- Request throughput
- Response time metrics
- Health status badges
- Responsive layout
- Pure HTML & CSS

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable animation-first component for load balancing dashboards, cloud infrastructure, reverse proxies, and DevOps monitoring platforms.
