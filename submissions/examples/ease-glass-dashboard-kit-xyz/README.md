# Glass Dashboard UI Kit (`.ease-dash-*-xyz`)

A comprehensive, fully responsive Glassmorphism Dashboard layout built with pure CSS. This kit includes a sidebar, header, stat cards, and a data table, all utilizing smooth "EaseMotion" cubic-bezier transitions and beautiful backdrop-filter effects.

## Components Included

1. **Sidebar Navigation** (`.ease-dash-sidebar-xyz`) - Glassmorphic sidebar with animated hover states.
2. **Header & Search** (`.ease-dash-header-xyz`) - Top bar with a glowing focus search input and profile avatar.
3. **Stat Cards** (`.ease-dash-stat-card-xyz`) - Animated metric cards with color-coded top borders.
4. **Data Table** (`.ease-dash-table-xyz`) - Clean, interactive table with hover row highlights.

## Usage

Copy the HTML structure and the CSS classes into your project. Ensure you include the CSS variables defined at the top of `style.css` for easy theming.

```html
<!-- Example: Stat Card -->
<div class="ease-dash-stat-card-xyz ease-dash-card-blue-xyz">
    <div class="ease-dash-stat-info-xyz">
        <p>Total Revenue</p>
        <h3>$45,231</h3>
    </div>
    <div class="ease-dash-stat-icon-xyz">💰</div>
</div>
