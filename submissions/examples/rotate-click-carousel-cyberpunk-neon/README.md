# CSS Rotate-Click Carousel – Cyberpunk Neon

A futuristic **Cyberpunk Neon Rotate-Click Carousel** built using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Rotate-click interaction
- Neon glow effects
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required

## CSS Variables

```css
:root{
  --primary:#00f7ff;
  --secondary:#ff00ff;
  --rotate:8deg;
  --duration:.35s;
}
```

## Folder Structure

```
rotate-click-carousel-cyberpunk-neon/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser. Hover, click, or focus on a card to see the rotate-click animation with neon glow effects. Customize colors, rotation angle, and animation timing using CSS variables.
