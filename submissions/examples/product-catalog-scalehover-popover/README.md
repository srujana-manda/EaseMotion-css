# CSS Scale-Hover Popover for Product Catalog Layouts

A responsive **pure CSS Scale-Hover Popover** built for modern product catalog interfaces. The component reveals additional product information using a smooth scale-hover animation without requiring JavaScript.

---

## Features

- Pure HTML & CSS
- Smooth Scale-Hover animation
- Product catalog inspired UI
- CSS Custom Properties
- Responsive layout
- Keyboard accessible (`:focus-within`)
- No JavaScript required

## Customization

The component exposes the following CSS variables:

```css
--popover-bg
--popover-text
--popover-width
--popover-radius
--popover-duration
--popover-scale
--popover-easing
--primary
--price
```

---

## Accessibility

- Keyboard accessible using `:focus-within`
- Uses `aria-describedby`
- Uses `role="dialog"`
- Semantic HTML structure

---

## Preview

The demo displays a product catalog card where hovering or focusing on the **View Details** button reveals a smooth scale-hover popover containing key product highlights.
