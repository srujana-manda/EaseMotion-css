# Ease Load Balancer Traffic Distribution

## Overview
A responsive HTML and CSS component for monitoring traffic distribution across backend servers behind a load balancer.

## Features
- Backend traffic overview
- Health status badges
- Active connection display
- Responsive design
- CSS variables
- Hover animations
- Pure HTML & CSS

## Folder Structure

demo.html
style.css
README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Responsive

Optimized for desktop, tablet, and mobile devices.

## Why EaseMotion CSS?

This component provides a reusable, animation-first UI for visualizing load balancer traffic, backend server health, and connection distribution in cloud infrastructure dashboards.
