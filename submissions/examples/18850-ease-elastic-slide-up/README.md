# Animation: Ease Elastic Slide Up

## Overview

Ease Elastic Slide Up is a modern entrance animation that combines a smooth upward slide, fade-in effect, and subtle elastic bounce. The animation starts slightly below its final position and overshoots before naturally settling into place, creating a polished and engaging motion.

## Features

- Smooth fade-in transition
- Upward slide motion
- Elastic bounce effect
- Lightweight CSS-only implementation
- Modern cubic-bezier easing
- Accessibility support using `prefers-reduced-motion`
- Reusable across multiple UI components

## Use Cases

- Hero section headings
- Landing page content
- Feature cards
- Dashboard widgets
- Call-to-action sections
- Portfolio websites
- Product showcases

## Example Usage

```html
<div class="ease-elastic-slide-up">
  Welcome to EaseMotion CSS
</div>
```

## Animation Behavior

1. Starts below the final position.
2. Fades in while moving upward.
3. Slightly overshoots the final position.
4. Gently settles into place.
5. Provides a smooth and natural entrance effect.

## Accessibility

Supports users who prefer reduced motion using the `prefers-reduced-motion` media query.

## Browser Compatibility

Compatible with modern browsers supporting:

- CSS Keyframes
