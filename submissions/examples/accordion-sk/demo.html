What is EaseMotion CSS?

EaseMotion CSS is a human-readable, animation-first CSS framework. It ships as zero-dependency plain CSS files. No build step, no JavaScript required for core styling.

How do I install it?

Drop easemotion.css in your project and link it. Or install via npm: npm install easemotion-css. No build step or config required.

Does it work without JavaScript?

Yes. This accordion uses native HTML <details> elements with the name attribute for group exclusivity — purely semantic HTML and CSS. No script tags.

Multi Accordion — Multiple Panels Open

Getting Started

Add accordion markup using native <details> and <summary> elements. Apply the .accordion class to each details element.

Styling Options

The accordion supports three visual variants: .accordion--bordered, .accordion--elevated, and the default bare style. All variants support dark mode automatically.

Accessibility

Built on semantic HTML. Keyboard navigable with Tab and Enter/Space. Screen readers announce expanded/collapsed state through native roles.

Themed Variant — Bordered

Border variant

Add .accordion--bordered for a card-like layout with visible borders separating each panel.

Another panel

Each item has a defined border radius and spacing, creating a clean stacked appearance suitable for documentation.

Themed Variant — Elevated

Elevated variant

Add .accordion--elevated for floating card panels with subtle shadows and larger border radius.

Marketing FAQ panel

The elevated style works well in hero sections, pricing pages, and landing layouts where cards need visual separation from the background.
