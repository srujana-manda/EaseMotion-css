# ease-carousel-slide-auto

An image carousel with auto-slide transition and dot navigation. The track translates horizontally to reveal each slide.

## Features
- 4 slides with smooth translateX transition
- Dot navigation indicators
- Button controls for direct slide access
- Dark theme with shadow

## Usage
Set `--slide` (0-3) on the `.csa-carousel` element via `style.setProperty()` to navigate to a slide. Active dot highlights automatically.
