# Swing Hover Tabs - Dashboard Analytics Layouts

## Description

A pure CSS animated tab component with a smooth swing-hover interaction designed for dashboard analytics interfaces.

The component provides an engaging hover motion without requiring JavaScript.

## Features

- Pure CSS animation
- Smooth swing hover effect
- Responsive layout
- Keyboard accessible focus states
- Custom CSS variables for animation control


## Usage

Add the class:

```html
<button class="swing-tab">
    Overview
</button>
