# Pricing Table Toggle

Pricing table with toggle between monthly/yearly billing. Prices update via `--pt-amount` custom property. Toggle switch animates smoothly.

## Features

- Toggle between monthly/yearly pricing
- Price amounts update via `--pt-amount`
- Toggle switch with sliding thumb animation
- Card hover elevation effect
- Save badge on yearly selection

## Usage

Set `--pt-amount` on `.pt-price`. JS updates amount and period on toggle. CSS handles switch animation and hover effects.
