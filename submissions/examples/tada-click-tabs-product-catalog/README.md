# CSS Tada-Click Tabs for Product Catalog Layouts

## Overview
A responsive pure CSS animated tab component with a **Tada-Click** interaction for Product Catalog interfaces.

## Features

- Pure HTML & CSS
- Responsive
- Keyboard accessible
- No JavaScript required
- Customizable animation using CSS

## Usage

```html
<link rel="stylesheet" href="style.css">
```

Open `demo.html` directly in your browser.
