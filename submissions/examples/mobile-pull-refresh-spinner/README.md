# Mobile Pull Refresh Spinner

A mobile-style pull-to-refresh visual spinner — pure CSS.

## Features
- Circular arc spinner with rotation
- Simulated pull-down overscroll
- Mobile-style loading indicator

## Usage
```html
<div class="refresh-spinner">
  <div class="arc"></div>
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
