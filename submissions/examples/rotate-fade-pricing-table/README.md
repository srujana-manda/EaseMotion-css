# Rotate-Fade Pricing Table

A pure CSS pricing table component with smooth rotate and fade hover animations.

## Features

- Pure HTML and CSS
- Rotate hover interaction
- Fade transition effect
- Responsive layout
- No JavaScript required
- Accessible reduced motion support


## Usage

Example:

```html
<div class="card">
    <h2>Plan</h2>
    <h3>$20/month</h3>
</div>
