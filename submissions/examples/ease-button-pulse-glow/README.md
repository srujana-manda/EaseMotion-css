# Button Pulse Glow

A button with pulsing glow animation — pure CSS.

## Features
- Pulsing box-shadow glow effect
- Optional glowing ring pseudo-element
- Secondary outlined variant
- Smooth easing animation

## Usage
```html
<button class="pulse-glow-btn">Click Me</button>
<button class="pulse-glow-btn secondary">Secondary</button>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
