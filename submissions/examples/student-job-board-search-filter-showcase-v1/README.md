# Student Job Board Search & Filter Showcase

A fully responsive, modern UI for a Student Job Board, emphasizing search and filter functionalities. This project is built using pure HTML5 and CSS3 without any JavaScript.

## Features

- **Search Bar UI:** Intuitive search interface to find specific job titles or keywords.
- **Advanced Filtering:** Dropdowns for Branch and Minimum CGPA, and an input for specific Skills.
- **Active Filter Chips:** Visual representation of currently applied filters.
- **Results Counter:** Dynamic display of the number of jobs found.
- **Job Cards Grid:** Responsive and detailed job cards showcasing job title, company, location, required branch, minimum CGPA, and specific skills.
- **Modern Glassmorphism Design:** Utilizes CSS `backdrop-filter` for a premium frosted glass effect on panels.
- **Fully Responsive:** Adapts seamlessly to desktop, tablet, and mobile viewing environments.
- **Accessibility Friendly:** Semantic HTML and ARIA labels implemented for improved accessibility.

## Design Highlights

- **Glassmorphism Panels:** Semi-transparent backgrounds with background blurs to create depth.
- **Soft Gradients:** Subtle and vibrant gradients used for company logos to give a pop of color.
- **Smooth Hover Animations:** Lift and shadow effects on job cards for interactive feel.
- **Professional Typography:** Utilizes the 'Inter' font for a clean, legible, and modern appearance.
- **Floating Background Shapes:** CSS animations used to create a dynamic, living background without JavaScript.

## Accessibility Features

- Clear and contrasting colors.
- Use of `aria-label` for icon-only buttons (like search, clear, bookmark, and remove filter buttons).
- Semantic structure using `<main>`, `<section>`, `<article>`, and `<header>`.

## How to Run

1. Navigate to the `submissions/examples/student-job-board-search-filter-showcase/` directory.
2. Open `demo.html` in your favorite modern web browser.
3. No build tools or servers are required.
