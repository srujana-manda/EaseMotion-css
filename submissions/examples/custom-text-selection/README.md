# Text Selection Highlight Utility

## Description

This utility provides reusable classes for customizing text selection colors using the `::selection` pseudo-element.

## Features

- Custom text selection colors
- Lightweight and reusable
- Easy to apply
- Improves UI consistency

## Usage

```html
<p class="ease-selection-primary">
  Select this text.
</p>

<p class="ease-selection-success">
  Select this text.
</p>

<p class="ease-selection-danger">
  Select this text.
</p>
