# Video Play Button

An animated play button with expanding ripple rings and scale effect. State via `--vp-state` (0 or 1). CSS handles the button scale, background, and ripple ring keyframes.

## Features

- Animated play button with ripple rings
- State via `--vp-state` CSS variable
- Expanding ripple animation with @keyframes
- Scale transition on hover and state change
- Card layout with duration badge

## Usage

Set `--vp-state` (0 or 1) on `.vp-play-btn`. CSS handles the button scale and background transition. The ripple rings expand outward continuously.
