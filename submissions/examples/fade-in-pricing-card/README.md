# Fade In Pricing Card

A pure CSS animated pricing card component for EaseMotion CSS.

## Features

- Smooth fade-in entrance animation
- Modern pricing card layout
- Hover lift interaction
- Responsive design
- No JavaScript required


## Usage

Include the HTML structure and add the provided CSS file.

Example:

```html
<div class="pricing-card">
    <h2>Pro Plan</h2>
    <p>$29/month</p>
</div>
