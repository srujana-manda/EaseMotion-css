# Liquid Blob Hover Button Animation

A button with a liquid blob fill animation that expands smoothly from the center on hover, using CSS keyframes only. No JavaScript required.

## Features

- 💧 Blob expands from the button's center on hover, filling it completely
- 🌊 Wobbling `border-radius` keyframe animation gives the fill a soft, liquid feel
- 🎨 Easy color theming via a modifier class (`liquid-btn--green` included as an example)
- 📱 Responsive — scales down padding and font size on small screens
- ♿ Respects `prefers-reduced-motion` for accessibility
- 🧩 Pure HTML + CSS — CSS keyframes only, no JavaScript dependencies

## Usage

Include `style.css` and add the button markup:

```html
<button class="liquid-btn">Get Started</button>
