# Animated Call-to-Action Banner (`ease-cta-banner-cv`)

A responsive call-to-action banner built using **HTML5** and **Pure CSS**.

## Features

- Animated entrance
- Floating decorative background
- Gradient banner
- Hover-animated CTA buttons
- Responsive layout
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-cta-banner-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

Works in all modern browsers.
