# Animated Pricing Cards with Billing Toggle

This submission showcases a modern **SaaS Pricing Section** with a Monthly/Yearly billing toggle. It demonstrates how to use **EaseMotion CSS** to create delightful micro-interactions, specifically triggering bounce animations when dynamic content (prices) updates.

## ✨ Features
- **Interactive Toggle**: Switches between Monthly and Yearly pricing dynamically.
- **EaseMotion Integration**: 
  - `ease-bounce-in`: Triggered via JavaScript whenever the price text updates, drawing the user's eye to the new value.
  - `ease-hover-lift`: Applied to all cards and primary buttons for tactile, premium hover feedback.
  - `ease-hover-glow`: Subtle border/text glow on secondary outline buttons.
  - `ease-fade-in`, `ease-slide-down`, `ease-scale-up`: Staggered, elegant entrance animations for the entire page layout.
- **Responsive Grid**: Adapts seamlessly from mobile to desktop using CSS Grid.

## 🚀 How to Use
1. Open `demo.html` in any modern web browser.
2. Observe the staggered entrance animations on page load.
3. Click the **Monthly/Yearly toggle switch**. Notice how the prices smoothly update with an `ease-bounce-in` effect.
4. Hover over the cards and buttons to see the `ease-hover-lift` and `ease-hover-glow` micro-interactions.

## 📂 Files
- `demo.html`: Standalone demo with vanilla JS toggle logic.
- `style.css`: Custom layout, typography, and toggle switch styling.
