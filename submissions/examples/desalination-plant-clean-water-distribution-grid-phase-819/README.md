# Desalination Plant Clean Water Distribution Grid (Phase #819)

Closes #28367 — A responsive, 60fps UI showcase built with EaseMotion CSS.

## Features
- Pure CSS animations: float, fade‑in, fill‑bar, and scroll‑triggered number counters.
- Four distribution nodes (Plant A, Station B, Agri Hub, City Supply) with animated fill bars.
- Live metrics section with animated numbers (Water Quality, Flow Rate, pH).
- Fully responsive layout.

## EaseMotion Classes Used
- `float` – smooth up‑down floating on cards.
- `fadeDown` – header entrance.
- `fillBar` – animated progress bars.
- Scroll‑triggered counters (vanilla JS, zero dependencies).

## How to Preview
Open `demo.html` directly in any modern browser.

## File Structure
- `demo.html` – main UI
- `style.css` – all custom styles and keyframes
- `README.md` – this documentation
