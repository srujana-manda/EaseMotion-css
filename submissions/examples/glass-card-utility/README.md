# Glassmorphism Card Utility

## Description

A reusable Glassmorphism Card Utility that creates a modern frosted-glass effect using transparency, blur, and subtle borders.

## Features

- Frosted glass appearance
- Responsive card layout
- Lightweight and reusable
- Modern UI design
- Easy to customize

## Usage

```html
<div class="glass-card">
  <h2>Glass Card</h2>
  <p>Your content here.</p>
</div>
