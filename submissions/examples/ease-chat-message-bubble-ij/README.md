# Chat Message Bubble

Chat message bubble that slides in with animation. Sent vs received styling with tail-like border-radius.

## Features

- Chat bubble with slide-in animation
- Sent (right) vs received (left) styling
- Tail pointer via border-radius
- Smooth entrance animation

## Usage

Use `.cmb-sent` and `.cmb-received` classes on `.cmb-bubble` elements. JS adds new messages dynamically.
