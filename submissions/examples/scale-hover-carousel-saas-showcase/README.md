# CSS Scale-Hover Carousel – SaaS Showcase

A responsive **SaaS Showcase Carousel** built using pure HTML and CSS with smooth scale-hover interactions.

## Features

- Pure HTML & CSS
- Modern SaaS card layout
- Smooth scale-hover animation
- Responsive design
- Keyboard accessible
- CSS custom properties
- No JavaScript required

## CSS Variables

```css
:root{
  --primary:#2563eb;
  --secondary:#60a5fa;
  --scale:1.08;
  --duration:.35s;
}
```

## Folder Structure

```
scale-hover-carousel-saas-showcase/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any browser. Customize colors, scale, duration, and spacing through the CSS variables.
