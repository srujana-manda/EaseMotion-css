# Autonomous Hypersonic Aerospace Control Command — Phase #766

A hypersonic aerospace command dashboard for autonomous flight control. Built entirely with **HTML5**, **CSS3**, and **vanilla JavaScript**. Part of the EaseMotion CSS design system showcase.

---

## Features

- Animated aerospace background with atmosphere layers and contrails
- Shockwave expansion effect
- Flight grid overlay
- Mach gauge with ring dial and live stats
- 4 vehicle fleet cards with fuel, signal, heading metrics
- Animated vehicle hover, heat pulse, and contrail fade
- Flight trajectory display with SVG path and animated vehicle dot
- Live telemetry with animated spectra bars
- Environmental sensor grid
- Fuel reserve ring gauge
- Toast notification system
- Responsive layout
- Reduced motion support
- Zero external dependencies
