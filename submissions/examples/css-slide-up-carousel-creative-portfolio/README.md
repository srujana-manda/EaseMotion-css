# CSS Slide-Up Carousel (Sample)

A sample pure CSS Slide-Up Carousel for creative portfolio layouts.

## Features

- Pure HTML & CSS
- Slide-up hover animation
- Responsive layout
- Smooth transitions
- No JavaScript

## Files

demo.html
style.css
README.md

## Usage

Open `demo.html` in any browser.

## Customization

You can modify:

- Card width
- Border radius
- Transition speed
- Overlay color
- Font styles
