# Global Smart Grid Renewable Energy Dispatch Showcase (Phase #814)

1. **What does this do?**  
   It renders a dark-themed green energy smart grid dispatch dashboard displaying regional sector logs, renewable generation yields (solar/wind/hydro), and battery storage capacities.

2. **How is it used?**  
   Open `demo.html` in any standard web browser. The visual grids, rotating turbines, battery cell animations, and solar trackers will render automatically.
   
3. **Why is it useful?**  
   It demonstrates interactive energy grid telemetry mockups utilizing clean CSS grids, turbine rotation loops, and battery charge simulators.
