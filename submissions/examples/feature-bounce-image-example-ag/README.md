# Bounce Image Example (Standard)

## Description
A standard HTML/CSS example demonstrating a "Bounce" Attention Seeker animation applied to an image. The image automatically bounces up and down in a loop, squashing slightly when it hits the "floor", simulating a physical rubber ball to draw the user's focus.

## Files
- `demo.html`: Contains the image markup wrapped in a container that allows space for the vertical movement.
- `style.css`: Contains the image styling and the `@keyframes image-attention-bounce-ag` looping animation.

## Accessibility
- Used an empty `alt` attribute for decorative elements, or descriptive `alt` for functional images.
- **Reduced Motion**: Disables the bouncing animation entirely. The image remains completely static, preventing continuous repetitive motion from distracting or causing discomfort.
