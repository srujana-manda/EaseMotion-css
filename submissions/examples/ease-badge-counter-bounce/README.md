# Badge Counter Bounce

A badge counter with bounce animation — pure CSS.

## Features
- Bouncing scale animation on the counter
- Multiple color variants (default, danger, success)
- Positioned relative to parent button
- Smooth keyframe animation

## Usage
```html
<div class="badge-wrapper">
  <button class="badge-btn">Label</button>
  <span class="badge-counter">3</span>
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
