# Accessibility & Responsive Demo Components

## Overview

This submission provides a collection of responsive and accessible UI components built for EaseMotion CSS. It demonstrates practical real-world examples that developers can use as a reference.

## Components Included

- Responsive Pricing Cards
- FAQ Accordion
- Login Form

## Features

- Responsive layout for desktop, tablet, and mobile devices
- Keyboard-friendly navigation
- Visible `:focus-visible` styles for accessibility
- Clean and modern UI
- Semantic HTML structure
- Easy to customize and extend
