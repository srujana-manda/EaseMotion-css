## Google Ripple Bars
Sound wave style loading animation with Google's signature colors. Features animated vertical bars that ripple like audio visualizers, with staggered delays for a fluid wave effect.

## Features
- Animated ripple/sound wave effect
- Staggered animation delays for fluid motion
- Google brand colors (Blue, Red, Yellow, Green)
- Multiple size variants (standard, large, small)
- Gradient color option for rainbow effect
- Customizable via CSS variables
- Pure CSS, no JavaScript required
- Reduced motion accessibility support
- Responsive design

## Usage

### Basic Usage
```html
<div class="ripple-bars">
  <span class="ripple-bar"></span>
  <span class="ripple-bar"></span>
  <span class="ripple-bar"></span>
  <!-- Add more bars for longer wave -->
</div>
