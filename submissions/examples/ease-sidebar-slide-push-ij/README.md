# ease-sidebar-slide-push

A sidebar component that slides in from the left and pushes the main content.

## Features

- Sidebar slides in from left with smooth cubic-bezier transition
- Main content shifts right by the sidebar width
- Controlled via `--sidebar-open` CSS custom property
- Dark theme with modern card layout

## Usage

Set `--sidebar-open: 1` on the `.ssp-wrapper` element to open the sidebar. Set it to `0` to close.
