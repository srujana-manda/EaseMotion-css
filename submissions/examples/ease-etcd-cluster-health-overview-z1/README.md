# Ease Etcd Cluster Health Overview

## Overview

A responsive HTML and CSS dashboard component for monitoring Kubernetes etcd cluster health and quorum status.

## Features

- etcd member health overview
- Leader synchronization tracking
- Replication latency monitoring
- Quorum status alerts
- Responsive layout
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable infrastructure component for monitoring etcd health, identifying quorum risks, and improving Kubernetes control-plane reliability.
