# Ease Cross Region Replication Health

## Overview

A responsive HTML and CSS component for monitoring cross-region replication performance, synchronization, and disaster recovery readiness.

## Features

- Primary region health
- Replication lag monitoring
- Synchronization throughput
- Failover readiness
- Responsive layout
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable infrastructure monitoring component for cloud databases, replication monitoring, disaster recovery planning, and enterprise operations dashboards.
