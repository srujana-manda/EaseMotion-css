# CSS Elastic-Slide Dropdown

A responsive dropdown menu built using pure HTML and CSS featuring a smooth elastic slide animation.

## Features

- Pure HTML & CSS
- Elastic Slide Animation
- Responsive Design
- Keyboard Accessible (`:focus-within`)
- CSS Variables
- `prefers-reduced-motion` Support
- No JavaScript Required

## Folder Structure

```text
elastic-slide-dropdown/
├── demo.html

```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

- Supports keyboard navigation using `:focus-within`.
- Supports the `prefers-reduced-motion` media query to reduce animation for users who prefer reduced motion.
