# Holographic Hover Card (`.ease-holo-card-xyz`)

A modern, glassmorphism-inspired card component featuring a smooth, rotating holographic border that activates on hover. Built with a focus on premium easing curves to match the "EaseMotion" philosophy.

## Usage

Add the `ease-holo-card-xyz` class to your card container.

```html
<div class="ease-holo-card-xyz">
    <div class="ease-holo-card-content-xyz">
        <div class="ease-holo-card-icon-xyz">🚀</div>
        <h2>Your Title</h2>
        <p>Your description goes here.</p>
    </div>
</div>
