# Ease Network Policy Compliance Panel

## Overview

A responsive HTML and CSS component for monitoring Kubernetes network policy coverage, blocked traffic, and compliance health.

## Features

- Policy coverage overview
- Blocked traffic metrics
- Namespace compliance monitoring
- Cluster security score
- Responsive layout
- Pure HTML & CSS
- CSS variables and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable UI component for Kubernetes security dashboards, Zero Trust monitoring, cloud networking, and enterprise compliance workflows.
