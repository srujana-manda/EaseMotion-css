Theme Switcher

A multi-theme switcher component using CSS custom properties — pure CSS.

Demo

Themed Card

This card's colors change with the selected theme.

Usage

/* Define themes via CSS custom properties on :root */
body:has(#light:checked) { --bg: #fff; --text: #111; }
body:has(#dark:checked) { --bg: #111; --text: #eee; }
