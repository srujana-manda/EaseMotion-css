# Live Search Input

Search input with live filtering animation. Results appear/disappear based on query. Clear border highlight on focus.

## Features

- Search input with icon
- Results filter in real-time via data-term matching
- Fade transition on show/hide
- Focus border highlight
- Item count display

## Usage

Each `.ls-item` has a `data-term` attribute for matching. JS handles filtering; CSS handles the visual transitions and focus state.
