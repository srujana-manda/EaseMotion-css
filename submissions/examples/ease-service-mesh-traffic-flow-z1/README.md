# Ease Service Mesh Traffic Flow

## Overview
A responsive HTML and CSS component for monitoring service-to-service communication inside a service mesh.

## Features
- Service communication cards
- Latency and request metrics
- Health status indicators
- Responsive layout
- CSS variables
- Hover animations
- Pure HTML & CSS

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` in any modern browser.

## Browser Support
- Chrome
- Firefox
- Edge
- Safari

## Responsive
Optimized for desktop, tablet, and mobile devices.

## Why EaseMotion CSS?
Provides a reusable animation-first component for cloud-native dashboards, Kubernetes observability tools, and service mesh monitoring platforms.
