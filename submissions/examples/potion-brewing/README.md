# Potion Brewing

An animated magical potion brewing component built using HTML, CSS and Vanilla JavaScript.

## Features

- 🧪 Glass potion flask
- 🫧 20 animated bubbles
- 🌈 8 potion colors
- ✨ Sparkle effects
- 💨 Steam animation
- 🍾 Cork pop brewing animation
- ⚡ Speed control
- 🔄 Reset button
- 📱 Responsive design
- 🚫 No external libraries

## Files

```
potion-brewing/
├── demo.html
├── style.css
├── script.js
└── README.md
```

## Usage

Open `demo.html` in any modern browser.

Use the controls to:
- Brew a potion
- Change potion color
- Adjust animation speed
- Reset everything
