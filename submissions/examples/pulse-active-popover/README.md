# CSS Pulse-Active Popover

A responsive **Pure CSS Pulse-Active Popover** designed for **Product Catalog layouts**. The component features a smooth pulse animation, hover/focus interactions, keyboard accessibility, and customizable styling using CSS custom properties.

## Features

- Pure HTML & CSS
- Responsive Design
- Keyboard Accessible (`:focus-visible`)
- Smooth Pulse Animation
- Hover & Focus Popover
- CSS Custom Properties
- No JavaScript Required

## Technologies Used

- HTML5
- CSS3

## CSS Variables

```css
--pulse-duration
--pulse-scale
--popover-bg
--popover-color
--radius
--shadow
--transition
```
