# Pulse Active Tabs - Creative Portfolio


A pure CSS animated tab component with a smooth pulse-active interaction,
created for creative portfolio interfaces.


## Features

- Pure CSS animation
- No JavaScript required
- Responsive design
- Keyboard accessible focus states
- Custom CSS variables
- Modern portfolio UI style


## Usage


```html
<div class="tabs">

<button class="tab active">
Projects
</button>

<button class="tab">
Designs
</button>

</div>
