# Stagger Button · Sequential Dot / Letter Animation

> A playful button where individual dots or letters pop into view one after another, creating a delightful staggered entrance effect.

---

## 📖 Description

The **Stagger Button** brings micro-interactions to life by animating each character (dots or letters) in sequence. When triggered (on click or hover), each element transitions smoothly with a slight delay, creating a wave-like ripple effect.

This component comes in two variations:
- **Dots version** – five dots pop in sequence (great for loading states)
- **Letters version** – the word "Stagger" spells out one letter at a time

Perfect for:
- Loading / processing indicators
- Playful call-to-action buttons
- Branding moments in UI
- Educational demos of staggered animations

---

## 🎯 Features

- ✅ Sequential animation with CSS `transition-delay`
- ✅ Two variants: dots and letters
- ✅ Trigger on click (with optional hover support)
- ✅ Smooth `cubic-bezier` bounce effect
- ✅ Click pulse feedback
- ✅ Fully responsive
- ✅ Accessible (`:focus-visible` support)
- ✅ No dependencies (except Font Awesome for icons)

---

## 🚀 Quick Start

1. Place `index.html` and `style.css` in the same folder.
2. Open `index.html` in your browser.
3. Click either button to see the stagger effect.
4. Watch the dots or letters animate in sequence!
