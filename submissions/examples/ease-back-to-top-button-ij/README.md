# ease-back-to-top-button

A fixed circular button that appears when the user scrolls down and scrolls back to top on click.

## Features

- Fixed position at bottom-right corner
- Fades in with smooth opacity and translateY transition
- Arrow up SVG icon
- Controlled via `--visible` CSS custom property

## Usage

Set `--visible: 1` on the `.btt-container` element to show the button, or `0` to hide it. Typically toggled based on scroll position.
