EaseMotion CSS is a human-readable, animation-first CSS framework. It ships as zero-dependency plain CSS files — no build step required.

  • 60+ utility classes for layout, spacing, and typography
  • 30+ animation keyframes with entrance, hover, and looping effects
  • 20+ responsive component styles (buttons, cards, forms, etc.)
  • Full dark mode and prefers-reduced-motion support

EaseMotion CSS is free and open source under the MIT license. No subscription, no registration — just drop the CSS file and build.
