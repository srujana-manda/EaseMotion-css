# Ease CI Pipeline Stage Board

## Overview
A CSS-only dashboard component for visualizing CI/CD pipeline stages.

## Features
- Pipeline stage cards
- Animated running state
- Responsive layout
- CSS variables
- Pure HTML & CSS

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` directly in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari

## Responsive
Optimized for desktop, tablet, and mobile devices.

## Why EaseMotion CSS?
Provides a reusable animation-first component for CI/CD dashboards, DevOps tools, deployment monitoring, and engineering portals.
