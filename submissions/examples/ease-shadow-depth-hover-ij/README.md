# Shadow Depth Hover (#31850)

An element with layered shadows that deepen on hover, creating a physical lifting effect with a smooth transition.

## Features
- Pure CSS – no JavaScript required
- Multi-layered `box-shadow` for realistic depth
- Smooth `translateY` lift combined with shadow transition
- Triggered by `:hover`

## Usage
Add the `.depth-card` class to any element. Hover to see the shadow deepen and the card lift.
