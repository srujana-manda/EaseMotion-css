ease-mixin Color Variants vs Custom Property Fallbacks — Conflict Fix

Previously, the SCSS color-variant() mixin hardcoded its own color values directly into rules, while components elsewhere referenced var(--ease-color-primary, #someFallback) with a DIFFERENT fallback color than what the mixin actually output.
