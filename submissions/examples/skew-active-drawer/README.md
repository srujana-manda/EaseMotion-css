# CSS Skew Active Drawer

A responsive **Skew Active Drawer** built using pure HTML and CSS for creative portfolio layouts.

## Features

- Pure HTML & CSS
- Smooth skew-on-hover animation
- Glassmorphism design
- Fully responsive
- No JavaScript required
- Supports `prefers-reduced-motion`

## Folder Structure

```
skew-active-drawer/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

Hover over the drawer to activate the skew animation.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

Users who enable **Reduce Motion** automatically receive a version without animations.
