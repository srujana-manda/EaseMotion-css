# Rotate-Click Carousel for Dashboard Analytics Layouts

**Issue:** #50466

## Description

A responsive pure CSS Rotate-Click Carousel designed for Dashboard Analytics layouts. Cards rotate and scale smoothly on interaction without JavaScript.

## Features

- Pure HTML & CSS
- Responsive
- Keyboard accessible
- CSS Custom Properties
- Lightweight
- Easy to customize
