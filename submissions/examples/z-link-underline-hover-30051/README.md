# Link Underline Hover Component

A minimal CSS component that creates a smooth animated underline effect on hover.

## Features

- Pure CSS
- No JavaScript required
- Responsive
- Accessible
- Keyboard focus support
- Smooth animation

## Preview

Hover over links to reveal a sliding underline animation.

## Usage

```html
<a class="underline-link">

Documentation

</a>
```
