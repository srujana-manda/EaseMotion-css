# Pull Handle Spinner

Pull-down interaction with spinner that rotates proportionally to drag distance via `--pull-progress`. Past threshold triggers refresh animation.

## Features

- Spinner rotation tied to pull progress
- CSS handles visual rotation via --pull-progress
- JS sets progress variable from drag distance
- Past threshold triggers spin animation

## Usage

Set `--pull-progress` (0-1) on `.phs-spinner`. Add `.phs-refresh` class for continuous rotation.
