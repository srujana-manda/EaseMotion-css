# Loading Progress Bar

Animated indeterminate progress bar for loading states. A gradient bar slides back and forth within a track to indicate ongoing activity.

## Features

- Indeterminate progress animation
- Gradient fill bar that slides back and forth
- Smooth `@keyframes` animation
- Pure CSS, no JS required

## Usage

Add `.lpb-bar` as the track container with `.lpb-fill` as the animated child. Animation runs automatically.
