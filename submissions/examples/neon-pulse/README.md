# Neon Pulse Animation

A vibrant, glowing pulse animation effect for buttons, cards, and text elements. Perfect for creating eye-catching call-to-action elements with a cyberpunk/neon aesthetic.

## Preview

This animation creates a pulsating neon glow effect that draws attention to interactive elements.

## Features

- ✨ **Multiple Color Variants**: Blue, Pink, Green, Purple
- 🎯 **Three Element Types**: Buttons, Cards, Text
- 🌊 **Smooth Pulse Animation**: Continuous glowing effect
- 🎨 **Hover Interactions**: Enhanced effects on hover
- 📱 **Fully Responsive**: Works on all screen sizes
- ⚡ **Pure CSS**: No JavaScript required

## Usage

### Button

```html
<button class="neon-btn neon-blue">Click Me</button>
<button class="neon-btn neon-pink">Subscribe</button>
<button class="neon-btn neon-green">Download</button>
<button class="neon-btn neon-purple">Sign Up</button>
