# CSS Draw-Border Line Tabs

A pure CSS animated tab component with a smooth draw-border line interaction designed for responsive dashboard interfaces.

## ✨ Features

- Pure CSS
- Draw-border hover animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required

## Usage

```html
<div class="tabs">
  <button class="tab active">Dashboard</button>
  <button class="tab">Analytics</button>
</div>
```

## Custom Properties

```css
:root{
  --border-color:#2563eb;
  --duration:.35s;
  --easing:cubic-bezier(.25,.8,.25,1);
}
```

## Technologies

- HTML5
- CSS3

## Why EaseMotion CSS?

This component provides a clean, reusable border-drawing animation using only CSS. It aligns with the EaseMotion CSS philosophy by being lightweight, responsive, accessible, and customizable without JavaScript.
