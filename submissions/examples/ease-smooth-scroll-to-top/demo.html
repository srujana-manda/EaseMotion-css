New Component

ease-smooth-scroll-to-top

A responsive, lightweight scroll-to-top button. Tracks page height and fills a circular progress ring around the arrow via pure CSS scroll timelines, falling back to a vanilla JS listener on older browsers.

Scroll Progress Demo

Please scroll down this page. The button is completely invisible at the top and will fade in dynamically as you scroll down. The circle around the button will fill up representing your scroll depth.

↓ Scroll Down ↓
Section 01 // 25% Scroll Depth

Pure CSS Scroll Timelines

On modern browsers (Chrome, Edge), the button fades in and its progress ring fills up completely inside the browser's compositor layer without a single line of JavaScript. This guarantees a smooth 60fps scroll feel without layout thrashing.

Section 02 // 50% Scroll Depth

Spring Bounce Hover Actions

Hovering over the button triggers a scaling transition and causes the arrow inside to oscillate upwards. This is handled by a custom cubic-bezier spring easing transition variable.

Section 03 // 75% Scroll Depth

Cross-Browser Fallback System

Older browsers (like older Safari/Firefox) that lack scroll-timeline support automatically fall back to a lightweight, throttled window scroll event listener that updates progress attributes, ensuring total reliability.

Installation & Usage

Place the markup at the bottom of your body tag. The anchor link target should match the ID of your top page anchor container.

<!-- Scroll to top button -->
<a href="#top" class="ease-scroll-to-top" aria-label="Scroll to top">
  <svg class="ease-scroll-to-top-progress" viewBox="0 0 36 36">
    <circle class="ease-scroll-to-top-progress-bg"></circle>
    <circle class="ease-scroll-to-top-progress-bar"></circle>
  </svg>
  <div class="ease-scroll-to-top-btn">
    <svg class="ease-scroll-to-top-icon" viewBox="0 0 24 24">...</svg>
  </div>
</a>

Modifier Classes

Class Name Target Element Description
.ease-scroll-to-top Container Anchor Base container. Handles fixed positioning at the bottom right.
.ease-scroll-to-top--visible Container Anchor Forces visibility of the button (used as fallback for older browsers).
.ease-scroll-to-top-progress SVG Wrapper Absolute layered vector canvas. Rotates -90deg to orient start at top.
.ease-scroll-to-top-btn Inner Circle Button The dark clickable core circle carrying arrow icons. Scales and glows on hover.
