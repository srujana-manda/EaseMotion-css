# Ease Observability Alert Correlation Board

## Overview

A responsive HTML and CSS component for visualizing correlated alerts across infrastructure, applications, databases, and networking.

## Features

- Alert correlation overview
- Root cause tracking
- Investigation status
- Service health indicators
- Responsive layout
- Pure HTML & CSS
- CSS variables and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable observability dashboard component for SRE teams, cloud monitoring platforms, incident management, and enterprise operations centers.
