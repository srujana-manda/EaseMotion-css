# Animated Testimonial Card (`ease-testimonial-cv`)

A responsive testimonial card component built entirely with **HTML5** and **Pure CSS**.

## Features

- Animated testimonial cards
- Floating profile avatar
- Hover lift effect
- Shimmer hover animation
- Star rating section
- Responsive grid layout
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-testimonial-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

Works in all modern browsers.
