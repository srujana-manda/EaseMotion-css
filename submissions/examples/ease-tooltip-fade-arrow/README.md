# Tooltip Fade Arrow

A tooltip with fade-in and arrow indicator — pure CSS.

## Features
- Four directions: top, right, bottom, left
- Arrow indicator using rotated square
- Fade-in animation on hover
- No JavaScript needed

## Usage
```html
<div class="tooltip-wrap">
  <button>Hover</button>
  <span class="tooltip top">Tooltip text<span class="arrow"></span></span>
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
