# CSS Fade-In Dropdown

A lightweight dropdown component built using pure HTML and CSS with a smooth fade-in animation.

## Features

- Pure HTML & CSS
- Smooth Fade-In Animation
- Responsive Layout
- Keyboard Accessible using `:focus-within`
- CSS Custom Properties
- `prefers-reduced-motion` Support
- No JavaScript Required

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

- Supports keyboard navigation with `:focus-within`.
- Respects `prefers-reduced-motion` to reduce or disable animations.
- Fully responsive across desktop, tablet, and mobile devices.
