# Mega Menu Flyout

A hover-triggered mega menu flyout with columns of links, animated via opacity and translateY transition. Uses `.mmf-open` class toggle for visibility.

## Features

- Flyout positioned below nav with smooth fade + slide animation
- `.mmf-open` class controls visibility (opacity + translateY)
- Three-column layout with category headers
- CSS custom properties in `:root` for easy theming
- Google Fonts Inter integration
- Hover highlight on links

## Usage

Add `.mmf-open` to `.mmf-dropdown` to show the flyout. CSS handles opacity, transform, and pointer-events transitions. Columns use flex layout with styled category headings and link hover effects.
