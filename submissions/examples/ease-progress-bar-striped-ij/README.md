# ease-progress-bar-striped

A horizontal progress bar with animated diagonal stripes.

## Features

- Striped fill with sliding animation
- Width controlled via `--progress` CSS custom property (0–100)
- Smooth width transition on value change
- Dark theme with accent purple color

## Usage

Set `--progress` (0–100) on the `.pbs-fill` element to control the progress bar width. Stripes animate continuously.
