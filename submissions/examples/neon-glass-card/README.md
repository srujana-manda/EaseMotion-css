# Neon Glass Card Example

This submission showcases a modern **Glassmorphism Card** enhanced with **EaseMotion CSS** animations. It demonstrates how to combine custom CSS styling (backdrop-filter, gradients) with EaseMotion's utility classes for a polished, interactive UI component.

## ✨ Features
- **Glassmorphism Design**: Uses `backdrop-filter` and semi-transparent backgrounds for a modern frosted-glass look.
- **EaseMotion Animations**: 
  - `ease-fade-in` & `ease-duration-slow`: Smooth, elegant card entrance.
  - `ease-slide-up` & `ease-delay-200`: Staggered content reveal for a premium feel.
  - `ease-hover-glow`: Interactive text feedback on the heading.
  - `ease-hover-lift`: Tactile button feedback on hover.

## 🚀 How to Use
1. Ensure you have the EaseMotion CSS CDN linked in your `<head>` (as shown in `demo.html`).
2. Copy the `glass-card` HTML structure into your project.
3. Include the `style.css` rules for the specific glassmorphism styling, or adapt the CSS variables to match your own theme.

## 📂 Files
- `demo.html`: The standalone, responsive HTML demonstration.
- `style.css`: Custom styling for the glass effect and layout.
