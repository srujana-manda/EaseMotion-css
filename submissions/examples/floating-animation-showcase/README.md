# Floating Animation Showcase

## Description

A reusable floating animation utility that creates a smooth up-and-down motion for UI elements.

## Features

- Smooth infinite floating animation
- Lightweight and reusable
- Suitable for hero sections, icons, cards, badges, and CTA buttons
- Pure HTML and CSS

## Usage

```html
<div class="float">
  🚀 Launch Your Project
</div>
```

```css
.float {
  animation: float 3s ease-in-out infinite;
}
```
