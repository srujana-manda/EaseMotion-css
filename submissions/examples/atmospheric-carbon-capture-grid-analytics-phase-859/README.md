# Atmospheric Carbon Capture Grid Analytics — Phase #859

A clean, data-rich analytics dashboard for global carbon capture facilities. Built entirely with **HTML5**, **CSS3**, and **vanilla JavaScript**. Part of the EaseMotion CSS design system showcase.

---

## Features

- Atmospheric cloud background with drift animation
- Subtle grid overlay toggle
- Live status indicator
- Animated KPI counters with trend badges
- SVG area chart with draw animation
- Facility grid with status-coded map pulses
- Interactive control panel with filters
- Circular progress target ring
- Slide-up alert banner
- Responsive layout
- Reduced motion support
- Zero external dependencies
