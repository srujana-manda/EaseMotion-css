# 🚀 Slide-Out Nav – Slides from Edge with Spring

> A smooth, animated slide-out navigation that springs from the edge with a satisfying bounce effect, perfect for mobile menus and sidebars.

---

## 📖 Description

The **Slide-Out Nav** is a premium navigation component that slides in from the left edge with a spring-like animation on toggle. The navigation panel glides into view with a smooth, bouncy transition, while the main content scales down and shifts to the right, creating a polished, app-like feel. A semi-transparent overlay with backdrop blur dims the background, focusing attention on the navigation.

The component features a clean, dark-styled navigation panel with icons, dividers, and a footer. The hamburger toggle morphs into an X when the navigation is open, and a close button provides an additional way to dismiss the panel. Keyboard shortcuts (Escape key) and overlay clicks also close the navigation, making it accessible and user-friendly.
