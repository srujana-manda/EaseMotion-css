# Filter Dropdown Pop

This example adds a filter dropdown that pops open using native disclosure markup.

## Files

- `demo.html` contains the filter dropdown with checkbox options.
- `style.css` defines the pop animation, summary icon rotation, and reduced-motion fallback.

## Highlights

- Uses native `details` and `summary` elements.
- Keeps the trigger height stable while the menu appears.
- Includes real checkbox controls for filter options.
- Removes menu animation and icon transition for reduced-motion users.
