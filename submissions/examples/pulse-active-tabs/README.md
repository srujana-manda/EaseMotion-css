# Pulse Active Tabs

A pure CSS animated tab component with a smooth pulse-active interaction effect.

## Features

- Pure CSS animation
- No JavaScript dependency
- Responsive layout
- Keyboard accessible focus states
- Custom CSS variables for animation control
- Modern interactive interface design


## Usage

Add the required class names:

```html
<div class="tabs">

<button class="tab active">
Dashboard
</button>

<button class="tab">
Analytics
</button>

</div>
