# Pricing Tier Comparison Motion Board

1. **What does this do?**  
   Demo of Monthly/Yearly SaaS pricing with number crossfades, card lift/sink emphasis, and a one-time Most Popular badge pulse.

2. **How is it used?**  
   Open `demo.html` and use the billing toggle:
   ```html
   <button class="toggle-btn" data-period="yearly">Yearly</button>
   <article class="tier is-popular">...</article>
   ```

3. **Why is it useful?**  
   Pricing tables often swap numbers with no motion feedback. This shows a focused comparison choreography that fits EaseMotion’s readable, animation-first philosophy.
