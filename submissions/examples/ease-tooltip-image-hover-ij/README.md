# Tooltip Image Hover

Avatars with a hover tooltip that shows a larger image and info. Tooltip animates with scale and translateY transition. Background color via `--ti-bg`.

## Features

- Tooltip appears on hover with scale animation
- Background color via `--ti-bg` CSS variable
- Avatar image in tooltip with info card
- Hover scale effect on avatar
- Bounce cubic-bezier transitions

## Usage

Each `.ti-item` uses `--ti-bg` for the avatar background. The `.ti-tooltip` appears on hover with a scale + translate animation.
