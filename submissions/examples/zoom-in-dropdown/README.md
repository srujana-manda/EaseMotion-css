# Zoom-In Dropdown

A lightweight Zoom-In Dropdown component built with pure HTML5 and CSS3 for creative portfolio layouts.

## Features

- Pure HTML5 & CSS3
- Smooth Zoom-In animation
- Responsive design
- Hover & keyboard accessibility (`:focus-within`)
- Modern portfolio UI
- Supports `prefers-reduced-motion`
- No JavaScript required

## Folder Structure

```
zoom-in-dropdown/
├── demo.html
├── style.css
└── README.md
```

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Usage

Open `demo.html` in any modern browser.
