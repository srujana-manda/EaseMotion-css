# ease-slider-range-dual

A dual-range slider with two thumb handles. The track between the handles is colored to show the selected range.

## Features
- Two range input thumbs for min/max selection
- Colored track fill between handles
- Smooth thumb hover scaling
- Real-time value display

## Usage
Drag either thumb to adjust the range. The `--min` and `--max` CSS custom properties on the `.srd-fill` element control the track fill position and width.
