# 📊 Fade-Out Tabs Component Variant (Dashboard Analytics Style)

## 📋 Component Overview

A pure CSS animated **Tabs** component featuring smooth **Fade-Out** interaction transitions, styled for **Dashboard Analytics** interfaces. Data-driven dashboards with metrics, charts, and reports.

## 🛠️ Folder & Structural Architecture

```
submissions/examples/fade-out-tabs-dashboard-analytics-oh/
├── demo.html     # Live preview with dashboard analytics design
├── style.css     # Dashboard Analytics fade-out tab animation styles
└── README.md     # Production integration documentation
```

## 🔧 CSS Custom Properties

| Property | Default | Description |
|----------|---------|-------------|
| `--tab-transition-duration` | `0.4s` | Animation transition duration |
| `--tab-transition-ease` | `cubic-bezier(0.4, 0, 0.2, 1)` | Easing function |
| `--tab-fade-opacity` | `0` | Opacity for inactive tabs |

## 🎯 Animation Variants

### 1. **Fade** - Simple opacity fade
### 2. **Slide** - Fade with vertical slide
### 3. **Scale** - Fade with scale effect

## 🎨 Visual Features

1. **Stats Grid**: KPI cards with value, label, and change indicator
2. **Chart Preview**: Bar chart visualization
3. **Revenue Breakdown**: Progress bars for revenue sources
4. **Metrics List**: Horizontal progress bar metrics
5. **Reports List**: Document cards with status badges
6. **IBM Plex Sans Typography**: Professional dashboard font
