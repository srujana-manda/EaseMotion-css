# Hero Banner Carousel

Hero banner with sliding carousel of featured items. Slides transition with translateX animation via `--hb-current`. Dot navigation and auto-play with pause on hover.

## Features

- Slides transition with translateX via `--hb-current`
- Dot navigation indicators with active widening
- Auto-play every 4 seconds with pause on hover
- Prev/next arrow controls
- Bounce cubic-bezier transition

## Usage

Set `--hb-current` (0-2) on `.hb-track`. CSS handles slide transform. JS controls auto-play and navigation.
