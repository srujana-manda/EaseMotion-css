Gooey liquid physics in pure CSS

  • feGaussianBlur: Blurs adjacent elements together. When two circular blobs slide close to each other, their blurred edges intersect.
  • feColorMatrix: Adjusts the alpha channel threshold. By multiplying alpha levels high and offsetting them back, blurred color overlaps are sharpened into smooth solid curves.
  • CSS transition-delay: Staggers the translation of each blob (0ms, 40ms, 80ms, 120ms) to create an undulating liquid wave effect as the blobs rise.
  • Zero JavaScript: Uses GPU compositor threads to calculate SVG filters and coordinate transitions, ensuring stable 60fps states.
