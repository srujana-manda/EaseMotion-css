# Ease Badge Alert Pulse

## Overview

A reusable notification badge component featuring a smooth pulse animation for alerts, messages, and notifications.

## Features

- Animated pulse effect
- Multiple badge styles
- Responsive layout
- Keyboard accessibility with `:focus-visible`
- Easy to customize

- demo.html
- style.css

- Keyboard-focusable badges
