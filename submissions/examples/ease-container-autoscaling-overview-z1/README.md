# Ease Container Autoscaling Overview

## Overview

A responsive HTML and CSS component for monitoring container autoscaling activity, replica counts, and resource utilization.

## Features

- Replica monitoring
- CPU utilization overview
- Recent scaling activity
- Capacity warning indicators
- Responsive layout
- Pure HTML & CSS
- CSS variables and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable autoscaling dashboard component for Kubernetes HPA, KEDA, cloud-native infrastructure, and DevOps monitoring interfaces.
