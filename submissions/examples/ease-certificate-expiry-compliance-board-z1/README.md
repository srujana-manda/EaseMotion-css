# Ease Certificate Expiry Compliance Board

## Overview

A responsive HTML and CSS component for monitoring SSL/TLS certificate validity, expiration schedules, and renewal status.

## Features

- Certificate expiration tracking
- Renewal reminders
- Issuer information
- Compliance status badges
- Responsive layout
- Pure HTML & CSS
- CSS variables and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable security dashboard component for certificate lifecycle management, DevSecOps workflows, cloud infrastructure, and enterprise compliance monitoring.
