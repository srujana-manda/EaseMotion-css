# Rotate-Click Carousel – Product Catalog

**Issue:** #50468

## Overview

A pure CSS Rotate-Click Carousel designed for Product Catalog layouts. Cards rotate slightly and scale smoothly on hover or keyboard focus, creating an engaging product browsing experience without JavaScript.

## Features

- Pure HTML & CSS
- Rotate-click interaction
- Responsive layout
- Keyboard accessible
- CSS Custom Properties
- No JavaScript required

## Usage

```html
<link rel="stylesheet" href="style.css">

<div class="rotate-carousel">
    <button class="card">
        <h3>Product</h3>
        <p>Description</p>
    </button>
</div>
