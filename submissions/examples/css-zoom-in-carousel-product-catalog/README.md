# CSS Zoom-In Carousel – Product Catalog

A pure CSS animated carousel with a smooth zoom-in interaction designed for product catalog layouts.

## ✨ Features

- Pure CSS
- Smooth zoom-in animation
- Responsive design
- Keyboard accessible
- CSS custom properties
- No JavaScript required

## Usage

```html
<div class="carousel">
  <div class="card active">Product</div>
  <div class="card">Product</div>
</div>
```

## Custom Properties

```css
:root{
  --zoom-scale:1.1;
  --duration:.35s;
  --easing:ease;
}
```

## Technologies

- HTML5
- CSS3

## Why EaseMotion CSS?

This component provides a lightweight, reusable product carousel built entirely with HTML and CSS. It follows EaseMotion CSS principles by being responsive, accessible, customizable, and dependency-free.
