# CSS 3D Flip Carousel

A pure CSS animated carousel with a smooth 3D flip interaction.

## ✨ Features

- Pure CSS animation
- No JavaScript required
- 3D flip transition
- Responsive design
- Keyboard accessible
- Custom CSS variables


## 🚀 Usage

Include the HTML structure and stylesheet:

```html
<div class="carousel">
   <div class="card">
      Content
   </div>
</div>
