# ease-hover-reveal-card

A smooth hover animation card component where an inline background image dims while a customizable content overlay panel slides up from the bottom.

---

## 🤸 Features

- Pure CSS (Zero JavaScript dependencies)
- Supports inline `background-image` styling
- Smooth 0.4s transition animations
- Background image dimming overlay on hover
- Multiple variants: Default Slide-Up, Fade-In, and Partial Peek
- Perfect for portfolio grids, product showcases, and blog previews

---

## 🧼 Usage

```html
<div class="ease-hover-reveal-card" style="background-image:url('img.jpg')">
  <div class="ease-hover-reveal-content">
    <h3>Project Title</h3>
    <p>This content smoothly appears when you hover over the card.</p>
  </div>
</div>
