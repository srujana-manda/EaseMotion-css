# Animated Feature Card Grid (`ease-feature-grid-cv`)

A responsive feature card grid built entirely with **HTML5** and **Pure CSS**.

## Features

- Responsive CSS Grid layout
- Animated feature cards
- Floating icons
- Hover lift animation
- Shimmer hover effect
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-feature-grid-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

Works in all modern browsers.
