# Stratospheric Solar Drone Weather Observatory (Phase 781)

A futuristic single-page UI showcasing an AI-powered stratospheric solar drone observatory built using pure HTML and CSS with EaseMotion CSS animations.

---

## Features

- Responsive single-page layout
- Animated hero section
- Floating background effects
- Drone fleet showcase
- Atmospheric monitoring overview
- Weather observation timeline
- Live mission dashboard
- Sensor modules
- Atmospheric layers section
- Hover animations
- Smooth scrolling
- Pure HTML & CSS (No JavaScript)
