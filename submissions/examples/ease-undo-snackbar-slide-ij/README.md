# ease-undo-snackbar-slide

A snackbar that slides in from the bottom with a message and undo button.

## Features

- Slides up from the bottom with bounce cubic-bezier easing
- Displays a message with an undo action button
- Auto-hides after 4 seconds
- Controlled via `--show` CSS custom property

## Usage

Set `--show: 1` on the `.uss-snackbar` element to slide it in. Set to `0` to dismiss.
