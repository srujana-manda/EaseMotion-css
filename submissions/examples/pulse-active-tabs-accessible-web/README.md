# Pulse Active Tabs - Accessible Web Layouts


A pure CSS animated Pulse-Active Tabs component designed for accessible
web interfaces with smooth active-state transitions.


## Features

- Pure CSS animation
- No JavaScript required
- Keyboard focus support
- Responsive layout
- Accessible ARIA tab roles
- Custom CSS variables


## Usage


```html
<div class="tabs">

<button class="tab active">
Home
</button>

<button class="tab">
Services
</button>

</div>
