# Card Shimmer Loading

A card skeleton with shimmer loading animation — pure CSS.

## Features
- Shimmer sweep animation on gradient background
- Configurable accent color via `--shimmer-color`
- Image placeholder and text line widths
- Reusable card skeleton pattern

## Usage
```html
<div class="shimmer-card">
  <div class="shimmer-img"></div>
  <div class="shimmer-line w-80"></div>
  <div class="shimmer-line w-60"></div>
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
