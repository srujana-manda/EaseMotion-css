# Avatar Pulse Ring

An avatar with pulsing status ring — pure CSS.

## Features
- Avatar with colored status ring (online/offline/default)
- Pulsing animation for online status
- Initials-based avatar display
- Stackable layout

## Usage
```html
<div class="avatar-ring online">
  <div class="avatar-img">JD</div>
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
