# CSS Zoom-In Carousel – Minimalist Tech

A pure CSS animated carousel featuring a smooth zoom-in interaction designed for minimalist technology interfaces.

## ✨ Features

- Pure CSS
- Zoom-in hover animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required

## Usage

```html
<div class="carousel">
  <div class="card">Slide 1</div>
  <div class="card active">Slide 2</div>
  <div class="card">Slide 3</div>
</div>
```

## Custom Properties

```css
:root{
  --zoom-scale:1.1;
  --duration:.35s;
  --easing:cubic-bezier(.22,1,.36,1);
}
```

## Technologies

- HTML5
- CSS3

## Why EaseMotion CSS?

This component provides a clean and reusable zoom-in carousel interaction using only CSS. It aligns with EaseMotion CSS by being lightweight, responsive, accessible, and easily customizable through CSS variables.
