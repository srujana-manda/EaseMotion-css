# Progress Tracker Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be applied to a multi-step progress tracker.

## Components

- Four progress steps
- Completed state
- Active state
- Responsive layout

## Folder Structure

```
submissions/examples/progress-tracker-demo/
├── demo.html
├── style.css
└── README.md
```

## How to Run

1. Open `demo.html` in a browser.
2. Resize the browser window to verify the responsive layout.
3. Observe the EaseMotion CSS animations applied to the heading and steps.

## Features

- Responsive stepper
- Progress line
- Active and completed step styles
- Uses existing EaseMotion CSS animation classes
