# Paper Cut Shadow UI Kit (`.ease-paper-*-xyz`)

A clean, minimal UI kit that uses multi-layered CSS `box-shadow` to create realistic paper-cut depth and lift effects. Inspired by physical paper, folded layers, and soft ambient lighting.

## ✨ Features

- 📄 **Layered Shadows**: 3-tier shadow system (sm, md, lg) for realistic depth.
- 🪶 **Lift-on-Hover**: Elements smoothly rise and deepen their shadow on hover.
- 🎨 **Warm Palette**: Earthy, paper-like colors with a terracotta accent.
- 📐 **Premium Easing**: Custom `cubic-bezier` curves for natural motion.
- ♿ **Accessible**: Respects `prefers-reduced-motion`.
- 📱 **Responsive**: Adapts cleanly to all screen sizes.

## 🧩 Components

- Header with navigation
- Hero card with CTA buttons
- Feature grid cards
- Blockquote card
- Primary and ghost buttons

## 📦 Usage

```html
<div class="ease-paper-card-xyz">
    <div class="ease-paper-card-icon-xyz">🎨</div>
    <h3>Title</h3>
    <p>Description</p>
</div>
