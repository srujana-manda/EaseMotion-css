# Text Gradient Shimmer

Animated gradient text with shimmer sweep effect. Background-position slides to create a shimmer across text.

## Features

- Gradient text with shimmer sweep via `background-position` animation
- Hover pauses the animation
- Pure CSS, no JS required
- Smooth `@keyframes` animation

## Usage

Add class `tgs-title` to a heading. Set `data-text` attribute matching the text content for the shimmer overlay layer.
