# Ease Image Zoom Hover Lens

A simple CSS hover effect that smoothly zooms an image while displaying a circular lens overlay. Built using pure HTML and CSS.

## Features

- Smooth image zoom animation
- Circular hover lens effect
- Responsive layout
- Pure HTML & CSS
- Easy to customize

## Folder Structure

```
ease-image-zoom-hover-lens-iij/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in any modern web browser.
2. Hover over the image to see the zoom lens effect.

## Technologies Used

- HTML5
- CSS3
