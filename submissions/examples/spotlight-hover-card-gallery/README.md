# Spotlight Hover Card Gallery

A modern **CSS-only Spotlight Hover Card Gallery** featuring glowing glassmorphism cards with smooth hover animations.

## Features

- ✨ Pure HTML & CSS
- 💡 Spotlight glow hover effect
- 🚀 Smooth lift animation
- 📱 Fully responsive layout
- 🎨 Glassmorphism-inspired design
- ♻️ Reusable component

## Folder Structure

```
spotlight-hover-card-gallery/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in your browser.
2. Hover over the cards to see the spotlight glow animation.
3. Customize colors, spacing, fonts, and card content as needed.

## Technologies Used

- HTML5
- CSS3
  - Grid
  - Radial Gradients
  - Transitions
  - Backdrop Filter
  - Box Shadows
  - Transform
