# Card Hover Effects Module

A collection of premium, high-interaction card styles for modern UI/UX design.

## Included Styles

| Class | Style | Description |
| :--- | :--- | :--- |
| `.em-card-glass` | Glassmorphism | Uses `backdrop-filter` for a frosted glass look. |
| `.em-card-3d` | 3D Perspective | Uses `perspective` and `rotate` for a depth effect. |
| `.em-card-neon` | Neon Glow | Uses `box-shadow` and `text-shadow` for a glowing effect. |
