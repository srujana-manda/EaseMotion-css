# Ease Message Queue Throughput

## Overview
A CSS-only dashboard component for monitoring message queues, throughput, consumer activity, and processing status.

## Features
- Queue health indicators
- Throughput metrics
- Consumer and backlog information
- Responsive design
- CSS variables
- Hover animations
- Pure HTML & CSS

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` in any modern browser.

## Browser Support
- Chrome
- Firefox
- Edge
- Safari

## Responsive
Optimized for desktop, tablet, and mobile devices.

## Why EaseMotion CSS?
This reusable component provides an animation-first interface for monitoring messaging infrastructure such as RabbitMQ, Kafka, Amazon SQS, Azure Service Bus, and Google Pub/Sub dashboards.
