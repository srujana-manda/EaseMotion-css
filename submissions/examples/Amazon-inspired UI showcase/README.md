🚀 EaseShop – UI Showcase for EaseMotion CSS
📌 Overview

EaseShop is a fictional Amazon-inspired e-commerce UI demo built to showcase the power of EaseMotion CSS in a real-world application layout.

It demonstrates how motion design, micro-interactions, and modern UI patterns can be used together in a production-like e-commerce experience.

✨ Features
🏠 Animated homepage with hero banners & categories
🛍 Interactive product cards (hover, ratings, badges)
❤️ Wishlist animations
🛒 Slide-in shopping cart with live updates
🔍 Search with animated suggestions
📦 Order tracking with progress timeline
👤 User dashboard UI
🔔 Toast notifications
🌙 Dark mode + responsive design
🧪 How to Run

Simply open:

demo.html

No build tools or server required.

💡 Why This Project?

This demo shows how EaseMotion CSS can be used to build:

Real-world e-commerce interfaces
Smooth UI transitions and micro-interactions
Scalable component-based motion design
