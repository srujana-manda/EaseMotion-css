# Swing Hover Tabs – Cyberpunk Neon

## Issue
Closes #50191

## Overview

A responsive pure CSS Swing Hover Tabs component inspired by Cyberpunk Neon interfaces.

## Features

- Pure HTML & CSS
- Swing hover animation
- Responsive layout
- Keyboard accessible
- CSS Custom Properties
- No JavaScript required

## CSS Variables

```css
--duration
--ease
--scale
--rotate
--primary
--secondary
```

## Usage

```html
<link rel="stylesheet" href="style.css">

<div class="tabs">
    <button class="tab active">Home</button>
    <button class="tab">Projects</button>
    <button class="tab">Services</button>
    <button class="tab">Contact</button>
</div>
```

This component follows EaseMotion CSS principles by providing a lightweight, animation-first, reusable, customizable, and accessible UI component using only HTML and CSS.
