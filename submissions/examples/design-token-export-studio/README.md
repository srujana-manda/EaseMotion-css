# Design Token Export Studio

A responsive Design Token Export Studio built with HTML, CSS, and JavaScript.

## Features

- 🎨 Live design token editor
- 👀 Instant UI preview
- 📄 Export CSS `:root` variables
- 📦 Export JSON token map
- 📋 One-click copy for CSS and JSON
- 🔄 Reset to default values
- 📱 Responsive layout

## Tokens

- Primary Color
- Secondary Color
- Border Radius
- Spacing
- Animation Speed

## Files

```
design-token-export-studio/
├── demo.html

├── script.js
└── README.md
```

## Usage

1. Open `demo.html` in a browser.
2. Modify the design tokens.
3. Preview changes instantly.
4. Copy the generated CSS or JSON.

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
