# Avatar Medium Component

A medium-sized avatar component with image, initials, and status variants — pure CSS.

## Features
- Image and initials fallback support
- Online/busy status dot indicator
- Medium size (40x40px)

## Usage
```html
<div class="avatar av-md">
  <img src="..." alt="User" class="av-img">
</div>
<!-- or -->
<div class="avatar av-md">
  <span class="av-initials">JD</span>
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
