# Ease Service Level Objective Tracker

## Overview

A responsive HTML and CSS component for monitoring Service Level Objectives (SLOs), uptime compliance, latency, and error budgets.

## Features

- Uptime target tracking
- Error budget monitoring
- Latency compliance
- Service success rate
- Responsive design
- Pure HTML & CSS
- CSS variables and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

A reusable dashboard component designed for SRE teams, observability platforms, cloud-native infrastructure, and production monitoring interfaces.
