# 3D Card Flip

A 3D card flip component with perspective and backface-visibility — pure CSS.

## Features
- 3D rotation on hover via rotateY(180deg)
- Smooth 0.6s transition
- Dual-face design with backface-visibility: hidden
- Uses perspective on parent container

## Usage
```html
<div class="flip-card-3d">
  <div class="flip-card-inner">
    <div class="flip-card-front">Front</div>
    <div class="flip-card-back">Back</div>
  </div>
</div>
```

## Browser Support
- Chrome 12+, Firefox 10+, Safari 4+

## Tech Stack
- HTML + CSS only, no JavaScript
