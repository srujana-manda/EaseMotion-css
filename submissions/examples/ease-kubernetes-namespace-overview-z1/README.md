# Ease Kubernetes Namespace Overview

## Overview

A responsive HTML and CSS component for monitoring Kubernetes namespaces, workloads, and resource utilization.

## Features

- Namespace overview
- Pod and deployment metrics
- Resource quota monitoring
- Status badges
- Responsive layout
- CSS variables
- Pure HTML & CSS

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable, animation-first component for Kubernetes dashboards, container orchestration platforms, and cloud-native infrastructure monitoring.
