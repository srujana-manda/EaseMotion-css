# Swing-Hover Tabs for Interactive Interface

**Issue:** #50197

## Overview

A responsive Pure CSS Swing-Hover Tabs component designed for Interactive Interface layouts.

## Features

- Pure HTML & CSS
- Smooth Swing-Hover animation
- Responsive
- Keyboard accessible
- No JavaScript
- CSS custom properties

## Files

- demo.html
- style.css

## Usage

```html
<div class="tabs">
    <button class="tab active">Home</button>
    <button class="tab">Services</button>
    <button class="tab">Projects</button>
    <button class="tab">Contact</button>
</div>
```

## Customization

```css
:root{
    --duration:.5s;
    --ease:ease-in-out;
    --angle:12deg;
    --primary:#3b82f6;
}
```
