🎯 Slide-In Drawer Navigation

Pure CSS animated side drawers with multiple variants — no JavaScript required

How It Works

The drawer navigation uses the checkbox hack — a pure CSS technique where a hidden checkbox controls the drawer state. When checked, the drawer slides in with smooth animations. Click the buttons below to try different variants!

✨ Features: Left/Right slide-in • Overlay fade • Push-content variant • Customizable width & duration • Zero JavaScript • Mobile-friendly

📱 Left Slide-In

Classic mobile menu pattern. Drawer slides from the left with an overlay backdrop.

🛒 Right Slide-In

Perfect for shopping carts, notifications, or settings panels. Slides from the right.

📐 Push Content

Main content shifts aside instead of being overlaid. Great for dashboards and apps.

Usage Example

<!-- 1. Hidden checkbox toggle -->
<input type="checkbox" id="drawer-toggle" class="drawer-toggle" />

<!-- 2. Drawer content -->
<div class="drawer-nav drawer-left">
  <nav>
    <a href="#home">Home</a>
    <a href="#about">About</a>
    <a href="#contact">Contact</a>
  </nav>
</div>

<!-- 3. Overlay backdrop (closes drawer when clicked) -->
<label for="drawer-toggle" class="drawer-overlay"></label>

<!-- 4. Trigger button -->
<label for="drawer-toggle" class="drawer-trigger">☰ Menu</label>

Available Classes

  • .drawer-nav — Base drawer container
  • .drawer-left — Slide from left
  • .drawer-right — Slide from right
  • .drawer-push — Push main content aside
  • .drawer-overlay — Backdrop overlay
  • .drawer-toggle — Hidden checkbox control

Push Content Example

This content shifts to the side when the drawer opens, instead of being overlaid. Notice how the entire layout adapts to accommodate the sidebar.

This pattern works great for application dashboards, admin panels, and any interface where the drawer content should feel like part of the main layout rather than a temporary overlay.
