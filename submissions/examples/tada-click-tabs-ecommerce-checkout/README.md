# CSS Tada-Click Tabs for E-Commerce Checkout Layouts

## Overview
A pure CSS animated tab component featuring a Tada-Click effect, designed for e-commerce checkout pages.

## Features

- Pure HTML & CSS
- Responsive layout
- Keyboard accessible
- No JavaScript
- Customizable colors and animation

## Usage

```html
<link rel="stylesheet" href="style.css">
```

Use the HTML from `demo.html`.
