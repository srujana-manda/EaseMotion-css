# Magnetic Button Hover

A premium CSS hover effect that creates a tactile, "magnetic" lift illusion for buttons.

**What does this do?**
Applies a subtle scale and vertical translation on hover, while simultaneously projecting a soft box-shadow in the opposite direction to simulate depth and interactivity.

**How is it used?**
Add the `.ease-magnetic-btn` class to your button component:

```html
<button class="ease-btn ease-magnetic-btn">
  Hover Me
</button>
```

**Why is it useful?**
- **Engaging**: Provides immediate, premium tactile feedback that encourages clicks.
- **Pure CSS**: Achieved entirely through CSS transforms and cubic-bezier transitions.
- **Accessible**: Disables the transform movement if the user prefers reduced motion, falling back to a simple shadow change.
