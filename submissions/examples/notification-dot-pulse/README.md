# Notification Dot Pulse

This example adds a notification button with an unread dot that pulses gently.

## Files

- `demo.html` contains the notification button markup.
- `style.css` defines the unread dot, pulse animation, focus state, and reduced-motion fallback.

## Highlights

- Uses a real button for the notification control.
- Keeps the button layout stable while the dot pulses.
- Adds visible keyboard focus styling.
- Hides the pulse animation for reduced-motion users while keeping the dot visible.
