# Skew Active Carousel

## Overview
The Skew Active Carousel is a responsive dashboard-style component built entirely with HTML and CSS. It highlights the selected analytics card using a skew-to-flat transition and scaling effect, creating a clean and engaging user interface without relying on JavaScript.


## Features
- Pure HTML and CSS implementation
- Responsive layout for different screen sizes
- Smooth skew-active animation
- Keyboard-accessible using radio inputs
- Lightweight and easy to customize
- Suitable for dashboard and analytics interfaces


## Project Structure
skew-active-carousel-as/
├── demo.html

## Usage
1. Open `demo.html` in any modern web browser.
2. Click on any analytics card to make it active.
3. Modify the CSS values to customize the appearance and animation.

```html
<link rel="stylesheet" href="style.css">
```


## Customization
You can easily customize:
- Card colors
- Border radius
- Animation duration
- Scale effect
- Skew angle
- Spacing between cards


## Browser Compatibility
This component works in all modern browsers including:
- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari


## Why is it useful?
This component provides an attractive analytics dashboard layout with smooth visual transitions while keeping the implementation lightweight, responsive, and free from JavaScript dependencies. It aligns with EaseMotion CSS by demonstrating motion-focused UI design that is simple to integrate and customize.
