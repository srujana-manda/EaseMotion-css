# Neon Glow Text Utility

## Description

A reusable neon glow text effect built with pure CSS using text-shadow and keyframe animation.

## Features

- Animated neon glow
- Pure HTML & CSS
- Lightweight
- Reusable utility

## Usage

```html
<h1 class="neon-glow">Neon Glow Text</h1>
```
