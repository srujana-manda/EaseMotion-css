# CSS Rotate-Click Carousel – Accessible Web

A responsive **Rotate-Click Carousel** built using pure HTML and CSS with accessibility-focused interactions.

## Features

- Pure HTML & CSS
- Rotate-click animation
- Keyboard accessible
- Focus-visible support
- Responsive design
- CSS custom properties
- No JavaScript

## CSS Variables

```css
:root{
  --rotate:7deg;
  --scale:1.04;
  --duration:.35s;
}
```

## Folder Structure

```
rotate-click-carousel-accessible-web/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser. Use the **Tab** key to navigate between cards and **Enter/Space** to interact. Customize animation, colors, and spacing through CSS variables.
