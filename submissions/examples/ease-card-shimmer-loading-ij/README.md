# Card Shimmer Loading

Card skeleton with shimmer animation for loading state. Pulsing gradient sweep across placeholder elements.

## Features

- Card skeleton placeholder
- Shimmer gradient sweep via @keyframes
- Pulsing animation on image and lines
- Pure CSS, no JS required

## Usage

Structure `.csl-card` with `.csl-img` and `.csl-line` elements. Shimmer animation runs automatically.
