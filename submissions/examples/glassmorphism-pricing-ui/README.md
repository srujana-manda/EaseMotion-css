# 🔮 GlassMorph Pricing UI Kit

> A modern glassmorphism pricing section featuring animated pricing cards, comparison tables, testimonials, FAQs, and responsive layouts built with pure HTML and CSS.

The GlassMorph Pricing UI Kit is designed to give your SaaS product a premium, futuristic aesthetic without the overhead of heavy JavaScript frameworks. Everything from the smooth micro-interactions to the functional monthly/yearly billing toggle is powered entirely by CSS.

## ✨ Features

- **Zero JavaScript:** Built entirely with pure HTML5 and modern CSS3.
- **Functional Billing Toggle:** Uses the CSS Checkbox Hack (`:checked` + `~` sibling selector) to instantly switch between Monthly and Yearly pricing without a single line of JS.
- **Glassmorphism Design System:** Soft glowing shadows, frosted glass panels (`backdrop-filter`), and floating aurora background blobs.
- **EaseMotion Animation Library:** Includes a robust suite of reusable CSS animations (`.float`, `.pulse`, `.slide-up`, `.hover-glow`, `.card-lift`, and more).
- **Fully Responsive:** Adapts seamlessly from desktop layouts to tablet (2-column) and mobile (1-column) views.
- **Accessible:** Built with semantic HTML (including native `<details>` and `<summary>` for FAQs), proper keyboard focus states, and respect for `prefers-reduced-motion`.

## 📦 What's Inside?

A complete, conversion-optimized pricing flow:
1. **Hero Section:** High-impact headline with animated background elements.
2. **Billing Toggle:** Interactive switch for Monthly/Yearly pricing.
3. **Pricing Cards:** 6 distinct tiers, including a highlighted "Pro/Popular" card.
4. **Feature Comparison Table:** Detailed, scrollable matrix for deep-dive feature matching.
5. **Animation Showcase:** Live preview of the EaseMotion CSS utility classes.
6. **Testimonials & Social Proof:** Glass-styled cards for user reviews and trusted company logos.
7. **FAQ:** Expandable accordion built natively with HTML.
8. **Final CTA & Footer:** Newsletter signup and clean footer navigation.
