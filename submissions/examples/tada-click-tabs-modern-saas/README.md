# Tada-Click Tabs for Modern SaaS

**Issue:** #50198

## Overview

A responsive Pure CSS **Tada-Click Tabs** component designed for Modern SaaS interfaces.

## Features

- Pure HTML & CSS
- Responsive
- Keyboard accessible
- No JavaScript
- CSS custom properties
- Smooth Tada-Click animation

## Files

- demo.html
- style.css

## Usage

```html
<div class="tabs">
    <button class="tab active">Dashboard</button>
    <button class="tab">Features</button>
    <button class="tab">Pricing</button>
    <button class="tab">Support</button>
</div>
```

## Customization

```css
:root{
    --duration:.55s;
    --ease:ease;
    --scale:1.08;
    --primary:#4f46e5;
}
```
