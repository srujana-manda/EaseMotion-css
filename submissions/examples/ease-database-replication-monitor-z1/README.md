# Ease Database Replication Monitor

## Overview
A CSS-only dashboard component for monitoring database replication status.

## Features
- Replication health cards
- Animated sync status badges
- Responsive layout
- CSS variables
- Pure HTML & CSS

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` directly in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari

## Responsive
Optimized for desktop, tablet, and mobile devices.

## Why EaseMotion CSS?
Provides a reusable animation-first component for monitoring replication health across PostgreSQL, MySQL, MongoDB, Redis, and distributed database environments.
