# Ease Object Storage Bucket Overview

## Overview
A CSS-only dashboard component for visualizing object storage buckets and their operational status.

## Features
- Bucket overview cards
- Storage usage details
- Lifecycle policy display
- Security status badges
- Responsive layout
- Pure HTML & CSS

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` directly in any modern browser.

## Browser Support
- Chrome
- Firefox
- Edge
- Safari

## Responsive
Optimized for desktop, tablet, and mobile devices.

## Why EaseMotion CSS?
Provides a reusable animation-first component for cloud storage dashboards, infrastructure portals, and object storage management interfaces.
