# Button Interaction Module

A collection of highly interactive, tactile button styles designed to improve user engagement through micro-interactions.

## Included Button Styles

| Class | Style | Best Use Case |
| :--- | :--- | :--- |
| `.em-btn-glow` | Magnetic Glow | Primary Call-to-Actions (CTAs). |
| `.em-btn-liquid` | Liquid Fill | Secondary buttons or outline buttons. |
| `.em-btn-neu` | Neumorphic | Soft UI, dashboard controls, or settings. |
| `.em-btn-trace` | Border Trace | Premium features or "Dark Mode" UI. |

## Implementation

### 1. Magnetic Glow
Provides a lifting effect and a soft shadow on hover to indicate depth.
