# Text Typing Cursor

Text types out character by character with a blinking cursor. JavaScript increments revealed characters, and `--progress` (0–100) controls character reveal width.

## Features
- Monospace typing animation with blinking cursor
- `--progress` custom property for character reveal
- JS-driven typing with configurable speed
- Slider to manually control reveal progress

## Control
Click "Start Typing" to animate the typing effect. Use the slider to manually control progress.
