# Unfold Enter Effect

## What does this do?

Creates a smooth 3D unfold entrance animation for cards, panels, dialogs, or other UI containers using only HTML and CSS.

---

## How is it used?

```html
<section class="uge-unfold-panel">
    ...
</section>
```

Link the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

---

## Why is it useful?

This effect provides an elegant entrance animation by combining perspective, rotation, scaling, opacity, and blur. It helps draw attention to important UI elements while maintaining a clean and modern user experience.
