# Ease Service Dependency Impact Map

## Overview

A responsive HTML and CSS component for monitoring service dependencies, upstream health, downstream impact, and application availability.

## Features

- Upstream dependency monitoring
- Downstream impact tracking
- Service health indicators
- Incident propagation overview
- Responsive layout
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable dashboard component for microservice dependency visualization, SRE monitoring, incident management, and cloud-native operations.
