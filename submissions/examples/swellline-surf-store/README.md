# SwellLine — Surf Culture E-Commerce

A full-page, fully responsive e-commerce landing page designed for a high-quality surf gear and lifestyle brand. Built exclusively utilizing the `EaseMotion` CSS utility class architecture to deliver a relaxed, oceanic, and adventure-driven aesthetic.

## 🌊 Theme & Aesthetic
**Ocean Surf Lifestyle:**
* **Colors:** Deep Navy Slate (`#1A2E35`), Wave Blue (`#0087B2`), Sand/Foam Background (`#F4F7F6`), and Pure White (`#FFFFFF`).
* **Typography:** `Outfit` (a geometric, modern sans-serif providing a sporty/active feel) paired with `DM Sans` (a highly legible, friendly sans-serif for body copy).
* **Vibe:** Adventurous, high-quality, laid-back, and authentic.

## 🚀 How to Run
This template requires **zero build steps**. 
1. Clone or download the repository.
2. Ensure `demo.html` and `style.css` are in the same directory.
3. Open `demo.html` directly in your favorite web browser.

## 🛠️ EaseMotion Classes Showcased
This project relies entirely on the core EaseMotion CSS utility framework to build complex grids and smooth micro-interactions.

### Layout & Structure
* `ease-container`
* `ease-flex`, `ease-grid`
* `ease-grid-cols-2`, `ease-grid-cols-3`
* `ease-gap-4`, `ease-gap-6`, `ease-gap-8`, `ease-gap-12`
* `ease-items-center`, `ease-justify-between`, `ease-justify-center`
* `ease-relative`

### Theming Utilities
* `ease-bg-main`, `ease-bg-surface`, `ease-bg-primary`, `ease-bg-light`
* `ease-text-main`, `ease-text-primary`, `ease-text-white`, `ease-text-muted`
* `ease-border`, `ease-border-primary`, `ease-border-y`
* `ease-font-bold`, `ease-font-display`

### Animations & Scroll Reveals
* `ease-reveal`, `ease-reveal-up`, `ease-reveal-scale` (Intersection Observer triggered for smooth scroll-in)
* `ease-fade-in`, `ease-slide-up`, `ease-bounce-in` (Initial page load transitions)
* `ease-delay-100` through `ease-delay-500` (Staggered element loading)

### Micro-Interactions (Hover)
* `ease-hover-lift` (Y-axis translation + glowing shadow expansion for gear cards)
* `ease-hover-scale` (Image zoom effect applied to team rider portraits)
* `ease-hover-fade` (Subtle opacity shift on logos)
* `ease-hover-underline` (Animated bottom border on navigation links)

## 📱 Responsiveness
The layout adapts smoothly for different viewports. The 3-column gear grids and team rider profiles gracefully collapse to vertical stacks on mobile devices to ensure a seamless shopping experience. Typography utilizes `clamp()` to scale heading sizes perfectly across screens.
