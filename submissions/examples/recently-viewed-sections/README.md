# Recently Viewed Sections

A timeline-style recently viewed sections component with staggered reveal — pure CSS.

## Features
- Staggered slide-in animation
- Timeline connector line
- Reusable timeline item pattern

## Usage
```html
<div class="timeline">
  <div class="item" style="--d:0s">
    <div class="dot"></div>
    <div class="content"><h3>Title</h3><p>Time</p></div>
  </div>
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
