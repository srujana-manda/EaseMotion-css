# CSS Zoom-In Carousel – Glassmorphism UI

A pure CSS animated carousel featuring a smooth zoom-in interaction with a modern Glassmorphism UI design.

## ✨ Features

- Pure CSS
- Glassmorphism styling
- Smooth zoom-in animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required

## Usage

```html
<div class="carousel">
  <div class="card active">Dashboard</div>
  <div class="card">Analytics</div>
  <div class="card">Settings</div>
</div>
```

## Custom Properties

```css
:root{
  --zoom-scale:1.1;
  --duration:.35s;
  --easing:cubic-bezier(.22,1,.36,1);
}
```

## Technologies

- HTML5
- CSS3

## Why EaseMotion CSS?

This component delivers a reusable Glassmorphism-inspired carousel using only HTML and CSS. It follows the EaseMotion CSS philosophy by being lightweight, responsive, accessible, and fully customizable through CSS variables.
