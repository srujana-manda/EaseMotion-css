Scroll Progress Tracker Ring

A pure CSS scroll-linked progress tracker. Scroll down the page and observe the circular ring filling in the bottom-right corner.

1. The Power of Native Scroll Timelines

By leveraging CSS Scroll-Driven Animations, browsers can now bind visual transforms and SVG stroke offsets directly to the viewport scroll position. This removes layout-thrashing JavaScript scroll listeners, keeping page scrolls perfectly butter-smooth at 120Hz.

Traditional trackers require executing scroll handlers on the main thread, which blocks rendering and creates micro-stutters during heavy GPU graphics paint sweeps.

2. SVG Path Vector Calculations

This ring indicator uses a geometric circle path of radius exactly 15.9155. Because $2 \times \pi \times 15.9155 \approx 100$, the perimeter matches exactly 100 units.

This allows setting the `stroke-dasharray` and `stroke-dashoffset` parameters directly to percentages from 0 to 100, which coordinates with standard zero-to-one scroll timeline properties.

3. Modern Browser Support Fallbacks

When browsers do not support CSS scroll timelines, the indicator falls back gracefully. We also support accessibility queries, freezing motion properties when prefers-reduced-motion triggers to avoid disorienting visitors who prefer static layouts.
