New Component

ease-blur-overlay-modal

A premium, frosted-glass modal popup component. Uses CSS custom properties for backdrop filters, slide spring cards, and close spins with zero JavaScript.

Interactive Triggers

Click the buttons below to trigger modals with different blur levels and durations. Clicking the "X" button, the primary/secondary actions, or the background frosted space will dismiss the modal.

Installation & Usage

Include the base checkbox state, container, click-backdrop, and modal card elements in your layout structure:

<!-- Checkbox state controller -->
<input type="checkbox" id="modal-id" class="ease-modal-state">

<!-- Outer popup wrapper -->
<div class="ease-modal-container">
  <!-- Backdrop label that closes on click -->
  <label for="modal-id" class="ease-modal-backdrop"></label>

  <!-- Frosted content card -->
  <div class="ease-modal-card">
    <!-- Close label -->
    <label for="modal-id" class="ease-modal-close">
      <!-- Cross SVG -->
    </label>

    <!-- Header, Body & Actions -->
  </div>
</div>

Modifier Classes

Class Suffix Target Element Description
.ease-modal-state <input type="checkbox"> Hidden toggle checkbox state controller. Must precede modal.
.ease-modal-container Container Base container. Handles opacity transitions and flex setups.
.ease-modal-backdrop Label Sibling Span-fixed overlay layer. Transitioning backdrop-filter blurs.
.ease-modal-container--subtle Container Mod Decreases the backdrop-filter blur level to 4px.
.ease-modal-container--deep Container Mod Increases the backdrop-filter blur level to 24px.
.ease-modal-container--slow Container Mod Slows down transition speed to 0.75s.
.ease-modal-container--fast Container Mod Speeds up transition to 0.25s.

CSS Custom Customization Variables

Design Token Default Value Description
--ease-modal-duration 0.4s Length of time for transitions on scale, slide, and overlay.
--ease-modal-ease cubic-bezier(0.34, 1.56, 0.64, 1) Custom spring overshoot curve for entry cards.
--ease-modal-backdrop-blur 10px Frosted glass backdrop-filter intensity.
--ease-modal-radius 20px Border-radius applied to card corners.
