# Corner Border Draw Hover Effect

1. What does this do? Introduces a hover effect where thin borders animate simultaneously from the top-left and bottom-right corners of an element, meeting to form a complete outline.
2. How is it used? Apply the `.corner-border-hover` class to any relatively positioned element (e.g., button, card, link):

   ```html
   <button class="corner-border-hover">Hover Me</button>
   ```

3. Why is it useful? It provides a lightweight, pure CSS hover animation for interactive elements that aligns with EaseMotion CSS's animation-first philosophy, utilizing `currentColor` for high composability without breaking the element layout.
