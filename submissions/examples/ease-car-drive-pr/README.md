# ease-car-drive

A smooth and reusable CSS animation that showcases a car driving across a road with animated wheels, moving road lines, background trees, and exhaust smoke effects.

---

## ✨ Features

* 🚗 Smooth car driving animation
* 🛞 Rotating wheels
* 🛣️ Animated moving road lines
* 🌳 Scrolling background trees
* 💨 Exhaust smoke effect
* 📱 Responsive design
* ♿ Accessibility support with reduced motion preferences

---

## 📂 Files

```text
demo.html   → Demonstrates the animation
style.css   → Contains all animation styles and keyframes
README.md   → Documentation and usage guide
```

## How does a developer use it?

```html
<div class="ease-car-drive">
  ...
</div>
```

## Why does it fit EaseMotion CSS?

This animation follows EaseMotion CSS principles by using:

* Human-readable class names
* Self-contained implementation
* Smooth reusable animations
* Easy integration into web projects
