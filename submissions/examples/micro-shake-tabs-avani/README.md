# Micro Shake Tabs (#50164)

## Engineering Overview
A creative portfolio-focused tab component built with pure CSS. It integrates a subtle "Micro Shake" animation keyframe loop on hover to inject personality and engagement into creative user interfaces.

## Key Technical Features
- **Keyframe Precision:** Uses a lightweight multi-step `@keyframes micro-shake` rule combining translation and slight rotation.
- **Hardware Acceleration:** Relies entirely on layout-friendly `transform` properties for high-performance 60FPS rendering.
- **Zero JS:** Fully interactive using native CSS pseudo-classes.
- **Modularity:** Exposes variables for animation duration (`--shake-speed`) and accent colors.
