# Stepper Dot Wave

This example adds a compact setup stepper with an active dot wave animation.

## Files

- `demo.html` contains the setup stepper markup.
- `style.css` defines the step states, active dot wave, responsive layout, and reduced-motion fallback.

## Highlights

- Uses an ordered list for step semantics.
- Keeps each step at a stable height while the active dot pulses.
- Shows done, active, and pending states.
- Disables the dot wave when reduced motion is requested.
