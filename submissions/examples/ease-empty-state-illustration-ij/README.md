# Empty State Illustration

A centered empty-state card featuring a CSS-drawn folder icon with a gentle float/bob animation and a call-to-action button with a hover glow effect.

## Features
- Pure CSS folder illustration (no images)
- Smooth `@keyframes float` animation using `translateY`
- Button hover scale and glow transition
- Clean centered layout with Inter font

## Usage
Copy `style.css` and link it in your HTML. Wrap your empty state content in a `.card` container with the `.illustration` and `.btn` markup as shown in `demo.html`.
