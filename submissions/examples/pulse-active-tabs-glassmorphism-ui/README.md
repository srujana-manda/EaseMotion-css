# Pulse-Active Tabs – Glassmorphism UI

**Issue:** #50173

## Overview

A pure CSS Pulse-Active Tabs component with a modern Glassmorphism design, smooth pulse animation, and responsive layout.

## Features

- Pure CSS
- Glassmorphism effect
- Pulse-active animation
- Keyboard accessible
- Responsive
- CSS custom properties
- No JavaScript

## Files

```
demo.html
style.css
README.md
```

## Usage

```html
<link rel="stylesheet" href="style.css">

<div class="tabs">
    <button class="tab active">Home</button>
    <button class="tab">Features</button>
    <button class="tab">Pricing</button>
    <button class="tab">Contact</button>
</div>
```

## Customization

Edit the CSS variables:

```css
:root{
    --primary:#38bdf8;
    --duration:.35s;
    --radius:16px;
}
```

## Why EaseMotion CSS?

- Lightweight
- Animation-first
- Human-readable classes
- Easy to customize
- Pure CSS solution
