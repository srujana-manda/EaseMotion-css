# Zero‑Trust Holographic Security Operations Center Showcase

A responsive, interactive dashboard UI for a zero‑trust holographic security operations center. Built with **EaseMotion CSS** utility classes and minimal custom CSS. Zero JavaScript.

## Sections
- **Header** — navigation with hover effect
- **KPI Row** — active threats, trust score, policies active, events today
- **Holographic Threat Map** — placeholder visualization with live badge
- **Access Controls** — verification level, session timeout, MFA strictness sliders + apply button
- **Alert Log** — recent events

## EaseMotion Classes Showcased
- **Layout:** `ease-container`, `ease-flex`, `ease-grid`, `ease-gap-*`, `ease-justify-between`, `ease-items-center`, `ease-min-h-screen`, `ease-py-*`, `ease-px-*`, `ease-mb-*`, `ease-mt-*`, `ease-h-64`, `ease-lg-col-span-2`
- **Background:** `ease-bg-gray-900`, `ease-bg-gray-800`, `ease-bg-cyan-900`
- **Typography:** `ease-text-*`, `ease-font-bold`, `ease-font-semibold`, `ease-text-gray-*`, `ease-text-cyan-*`, `ease-text-green-*`, `ease-text-purple-*`, `ease-text-yellow-*`, `ease-text-white`
- **Components:** `ease-card`, `ease-btn`, `ease-btn-primary`, `ease-badge`
- **Hover Effects:** `ease-hover-scale-105`, `ease-hover-text-cyan-300`
- **Animations:** `ease-fade-in`, `ease-delay-*`, `ease-transition`
- **Utilities:** `ease-rounded-lg`, `ease-w-full`, `ease-block`, `ease-uppercase`, `ease-space-y-4`

## How to Use
1. Open `demo.html` in any modern browser.
2. Adjust the sliders and observe the dashboard.
