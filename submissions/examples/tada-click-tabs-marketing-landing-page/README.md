# CSS Tada-Click Tabs for Marketing Landing Page Layouts

## Overview

A pure CSS animated tab component featuring a smooth **Tada-Click** interaction designed for Marketing Landing Pages.

## Features

- Pure HTML & CSS
- Responsive design
- Keyboard accessible
- No JavaScript required
- Easy customization using CSS

## Usage

```html
<link rel="stylesheet" href="style.css">
```

Open `demo.html` directly in your browser.
