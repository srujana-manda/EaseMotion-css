# Animated Gradient Border Hover Effect

A reusable **CSS-only hover animation** that creates a beautiful animated gradient border around cards, buttons, pricing sections, feature boxes, and containers.

This example is designed for the **EaseMotion CSS** project and demonstrates how modern CSS can be used to build attractive UI components without JavaScript.

---

# 📖 Overview

The Animated Gradient Border Hover Effect provides a lightweight and reusable solution for adding interactive hover animations to modern web interfaces.

When a user hovers over an element, an animated gradient border appears around it while a soft glow effect enhances the visual experience.

The component is built entirely using HTML and CSS, making it easy to integrate into any website.

# ⚙ CSS Variables

The component uses CSS variables for customization.

```css
:root{

--bg-color:#0f172a;

--card-bg:#1e293b;

--gradient-1:#00d4ff;

--gradient-2:#8b5cf6;

--gradient-3:#ff3cac;

--gradient-4:#00f5a0;

--border-radius:18px;

--border-width:3px;

--animation-speed:5s;

}
```

# 🎯 Usage

Include the stylesheet.

```html
<link rel="stylesheet" href="style.css">
```

Create a card.

```html
<article class="gradient-card">

<span class="card-glow"></span>

<h2>Premium Card</h2>

<p>
Animated Border Hover Effect
</p>

<button class="btn">
Explore →
</button>

</article>
```
