# CSS Breadcrumb Navigation Component

A modern **CSS Breadcrumb Navigation Component** built using **HTML5** and **CSS3** for the EaseMotion CSS examples collection.

---

## ✨ Overview

This component demonstrates elegant breadcrumb navigation patterns with multiple visual styles, smooth hover animations, responsive layouts, and accessibility-friendly markup. It is designed as a reusable navigation component for modern websites and web applications.

---

## 🚀 Features

- 🏠 Basic breadcrumb navigation
- ➜ Chevron separator variant
- ➡️ Arrow separator variant
- 🔵 Rounded pill breadcrumb
- 🎨 Filled breadcrumb style
- 🪟 Glassmorphism variant
- ✨ Smooth hover animations
- 📍 Active page highlighting
- 📱 Fully responsive design
- 🌙 Dark mode support
- ♿ Reduced motion support
- 🎯 Semantic HTML structure

## 📱 Responsive Design

The component is optimized for:

- Desktop
- Tablet
- Mobile

Long breadcrumb paths automatically wrap to new lines, ensuring readability on smaller screens.

---

## ♿ Accessibility

The component follows accessibility best practices:

- Semantic `<nav>` element
- `aria-label="Breadcrumb"` support
- Keyboard-accessible links
- Visible focus states
- High-contrast colors
- `prefers-color-scheme` support
- `prefers-reduced-motion` support
