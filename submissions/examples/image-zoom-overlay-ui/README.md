# Image Zoom Hover VR

## 📌 Description
This feature demonstrates a smooth image zoom interaction on hover with an overlay indicator. It improves user experience in image previews, galleries, and product displays.

---

## ✨ Features
- Image scales smoothly on hover
- Overlay appears with zoom hint icon
- Cursor changes to zoom-in on hover
- Clean and responsive card UI
- Smooth CSS transitions for better UX

---

## 🧠 How it works
- `transform: scale()` is used to zoom the image on hover
- `transition` ensures smooth animation
- `position: absolute` is used for overlay display
- Overlay opacity changes on hover for visibility effect
- `cursor: zoom-in` indicates interactivity

---

## 🧩 Files Included
- `demo.html` → Main structure

## 🚀 How to Run
1. Open the folder `image-zoom-hover-vr`
2. Open `demo.html` in any browser
3. Hover over the image to see the effect
