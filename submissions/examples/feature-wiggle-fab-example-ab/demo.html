Wiggle FAB Example

A floating action button that enters with a playful wiggle, then settles into a calm hover/focus state. Built for issue #50779.

Live playground

Use the controls below to replay the entrance wiggle, change intensity, or try an extended FAB. The fixed FAB in the corner mirrors the selected variant.

Intensity maps to CSS custom properties --wiggle-fab-rotate-ab and --wiggle-fab-nudge-ab.

Variant gallery

Each button runs the entrance wiggle once on load. Hover or focus to see a lighter attention wiggle.

How to use

<button
  type="button"
  class="wiggle-fab-ab wiggle-fab-primary-ab is-wiggling-ab"
  aria-label="Compose new note"
>
  <span class="wiggle-fab-icon-ab" aria-hidden="true">+</span>
  <span class="wiggle-fab-label-ab">Compose</span>
</button>
  1. Add wiggle-fab-ab plus a color variant class.
  2. Add is-wiggling-ab to play the entrance animation.
  3. Remove the class after animationend if you need to replay it.
  4. Always provide a clear aria-label when the icon alone is not enough.

Accessibility notes

  • Uses a real <button> with visible focus rings.
  • Icon-only FABs include descriptive aria-label text.
  • Under prefers-reduced-motion: reduce, the wiggle is disabled and the FAB appears with a soft fade only.
  • Hover wiggle is limited to fine pointers; touch users keep a simple press scale.
