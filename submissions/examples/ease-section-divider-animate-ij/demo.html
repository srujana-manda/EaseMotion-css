Section Divider Animate

Animated section dividers that add visual rhythm between content blocks. Pure CSS animations — gradient sweeps, dot patterns, and flowing waves.

Gradient Sweep

A smooth multi-color gradient endlessly sweeps across the divider. Customize colors, height, and animation speed via CSS custom properties.

Dot Pattern

Dots march horizontally in an endless animated stream. Adjust dot size, color, and spacing to match your brand.

Dense Dots

A tighter dot pattern for a finer-grained animated divider. Play with density to create the perfect rhythm.

Wavy Line

A flowing sinusoidal wave glides across the divider. The organic motion adds a playful, energetic feel to page transitions.

Alternate Wave

Swap the wave color with a modifier class to match different sections or themes.

All CSS, No JS

Every divider animates automatically on page load using only CSS @keyframes. Drop in an <hr class="sda-divider sda-(style)"> and it just works.
