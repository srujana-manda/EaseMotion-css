# ease-modal-zoom-in

Modal dialog with overlay backdrop. On open, the modal zooms in from scale(0.8) to scale(1) with a smooth transition and the backdrop fades in.

## Features

- Semi-transparent overlay backdrop with opacity transition
- Modal scales in from 0.8 to 1.0 for a zoom effect
- Close via X button, Cancel button, or clicking outside
- Clean dark-themed header, body, and footer sections

## CSS Custom Property Control

- `--open`: Set to `1` to show the modal (backdrop fades in, modal scales up), `0` to hide. Controlled by the Open/Close buttons and overlay click.
