# Slide-Up Carousel – Modern SaaS Layout

## Issue

Closes #50419

## Description

A responsive pure CSS Slide-Up Carousel featuring smooth upward animations suitable for Modern SaaS landing pages.

## Features

- Pure CSS
- Responsive
- Keyboard Accessible
- Customizable CSS Variables
- No JavaScript Required

## Files

- demo.html
- style.css
- README.md

## Usage

```html
<div class="carousel">
    <div class="card">
        ...
    </div>
</div>
```

Customize using CSS variables.

```css
:root{
    --duration:.45s;
    --overlay:#2563eb;
    --radius:16px;
}
```
