# Blur Entrance Carousel

Issue: #50514

## What does this add?

A pure CSS Blur-Entrance Carousel component designed for SaaS Showcase layouts.

It creates smooth blur, fade, and movement animations without JavaScript.


## Features

- Pure CSS animation
- Responsive layout
- SaaS showcase friendly design
- Smooth blur entrance effect
- Hover interaction
- Custom CSS variables


## Usage

```html
<div class="blur-carousel">

  <article class="blur-card">
    Content here
  </article>

</div>
