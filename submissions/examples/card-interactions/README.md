# Card Interaction Effects

A set of hover and continuous animations for UI cards.

## Effects

| Class | Type | Description |
|---|---|---|
| `.em-card-lift` | Hover | Lifts the card and adds a deep shadow. |
| `.em-card-glow` | Hover | Adds a neon border and glow effect. |
| `.em-card-float` | Continuous | A smooth floating animation for circular elements. |

## How to use
Add the `style.css` to your project and apply the classes to your `div` elements.
