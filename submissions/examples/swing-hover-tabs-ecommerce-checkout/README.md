# Swing Hover Tabs - E-Commerce Checkout

## Overview

A pure CSS animated checkout tab component created for E-Commerce interfaces.

The component provides a smooth Swing-Hover interaction without JavaScript.

## Features

- Pure HTML and CSS
- Smooth swing hover animation
- Responsive layout
- Keyboard focus support
- Custom CSS variables
- Zero JavaScript dependency
