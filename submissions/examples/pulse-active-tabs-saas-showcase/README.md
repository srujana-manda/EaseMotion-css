# Pulse-Active Tabs – SaaS Showcase Layout

**Issue:** #50172

## Overview

A pure CSS Pulse-Active Tabs component designed for modern SaaS Showcase layouts with smooth pulse animation, responsive design, and keyboard accessibility.

## Features

- Pure CSS
- Pulse-active animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Lightweight and reusable
- No JavaScript

## Files

```
demo.html
style.css
README.md
```

## Usage

```html
<link rel="stylesheet" href="style.css">

<div class="tabs">
    <button class="tab active">Overview</button>
    <button class="tab">Features</button>
    <button class="tab">Pricing</button>
    <button class="tab">Integrations</button>
</div>
```

## Customization

```css
:root{
  --primary:#4f46e5;
  --duration:.35s;
  --radius:12px;
}
```

## Why EaseMotion CSS?

- Pure CSS implementation
- Human-readable class names
- Animation-first design
- Easy customization using CSS variables
- Responsive and accessible
