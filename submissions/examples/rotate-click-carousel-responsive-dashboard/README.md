# CSS Rotate-Click Carousel – Responsive Dashboard

A responsive dashboard carousel built with pure HTML and CSS featuring rotate-click interactions for dashboard statistic cards.

## Features

- Pure HTML & CSS
- Rotate-click animation
- Responsive dashboard cards
- Keyboard accessible
- CSS custom properties
- No JavaScript required

## CSS Variables

```css
:root{
  --rotate:6deg;
  --duration:.35s;
}
```

## Folder Structure

```
rotate-click-carousel-responsive-dashboard/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in a browser. Hover, click, or focus on cards to view the rotate animation. Customize colors, rotation, spacing, and timing using CSS variables.
