💎 Card Utility

Glassmorphism Hover Card Utilities

Five reusable CSS utility classes that add premium frosted-glass styling and interactive hover effects to any element — no JavaScript required.

Base Glass Card .ease-glass-card

The foundation class — applies a semi-transparent background, subtle border, backdrop blur, and rounded corners for a premium frosted-glass appearance.

🪟

Glass Card

Frosted glass foundation.

Hover Interaction .ease-glass-hover

Smooth hover transition — slightly increases background opacity and applies a subtle scale. No layout shift; compositing stays on the GPU. Hover the card below.

✨

Hover Effect

Smooth opacity & scale.

Lift Effect .ease-glass-lift

On hover the card lifts upward with an enhanced soft shadow, creating a tactile depth effect. Uses transform-based animation — no reflow.

🚀

Lift Card

Rises on hover with shadow.

Glow Effect .ease-glass-glow

Adds an elegant, premium glow aura around the card on hover. The glow remains subtle to avoid visual noise while still creating a "premium product" feel.

💜

Glow Card

Elegant aura on hover.

Enhanced Blur .ease-glass-blur

Increases backdrop blur intensity for a stronger frosted-glass effect. Pairs beautifully with the base glass card for ultra-premium compositions.

🌫️

Deep Blur

Stronger frosted glass.

Combined Effects .ease-glass-hover .ease-glass-lift .ease-glass-glow

All interactive utilities composed together on a single card — hover, lift, and glow work in harmony. Hover to experience the full premium interaction.

Quick Usage

<!-- Basic glass card -->
<div class="ease-glass-card">Glass Card</div>

<!-- Glass card + hover effect -->
<div class="ease-glass-card ease-glass-hover">Hover Card</div>

<!-- Glass card + lift on hover -->
<div class="ease-glass-card ease-glass-lift">Premium Card</div>

<!-- Glass card + glow aura -->
<div class="ease-glass-card ease-glass-glow">Glow Card</div>

<!-- Glass card + enhanced blur -->
<div class="ease-glass-card ease-glass-blur">Deep Frost</div>

<!-- Compose multiple effects -->
<div class="ease-glass-card ease-glass-hover ease-glass-lift ease-glass-glow">
  Full Premium Card
</div>
