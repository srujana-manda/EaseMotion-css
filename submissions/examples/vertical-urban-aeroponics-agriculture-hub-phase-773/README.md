# Vertical Urban Aeroponics Agriculture Hub

A modern single-page HTML/CSS showcase demonstrating a futuristic Vertical Urban Aeroponics Agriculture Hub built using the EaseMotion CSS framework. The interface highlights AI-powered farming, renewable energy, precision agriculture, smart irrigation, live monitoring, and sustainable food production through responsive layouts and smooth animations.

---

## Features

- Responsive single-page layout
- Hero section with animated background
- Farm statistics dashboard
- Smart farming overview cards
- Farming systems showcase
- Aeroponics workflow timeline
- Live farm monitoring dashboard
- Crop monitoring section
- Growing zones showcase
- Glassmorphism-inspired UI
- Hover animations and transitions
- Pure HTML & CSS (No JavaScript)

---

## Technologies Used

- HTML5
- CSS3
- EaseMotion CSS
- CSS Grid
- Flexbox
- CSS Animations
- Media Queries
