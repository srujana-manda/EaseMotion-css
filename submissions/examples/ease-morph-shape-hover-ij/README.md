# Morph Shape Hover

Shape morphs smoothly between a circle and rounded square on hover. Uses CSS `border-radius` transition for the morph effect.

## Features

- Circle-to-square morph on hover via `border-radius` transition
- Rotate and scale transform on hover
- Smooth ease-in-out transition
- Pure CSS, no JS required

## Usage

Add class `.msh-shape` to a `div`. The morph animation plays on hover automatically.
