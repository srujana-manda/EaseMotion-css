# Pulse Notification Dot

A lightweight animated status indicator built with pure CSS.

The Pulse Notification Dot provides a smooth pulsing effect for displaying notification states, live activity, and status indicators.

## Features

- Smooth pulse animation
- Multiple status variants
- Pure CSS implementation
- Customizable using CSS variables
- Lightweight and reusable
- Accessible with reduced-motion support

## Usage

Add the indicator element:

```html
<span class="pulse-dot"></span>
