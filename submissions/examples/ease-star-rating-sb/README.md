# Ease Star Rating

## Overview

The **Ease Star Rating** component is a reusable, pure CSS rating widget that provides an interactive star-based rating experience. It includes hover animations, selection feedback, responsive design, and accessibility features without requiring JavaScript.

## Features

- ⭐ Pure CSS implementation
- ⭐ Animated hover preview
- ⭐ Click confirmation pulse animation
- ⭐ Read-only rating example
- ⭐ Responsive layout
- ⭐ Keyboard accessibility with `:focus-visible`
- ⭐ Easy to customize and reuse

## Files

```
ease-star-rating-sb/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in any modern browser.
2. Hover over the stars to preview a rating.
3. Click a star to select a rating.
4. Customize colors, sizes, and animations in `style.css`.

## Accessibility

- Keyboard-friendly structure
- Visible focus indicators
- Responsive across desktop, tablet, and mobile devices
