## Google Loading Animation
Loading indicator with Google-style bouncing dots and animated progress bar. Dots bounce with staggered delays via --ldp-i and feature Google's signature colors.

## Features
- Bouncing dots with scale animation
- Staggered animation delays via --ldp-i
- Google brand colors (Blue, Red, Yellow, Green)
- Animated gradient progress bar
- Pure CSS, no JS required

## Usage
Add .dot elements inside .dot-wave container. Set --ldp-i (0-based index) for stagger delay. Include the progress bar and label for complete loading experience.
