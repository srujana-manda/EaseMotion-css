# Animated File Upload (`ease-upload-cv`)

A responsive animated file upload component built using **HTML5** and **Pure CSS**.

## Features

- Animated upload area
- Floating upload icon
- Hover glow effect
- Shimmer hover animation
- Styled file input
- Responsive layout
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-upload-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

> Note: This component demonstrates the visual upload interface only. File processing requires JavaScript or a backend implementation.

## Browser Support

Works in all modern browsers.
