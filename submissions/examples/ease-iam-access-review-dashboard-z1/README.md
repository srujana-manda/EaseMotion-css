# Ease IAM Access Review Dashboard

## Overview

A responsive HTML and CSS component for reviewing user access, privileged accounts, inactive identities, and pending access requests.

## Features

- User access review summary
- Pending approval tracking
- Inactive account monitoring
- Privileged access alerts
- Responsive layout
- Pure HTML & CSS
- CSS variables and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable identity governance component for IAM dashboards, security operations, compliance audits, and enterprise access review workflows.
