# CSS Magnetic Pull Popover

A pure CSS Magnetic Pull Popover designed for minimalist tech interfaces.

## Features

- Pure HTML & CSS
- Smooth magnetic pull animation
- Responsive design
- Keyboard accessible
- CSS Custom Properties
- No JavaScript

## Files

```
magnetic-pull-popover/
│── index.html
│── style.css
└── README.md
```

## Usage

1. Open `index.html` in your browser.
2. Hover or focus on the button to view the magnetic pull popover.

## Customization

Adjust the CSS variables in `style.css`.

```css
:root{
  --duration: .4s;
  --easing: cubic-bezier(.22,1,.36,1);
  --scale: 1.05;
}
```
