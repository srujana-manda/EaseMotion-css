# ⚡ Motion Studio Pro (Framer Edition)

A lightweight Framer-like animation editor built using HTML, CSS, and Vanilla JS.

---

## 🧠 What it does

- Live transform editor
- Timeline simulation
- Animation presets
- Real-time preview
- Framer-style UI layout

---

## 🚀 Features

- Drag-free animation control system
- Live transform engine
- Animation presets (fade, zoom, slide, bounce)
- Timeline playback simulation
- Minimal Framer-inspired UI

---

## 🎯 Goal

To simulate a **mini Framer Motion editor in pure frontend code**.

---

## 🔥 Future upgrades

- real keyframe editor
- drag & drop elements
- multi-layer animation system
- export JSON animation schema
