# Text Reveal Animations

A collection of smooth text-based animations to complement the EaseMotion library.

## Animations Included

| Class | Description |
|---|---|
| `.em-text-fade-up` | Smoothly fades text in while moving upwards. |
| `.em-text-expand` | Expands letter spacing for a cinematic feel. |
| `.em-text-glitch` | A subtle, looping glitch effect. |

## Usage
Simply include the `style.css` in your project and apply the classes to any text element.
