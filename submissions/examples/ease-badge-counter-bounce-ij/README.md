# Badge Counter Bounce

Notification badge counter with bounce animation on value change. Badge scales with bounce effect on update.

## Features

- Badge counter display
- Bounce animation on change via `.bcb-bounce` class
- Scale transform with bounce cubic-bezier
- JS triggers animation on value change

## Usage

Set `--bcb-count` (integer) on `.bcb-badge` and toggle `.bcb-bounce` class. JS handles value updates.
