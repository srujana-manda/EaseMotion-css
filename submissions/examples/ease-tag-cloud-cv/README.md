# Animated Tag Cloud (`ease-tag-cloud-cv`)

A responsive animated tag cloud component built using **HTML5** and **Pure CSS**.

## Features

- Floating animation
- Hover lift effect
- Shimmer hover animation
- Multiple tag sizes
- Responsive wrapping layout
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-tag-cloud-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

Works in all modern browsers.
