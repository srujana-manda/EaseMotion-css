# Hover Ripple Effect

A pure CSS button with an expanding ripple animation on hover — no JavaScript.

## Features
- Center-origin ripple via pseudo-element scale
- Smooth transform transition
- Dual color variant support

## Usage
```html
<button class="ripple-btn">Hover Me</button>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
