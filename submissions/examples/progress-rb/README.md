# Progress Bar & Circular Progress Component

Pure CSS linear and circular progress indicators. No SVG or JavaScript required.

## Files

- `demo.html` — Live preview with linear, striped, animated, circular, and dark mode variants
- `style.css` — All progress styles, keyframes, and layout helpers
