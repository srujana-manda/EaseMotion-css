# ease-drawer-slide-right

Side drawer panel that slides in from the right edge of the viewport. An overlay backdrop fades in behind the drawer when open.

## Features

- Drawer slides in from the right using translateX
- Semi-transparent overlay backdrop with opacity transition
- Close via X button, Cancel button, or clicking the overlay
- Header, scrollable body, and footer sections

## CSS Custom Property Control

- `--open`: Set to `1` to slide the drawer in (translateX 0) and show the backdrop, `0` to slide it back out.
