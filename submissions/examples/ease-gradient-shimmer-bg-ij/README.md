# Gradient Shimmer BG

Animated gradient background with shimmer sweep overlay. The background cycles through a multi-color gradient while a diagonal shimmer sweeps across.

## Features

- Multi-color animated gradient via `background-position` keyframes
- Shimmer sweep overlay for depth
- Pure CSS, no JS required
- Customizable via CSS custom properties

## Usage

Add the `.gsb-bg` container with a `.gsb-shimmer` child overlay and `.gsb-content` for text. Animation runs automatically.
