# Blur-Entrance Carousel – Product Catalog

A pure CSS Blur-Entrance Carousel component for EaseMotion CSS.

## Features

- Pure CSS (No JavaScript)
- Blur-Entrance transition animation
- Product Catalog themed design
- Responsive layout
- Keyboard accessible navigation
- Customizable using CSS custom properties

## Custom Properties

```css
--ease-duration
--ease-easing
--ease-blur
--ease-scale
--primary
```
