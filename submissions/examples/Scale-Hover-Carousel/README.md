# CSS Scale-Hover Carousel – Cyberpunk Neon

A responsive pure CSS carousel featuring a cyberpunk neon theme with smooth scale-hover animations.

## Features

- Pure HTML & CSS
- Responsive layout
- Scale-hover animation
- Neon glow effect
- Keyboard accessible
- CSS custom properties for easy customization

## CSS Variables

```css
:root{
  --primary:#00f7ff;
  --secondary:#ff00ff;
  --scale:1.12;
  --transition:0.4s;
}
```
