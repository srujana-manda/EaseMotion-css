# Ripple Click Effect

Ripple expands from the click point on the element. Uses JavaScript to capture click coordinates and CSS keyframes for the ripple animation.

## Features

- Ripple expands from click point
- Opacity fades as ripple grows
- Pure CSS animation, minimal JS for coordinate capture
- Customizable via CSS custom properties

## Usage

Click the `.rce-element`. JS creates a `.rce-ripple` span at click coordinates. CSS handles the scale and fade animation.
