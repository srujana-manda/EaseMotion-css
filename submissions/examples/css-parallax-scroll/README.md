# CSS Parallax Scroll

A pure CSS parallax scroll effect using perspective and transform — no JavaScript.

## Features
- Multi-layer parallax with different scroll speeds
- Uses CSS perspective and translateZ
- Foreground, midground, and background layers

## Usage
```html
<div class="parallax-wrap">
  <div class="parallax-layer layer-bg"></div>
  <div class="parallax-layer layer-mid"></div>
  <div class="parallax-layer layer-fore">...</div>
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
