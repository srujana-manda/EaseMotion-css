# Skeleton Block Full (#31846)

A full-width skeleton loading placeholder with a shimmering gradient sweep, mimicking content that is still loading.

## Features
- Large rectangular block placeholder
- Gradient sweep `@keyframes` for the shimmer effect
- Supports auxiliary skeleton lines (e.g. text rows)
- Always active (no toggle — represents persistent loading state)

## Usage
Use `.skeleton-block` for the main placeholder and `.skeleton-line` for text row placeholders within a card.
