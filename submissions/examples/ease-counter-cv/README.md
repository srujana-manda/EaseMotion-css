# Animated Number Counter (`ease-counter-cv`)

A lightweight animated number counter component built using **HTML5** and **Pure CSS**.

## Features

- Fade-up entrance animation
- Glow animation
- Gradient number styling
- Multiple counter cards
- Responsive grid layout
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-counter-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

> Note: Since this repository requires **Pure CSS**, the numbers are presented as static values with animated styling rather than dynamically counting upward.

## Browser Support

Works in all modern browsers.
