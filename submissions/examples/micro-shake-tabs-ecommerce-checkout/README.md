# CSS Micro Shake Tabs

## Description
A responsive CSS tabs component featuring a Micro Shake interaction effect.

## Features
- Responsive layout
- Keyboard accessible
- Pure HTML & CSS
- CSS custom properties
- Easy customization

## Files
- demo.html
- style.css

## Usage
Open `demo.html` in any modern browser.

## Customization
Modify the CSS variables in `:root` to change colors, duration, spacing, and other styles.

## Browser Support
- Chrome
- Firefox
- Edge
- Safari
