# ease-card-hover-border

A card with a conic gradient border that animates (rotates) on hover. Pure CSS — no JavaScript needed.

## Features
- Conic gradient border via a rotating pseudo-element
- Border rotation on hover
- Clean card layout with inner content area
- Dark theme styling

## Usage
Hover over the `.chb-card` element to start the border rotation animation. The gradient cycles through accent colors.
