# Ease StatefulSet Rollout Tracker

## Overview

A responsive HTML and CSS dashboard component for monitoring Kubernetes StatefulSet rollout progress and deployment health.

## Features

- StatefulSet rollout overview
- Rolling update monitoring
- Pending replica detection
- Rollout failure alerts
- Responsive layout
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable infrastructure dashboard for tracking StatefulSet updates, identifying rollout bottlenecks, and maintaining reliable deployments for stateful workloads.
