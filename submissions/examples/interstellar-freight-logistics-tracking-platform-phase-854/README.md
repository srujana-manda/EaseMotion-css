# Interstellar Freight Logistics Tracking Platform — Phase #854

A futuristic logistics dashboard for interstellar cargo operations. Built entirely with **HTML5**, **CSS3**, and **vanilla JavaScript**. Part of the EaseMotion CSS design system showcase.

---

## Features

- Animated warp trail background with streaking lines
- Dynamic star dust particle system
- Live fleet metrics with animated counters
- SVG route map with animated dash lines and pulsing nodes
- Shipment tracking cards with status-coded borders
- Cargo composition breakdown
- Hub capacity ring gauge
- Warp boost overlay with countdown
- Responsive layout
- Reduced motion support
- Zero external dependencies
