# Ease Kubernetes Pod Lifecycle

## Overview
A CSS-only dashboard component for displaying Kubernetes pod lifecycle information.

## Features
- Pod lifecycle status badges
- Restart count display
- Node assignment details
- Responsive layout
- CSS variables
- Hover animations
- Pure HTML & CSS

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` directly in any modern browser.

## Browser Support
- Chrome
- Firefox
- Edge
- Safari

## Responsive
Works well on desktop, tablet, and mobile devices.

## Why EaseMotion CSS?
Provides an animation-first, reusable Kubernetes monitoring component suitable for cloud-native dashboards, DevOps portals, and infrastructure management interfaces.
