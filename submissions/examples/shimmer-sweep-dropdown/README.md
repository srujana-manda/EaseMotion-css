# Shimmer Sweep Dropdown

A modern **Shimmer-Sweep Dropdown** built using pure HTML5 and CSS3 for creative portfolio layouts.

## Features

- Pure HTML & CSS
- Shimmer sweep hover effect
- Smooth dropdown animation
- Responsive layout
- Keyboard accessible (`:focus-within`)
- Supports `prefers-reduced-motion`
- No JavaScript required

## Folder Structure

```
shimmer-sweep-dropdown/
├── demo.html
├── style.css
└── README.md
```

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Usage

Open `demo.html` in a modern browser.
