New Component

ease-cookie-consent-banner

A responsive, human-readable pure CSS cookie consent banner. Implements smooth entrance, slide-out exit animations, positioning rules, and layout structures with zero JavaScript.

Position & Interactive Controls

Click the buttons below to toggle different positions of the cookie banner in the viewport. State tracking is handled entirely via CSS checkbox mechanisms — no JS involved! Clicking "Accept All", "Decline", or "Close" within a banner will run its slide-out dismiss animation.

Installation & Usage

Simply include the wrapper container at the bottom of your HTML body. Hook up your interactive labels to trigger the state checkbox.

<!-- Cookie Banner Component Wrapper -->
<div class="ease-cookie-banner-wrapper">
  <!-- State input (hidden) -->
  <input type="checkbox" id="cookie-accept" class="ease-cookie-state">

  <!-- Visible Banner -->
  <div class="ease-cookie-banner">
    <div class="ease-cookie-content">
      <div class="ease-cookie-icon-wrapper">
        <!-- SVG Cookie Icon -->
      </div>
      <div class="ease-cookie-text-wrapper">
        <h4 class="ease-cookie-title">Cookie Consent</h4>
        <p class="ease-cookie-description">We use cookies...</p>
      </div>
    </div>

    <!-- Action Buttons (Labels) -->
    <div class="ease-cookie-actions">
      <label for="cookie-accept" class="ease-cookie-btn ease-cookie-btn--text">Decline</label>
      <label for="cookie-accept" class="ease-cookie-btn ease-cookie-btn--primary">Accept All</label>
    </div>
  </div>
</div>

Modifier Classes

Class Name Target Element Description
.ease-cookie-state <input type="checkbox"> Hides checkbox controller and enables sibling checked state targeting.
.ease-cookie-banner <div> Banner Box Base container style. Applies slide-in and default bottom-right positioning.
.ease-cookie-banner--bottom-left <div> Banner Box Positions the banner at the bottom left of the viewport.
.ease-cookie-banner--bottom-center <div> Banner Box Positions and centers the banner at the bottom of the screen.
.ease-cookie-banner--top <div> Banner Box Positions/centers the banner at the top. Slides down on entrance.
.ease-cookie-banner--top-bar <div> Banner Box Spans full viewport width at the top like a header notification bar.
