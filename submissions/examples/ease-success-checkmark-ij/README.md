# ease-success-checkmark

An animated SVG checkmark that draws on with a circle scale-in effect.

## Features

- SVG circle scales in via stroke-dashoffset animation
- Checkmark path draws sequentially after circle
- Green color scheme for success state
- Controlled via `--show` CSS custom property

## Usage

Set `--show: 1` on the `.scm-checkmark` element to animate the checkmark in. Set to `0` to reset.
