# Toggle Switch Pill

A pill-shaped toggle switch with animated knob and labels. Toggle state via `--ts-on` (0 or 1). CSS handles knob translation, background color, and label opacity transitions.

## Features

- Animated pill toggle knob
- Knob position via `--ts-on` CSS variable
- Background color changes with state
- On/Off label opacity transition
- Keyboard accessible with Enter/Space

## Usage

Set `--ts-on` (0 or 1) on `.ts-pill`. CSS translates the knob, changes background, and fades labels accordingly.
