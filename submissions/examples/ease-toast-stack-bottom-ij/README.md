# ease-toast-stack-bottom

Multiple toast notifications stacking from the bottom center of the viewport. Each toast slides up with a smooth animation and auto-dismisses after 4 seconds.

## Features

- Toasts stack vertically from the bottom of the screen
- Slide-up entry animation for each new toast
- Color-coded left border for success, error, warning, and info variants
- Dismiss button on each toast
- Auto-removal after 4 seconds with fade-out transition
- JavaScript-powered add button for demonstration

## CSS Custom Property Control

- No `--` custom property needed for basic operation. Each toast uses an inline `--i` for staggered delay if desired. Auto-dismiss is handled by JavaScript.
