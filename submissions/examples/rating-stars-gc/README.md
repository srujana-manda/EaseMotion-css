# Rating Stars UI Example

## 1. What does this do?

This submission provides interactive star rating components with smooth hover fill animations and click-to-rate functionality. Stars animate from empty to filled with a satisfying visual transition, creating an engaging user experience for rating interactions.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example demonstrates multiple star ratings with different states. Copy the `.star-rating` markup and CSS into your project to integrate the custom star rating pattern. Include minimal inline JavaScript for click handlers.

---

## 3. Why is it useful?

Custom star ratings provide visual feedback that surpasses default form controls. Smooth animations on hover and click create intuitive interactions, while the clean design integrates seamlessly into modern UI systems.
