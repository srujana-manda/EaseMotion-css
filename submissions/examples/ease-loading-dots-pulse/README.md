# Loading Dots Pulse

Pulsing loading dots animation — pure CSS.

## Features
- Three-dot pulsing animation with staggered delays
- Configurable color via `--dot-color`
- Configurable size via `--dot-size`
- Large variant class `.lg`

## Usage
```html
<div class="dots">
  <span class="dot"></span>
  <span class="dot"></span>
  <span class="dot"></span>
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
