# Animated Glass Card (`ease-glass-card-cv`)

A modern glassmorphism card built entirely using **HTML5** and **Pure CSS**.

## Features

- Glassmorphism effect
- Backdrop blur
- Floating animated background blobs
- Hover lift animation
- Gradient action button
- Responsive layout
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-glass-card-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

Works in all modern browsers supporting `backdrop-filter`.
