# CSS Wobble-Focus Tabs for Dashboard Analytics Layouts

**Issue:** #50214

## Description

A pure CSS animated **Wobble-Focus Tabs** component designed for Dashboard Analytics layouts.

## Features

- Pure HTML & CSS
- Wobble-Focus animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript

## Usage

```html
<link rel="stylesheet" href="style.css">
```

Use the HTML structure provided in `demo.html`.

## Customization

```css
:root{
  --primary:#2563eb;
  --duration:.5s;
}
```
