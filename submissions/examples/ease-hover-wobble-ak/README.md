# Ease Hover Wobble

## Description

This submission introduces a new hover animation utility called **ease-hover-wobble**.

It creates a subtle left-right rotation when the user hovers over an element.

## Features

- Lightweight CSS animation
- 300ms duration
- Smooth ease-in-out timing
- Suitable for buttons, icons and cards

## Usage

```html
<button class="ease-hover-wobble">Hover Me</button>
```

```css
.ease-hover-wobble:hover {
  animation: ease-kf-hover-wobble 300ms ease-in-out both;
}
```
