# Dropdown Fade Slide

A dropdown menu with fade and slide animation — pure CSS.

## Features
- Fade-in and slide-down animation on hover
- Hover state for menu items
- Clean, minimal design

## Usage
```html
<div class="dropdown">
  <button class="dropdown-btn">Menu ▾</button>
  <ul class="dropdown-menu">
    <li>Item</li>
  </ul>
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
