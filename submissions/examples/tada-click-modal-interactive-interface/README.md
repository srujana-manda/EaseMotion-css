# CSS Tada-Click Modal for Interactive Interface Layouts

A responsive, pure CSS modal featuring a smooth **Tada-Click** animation designed for modern interactive interfaces.

## Features

- Pure HTML & CSS
- No JavaScript
- Responsive
- Keyboard accessible
- CSS custom properties
- Smooth Tada-Click animation
- Easy to customize

## Files

```
demo.html
style.css
README.md
```

## CSS Variables

```css
:root{
    --duration:.65s;
    --ease:cubic-bezier(.175,.885,.32,1.275);
    --scale-start:.8;
}
```

These variables allow developers to customize the animation timing, easing, and scale.

## Why EaseMotion CSS?

This component provides a lightweight, reusable, animation-first modal with zero JavaScript while maintaining accessibility and responsiveness.
