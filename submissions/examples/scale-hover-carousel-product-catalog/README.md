# CSS Scale-Hover Carousel – Product Catalog

A responsive **Product Catalog Carousel** built with pure HTML and CSS featuring smooth scale-hover animations.

## Features

- Pure HTML & CSS
- Product catalog card layout
- Smooth hover scale animation
- Responsive design
- Keyboard accessible
- CSS custom properties
- No JavaScript required

## CSS Variables

```css
:root{
  --primary:#4f46e5;
  --secondary:#6366f1;
  --scale:1.08;
  --duration:.35s;
}
```

## Folder Structure

```
scale-hover-carousel-product-catalog/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in a browser and customize colors, animation speed, and scale using the CSS variables.
