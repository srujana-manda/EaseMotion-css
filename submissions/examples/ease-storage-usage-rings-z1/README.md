# Ease Storage Usage Rings

## Overview
A CSS-only component for displaying storage usage with animated circular progress rings.

## Features
- Circular progress indicators
- Responsive grid layout
- Pure HTML & CSS
- CSS variables
- Smooth animations

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` directly in your browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Responsive
Optimized for desktop, tablet, and mobile devices.

## Why EaseMotion CSS?
Provides a reusable animation-first component for storage analytics, cloud dashboards, server monitoring, and infrastructure management tools.
