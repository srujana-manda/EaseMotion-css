# Ease CDN Edge Region Status

## Overview
A CSS-only component for monitoring CDN edge regions and their operational status.

## Features
- Region status cards
- Animated health badges
- Responsive layout
- CSS variables
- Pure HTML & CSS

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` directly in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari

## Responsive
Optimized for desktop, tablet, and mobile devices.

## Why EaseMotion CSS?
Provides a reusable animation-first component for CDN dashboards, cloud infrastructure portals, network operations centers, and DevOps monitoring systems.
