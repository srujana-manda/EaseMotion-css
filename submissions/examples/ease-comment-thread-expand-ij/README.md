# Comment Thread Expand

Reply form slides down smoothly beneath a comment on click. Uses max-height transition with overflow hidden.

## Features

- Smooth slide-down reply form
- Toggle via `.cte-expanded` class
- Customisable easing and duration via CSS variables
- Pure CSS transitions

## Usage

Toggle `.cte-expanded` on `.cte-comment`. JS handles click toggling of the Reply button. CSS handles the max-height transition.
