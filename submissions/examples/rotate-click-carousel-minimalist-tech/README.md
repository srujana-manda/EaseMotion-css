# CSS Rotate-Click Carousel – Minimalist Tech

A clean Rotate-Click Carousel built using pure HTML and CSS for modern minimalist technology interfaces.

## Features

- Pure HTML & CSS
- Rotate-click animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript

## Usage

Open `demo.html` directly in any browser.

Customize using CSS variables:

```css
:root{
  --rotate:6deg;
  --duration:.35s;
  --accent:#4f46e5;
}
```
