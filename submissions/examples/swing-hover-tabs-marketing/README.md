# Swing Hover Tabs - Marketing Landing Page

## Description

A pure CSS animated tab component with a smooth Swing-Hover interaction.
Designed for modern marketing landing page interfaces.

## Features

- Pure CSS animation
- No JavaScript required
- Responsive design
- Keyboard accessible
- Custom CSS variables for animation control

## Usage

Add the `tab` class to any button element.

Example:

```html
<button class="tab">
    Features
</button>
