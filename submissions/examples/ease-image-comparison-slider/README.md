# Ease Image Comparison Slider

A reusable before/after image comparison component for EaseMotion CSS.

## Overview

Ease Image Comparison Slider creates an interactive before-and-after image reveal effect using a draggable slider.

It is useful for:

- Photo editing showcases
- UI redesign previews
- Product transformations
- Design comparisons


## Features

- Drag-to-reveal interaction
- Smooth divider animation
- Custom slider handle
- Before/After labels
- Responsive layout
- CSS-only implementation
- Customizable variables
