# CSS Rotate-Click Carousel – Interactive Interface

A responsive **Rotate-Click Carousel** built with pure HTML and CSS. Cards rotate slightly when clicked, hovered, or focused to create an engaging interactive interface.

## Features

- Pure HTML & CSS
- Rotate-click interaction
- Smooth transitions
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required

## CSS Variables

```css
:root{
  --primary:#4f46e5;
  --rotate:8deg;
  --duration:.4s;
}
```

## Folder Structure

```
rotate-click-carousel-interactive-interface/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in any browser.
2. Hover, click, or tab to a card to see the rotation effect.
3. Customize colors, rotation angle, transition speed, and border radius using the CSS variables.
