# Input Focus Border

Text input with animated bottom border on focus. Border expands from center with label floating up.

## Features

- Input with bottom border animation
- Border expands from center on focus
- Label floats up on focus
- Smooth color and width transitions

## Usage

Structure `.ifb-field` containing `.ifb-input`, `.ifb-label`, and `.ifb-bar`. CSS handles focus states via `:focus`.
