# ease-chip-remove

Animated removable chip/tag component for EaseMotion CSS.

## Features

- Smooth chip removal animation
- Shrink and fade exit effect
- Bounce entrance animation
- Multiple color variants
- Interactive close button
- Pure CSS animation

## Files

- demo.html
- style.css
- README.md

## Use Cases

- Tags
- Filters
- Skills lists
- Multi-select components

## Customization

Users can modify:

- Chip colors
- Animation duration
- Border radius
- Button styles
