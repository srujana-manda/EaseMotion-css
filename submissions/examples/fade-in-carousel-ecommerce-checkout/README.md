# Fade-In Carousel for E-Commerce Checkout Layouts

**Issue:** #50409

## Overview

A lightweight pure CSS carousel with a smooth fade-in transition designed for modern e-commerce checkout pages.

## Features

- Pure CSS
- Fade animation
- Responsive layout
- Keyboard friendly
- Easy customization
- Uses CSS custom properties

## Files

- demo.html
- style.css
- README.md

## Usage

```html
<div class="fade-carousel">
    <div class="slide">...</div>
</div>
```

## Customization

```css
:root{
    --duration:8s;
    --radius:18px;
}
```
