# Flip Clock Timer

Flip clock timer with number flip animation. Each digit flips vertically on change with perspective transform and rotateX keyframes.

## Features

- Number flip animation on change via @keyframes
- Perspective transform for 3D effect (200px)
- Split flap visual style with top/bottom halves
- Start/Stop and Reset controls
- MM:SS format

## Usage

JS sets `--fc-val` and triggers the `fc-flip` class. CSS handles the `rotateX` keyframes for the flip animation.
