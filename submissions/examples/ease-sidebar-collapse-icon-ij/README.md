# Ease Sidebar Collapse Icon Component

A smooth, performance-focused sidebar menu toggle icon that morphs fluidly between a classic menu state and a directional cross toggle point.

## ✨ Motion Features & Easing Specifications

- **Elastic Morph Trajectory:** Rather than tracking along linear transition models, the component relies on an ultra-responsive `cubic-bezier(0.68, -0.60, 0.32, 1.60)` curve. This configuration creates a slight pull-back anticipation effect before snapping into cross alignment coordinates.
- **Micro-Interactions:** Includes hover-sensitive color loops that smoothly shift to deep indigo tones before triggering activation.
- **Layout Security:** Runs completely on hardware-accelerated transformation properties (`transform` and `opacity`) to ensure solid 60FPS refresh metrics across mobile and web layouts.
