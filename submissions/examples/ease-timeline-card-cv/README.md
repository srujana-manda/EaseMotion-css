# Animated Timeline Card (`ease-timeline-card-cv`)

A responsive animated timeline card component built using **HTML5** and **Pure CSS**.

## Features

- Vertical timeline layout
- Animated timeline nodes
- Fade-in card entrance
- Hover lift effect
- Responsive design
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-timeline-card-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

Works in all modern browsers.
