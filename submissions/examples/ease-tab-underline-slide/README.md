# Tab Underline Slide

A tab bar with sliding underline indicator — pure CSS.

## Features
- Sliding underline indicator using CSS :has()
- Smooth cubic-bezier transition
- Radio-based tab selection
- 4 tab variants

## Usage
```html
<div class="tab-bar">
  <label class="tab">
    <input type="radio" name="tabs" checked>
    <span class="tab-label">Tab</span>
  </label>
  <div class="tab-slider"></div>
</div>
```

## Browser Support
- Chrome 105+, Firefox 121+, Safari 15.4+

## Tech Stack
- HTML + CSS only, no JavaScript
