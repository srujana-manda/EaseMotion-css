# Magnetic Button Pulse

A button with a magnetic pulse effect using CSS custom properties — pure CSS.

## Features
- Pulsing glow animation on hover
- Radial gradient overlay for depth
- Dual color variant support

## Usage
```html
<button class="mag-btn">Hover Me</button>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
