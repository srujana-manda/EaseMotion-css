How does pure CSS validation work?

CSS uses the :valid, :invalid, and :placeholder-shown pseudo-classes combined with the :has() relational selector to style the strength meter, feedback messages, and requirement indicators — all without JavaScript.

The pattern attribute on the <input> defines a regex that the browser validates in real time. CSS reacts to the validation state changes.

  • Empty → :placeholder-shown hides the meter
  • Weak → :invalid shows red bar, weak feedback
  • Strong → :valid shows green bar, strong feedback

Note: Pure CSS cannot read the input value character-by-character, so the meter has three states rather than per-character granularity. Multi-segment real-time analysis requires JavaScript.
