# Ease Calendar Week View

## Overview

The **Ease Calendar Week View** is a responsive weekly calendar component that displays events across seven days. It is designed as a clean, modern, and accessible example for EaseMotion CSS.

## Features

- Weekly calendar layout (Monday–Sunday)
- Responsive CSS Grid design
- Interactive event cards
- Hover animations
- Keyboard accessibility using `:focus-visible`
- Mobile-friendly layout
- Easy to customize

## Files

```
ease-calendar-week-view-sb/
```

## Usage

1. Open `demo.html` in any modern web browser.
2. Customize the event cards, colors, or layout as needed.
3. Integrate the component into your project.

## Accessibility

- Keyboard-focusable event cards
- Visible focus indicators
- Semantic HTML structure
- Responsive across desktop, tablet, and mobile devices
