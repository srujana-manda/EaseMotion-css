# Drop Exit Effect

A simple CSS animation that makes an element appear to drop downward while fading out, creating a smooth exit effect.

## Features

- 🎯 Pure HTML & CSS
- 💨 Smooth drop exit animation
- 🎨 Customizable with CSS variables
- 🚫 No JavaScript required

## Usage

```html
<div class="ease-drop-exit">
  Bye 👋
</div>
```

## Customization

| Variable | Default |
|----------|---------|
| `--drop-duration` | `1.2s` |
| `--drop-distance` | `180px` |
| `--drop-rotate` | `12deg` |
| `--drop-easing` | `ease-in` |

## Browser Support

Works in all modern browsers supporting CSS animations.
