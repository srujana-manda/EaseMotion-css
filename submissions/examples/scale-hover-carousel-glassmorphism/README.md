# CSS Scale-Hover Carousel – Glassmorphism UI

A responsive **Glassmorphism-inspired Scale-Hover Carousel** built with pure HTML and CSS. The cards feature a frosted-glass appearance with smooth hover scaling and customizable animation settings.

## Features

- Pure HTML & CSS
- Glassmorphism design
- Smooth scale-hover animation
- Responsive layout
- Keyboard accessible
- CSS custom properties for easy customization
- No JavaScript required

## CSS Variables

```css
:root{
  --glass-bg: rgba(255,255,255,0.15);
  --glass-border: rgba(255,255,255,0.35);
  --blur: 14px;
  --scale: 1.1;
  --duration: .35s;
}
```

## Usage

1. Open `demo.html` in any browser.
2. Modify the CSS custom properties in `style.css` to change the blur, animation speed, scale factor, or colors.
