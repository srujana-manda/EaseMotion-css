How does pure CSS state switching work?

Radio inputs and the :has() selector drive all state changes:

  • Signal bars fill based on :has(#sig-{level}:checked)
  • Connection state (Connected/Connecting/Disconnected) toggles via :has(#state-{name}:checked)
  • Network selection highlights via input:checked + label
  • Connecting animation — CSS @keyframes spinner
