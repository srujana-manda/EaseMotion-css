# Ease Log Severity Timeline

## Overview
A CSS-only timeline component for displaying application and server logs with severity indicators.

## Features
- Vertical log timeline
- Animated severity indicators
- Responsive layout
- CSS variables
- Pure HTML & CSS

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` in any modern browser.

## Browser Support
Chrome, Firefox, Edge, Safari

## Responsive
Works on desktop, tablet, and mobile devices.

## Why EaseMotion CSS?
Provides a reusable animation-first component for observability dashboards, SIEM tools, backend monitoring systems, and DevOps platforms.
