📖 How to Use

<span class="ease-number-counter" data-target="2500" data-suffix="+">0</span>

Data Attributes:

  • data-target – The number to count up to (required)
  • data-duration – Animation duration in ms (default: 2000)
  • data-suffix – Text to append (e.g., "+", "%", "K")
  • data-precision – Decimal places (default: 0)
  • data-easing – "linear" or "ease-out" (default: "linear")
