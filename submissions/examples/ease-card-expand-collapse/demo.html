What is EaseMotion?−

EaseMotion is a pure CSS animation library for building smooth, accessible UI components without JavaScript.

How do I use it?+

Add the CSS classes to your HTML elements. Each component is self-contained and customizable via CSS custom properties.

Is it accessible?+

Yes! Components support keyboard navigation, focus-visible styles, and reduced-motion preferences.
