# Card Expand Collapse

An expandable card with content reveal animation — pure CSS.

## Features
- Uses native `<details>/<summary>` HTML element
- Smooth expand/collapse animation
- Icon rotation indicator
- Keyboard accessible by default

## Usage
```html
<details class="expand-card">
  <summary class="expand-header">Question?</summary>
  <div class="expand-body">Answer...</div>
</details>
```

## Browser Support
- Chrome 12+, Firefox 49+, Safari 6+

## Tech Stack
- HTML + CSS only, no JavaScript
