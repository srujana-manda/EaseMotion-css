# Shimmer Sweep Tabs

CSS-only tabs with a smooth shimmer sweep animation on the active tab, designed for creative portfolio layouts.

## Usage

Include `style.css` and structure your HTML with radio inputs, labels, and tab panels inside `.shimmer-sweep-tabs`. Customize via CSS custom properties (`--tabs-duration`, `--tabs-easing`, `--tabs-shimmer-color`, `--tabs-shimmer-width`, `--tabs-scale`).

## Why

Lightweight, keyboard accessible, and JavaScript-free — adds a polished animated tab experience without framework dependencies.
