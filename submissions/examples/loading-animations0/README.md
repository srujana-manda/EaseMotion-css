# Loading Animations

A collection of pure‑CSS, GPU‑accelerated loading indicators.

## Demo
Open `demo.html` in your browser to see:
- Pulsing circle
- Bouncing dots
- Spinning circle

## Usage
Include the stylesheet:
```html
<link rel="stylesheet" href="style.css" />

<!-- Pulsing loader -->
<div class="loader pulse"></div>

<!-- Bouncing dots -->
<div class="dots">
  <div class="dot"></div>
  <div class="dot"></div>
  <div class="dot"></div>
</div>

<!-- Spinning circle -->
<div class="spinner"></div>
