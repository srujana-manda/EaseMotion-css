# Pure CSS Shimmer Sweep Tabs

## Context
This example adds a modern SaaS Showcase-style tabs pattern to the EaseMotion examples library. It keeps state management in CSS only, while still giving users a polished motion cue through a shimmer sweep transition.

## Features
- Zero JavaScript.
- Keyboard accessible through native radio-button navigation.
- Responsive layout that stacks cleanly on smaller screens.
- Customizable through standard CSS custom properties like `--em-shimmer-duration`, `--em-shimmer-ease`, `--em-tab-scale`, and color tokens.

## Notes
The demo is intentionally self-contained so it can be reused as a pattern reference or dropped into a showcase layout without any scripting overhead.
