# Glassmorphic Profile Card

A beautiful, responsive profile card component demonstrating the power of **EaseMotion CSS** combined with modern glassmorphism design.

## ✨ Features
- **Zero-dependency animations**: Uses `ease-fade-in`, `ease-slide-up`, and `ease-hover-lift` for smooth, performant motion.
- **Staggered entrance**: Elements load sequentially using `ease-delay-*` classes for a polished feel.
- **Interactive hover states**: Avatar grows on hover (`ease-hover-grow`), and the card lifts with a shadow (`ease-hover-lift`).
- **Glassmorphism aesthetic**: Custom CSS provides a frosted glass effect that works beautifully on gradient backgrounds.
