# Slide Toggle Switch

A sliding toggle switch — pure CSS.

## Features
- Smooth sliding knob animation
- Checkbox-based toggle with spring-like cubic-bezier
- Disabled state styling
- Settings-row layout example

## Usage
```html
<label class="toggle">
  <input type="checkbox">
  <span class="toggle-track">
    <span class="toggle-knob"></span>
  </span>
</label>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
