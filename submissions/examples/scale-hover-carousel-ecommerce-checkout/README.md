# CSS Scale-Hover Carousel – E-Commerce Checkout

A responsive **E-Commerce Checkout Carousel** built using pure HTML and CSS with smooth scale-hover animations.

## Features

- Pure HTML & CSS
- E-commerce checkout card design
- Smooth scale-hover interaction
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required

## CSS Variables

```css
:root{
  --primary:#16a34a;
  --secondary:#22c55e;
  --scale:1.08;
  --duration:.35s;
}
```

## Folder Structure

```
scale-hover-carousel-ecommerce-checkout/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any browser and customize the colors, scale, transition duration, and spacing using CSS variables.
