# Pull to Refresh

A CSS pull-to-refresh overscroll animation utility — pure CSS.

## Features
- Three-dot pulsing spinner animation
- Simulated overscroll pull-down
- Reusable spinner pattern

## Usage
```html
<div class="spinner">
  <div class="spinner-dot d1"></div>
  <div class="spinner-dot d2"></div>
  <div class="spinner-dot d3"></div>
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
