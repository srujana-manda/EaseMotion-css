New Component

ease-circle-progress-ring

A responsive, vector-based circular progress ring. Updates filled dash offsets and increments inner percentage counters using 100% pure CSS custom variables and counters.

Syncing

Default Progress

Standard 75% progress fill. Renders static variables, styling rings and counter metrics.

--ease-progress: 75
Complete

Success Accent

Complete 100% state. Leverages a success color modifier class to update fill borders.

.ease-circle-progress-ring--success
Hover Me

Interactive Hover

Starts at 20%. Hovering over this card transitions progress value and inner numbers smoothly to 100%.

Hover to animate

Continuous Loader

.ease-circle-progress-ring--oscillate

Installation & Usage

Wrap standard SVG circles inside the ring container. Target fill levels directly using the inline CSS custom variable --ease-progress.

<!-- Progress Ring -->
<div class="ease-circle-progress-ring" style="--ease-progress: 75;">
  <svg class="ease-circle-progress-ring-svg" viewBox="0 0 36 36">
    <circle class="ease-circle-progress-ring-bg"></circle>
    <circle class="ease-circle-progress-ring-bar"></circle>
  </svg>
  <div class="ease-circle-progress-ring-text">
    <!-- Leave percent span empty to display CSS counter percentage -->
    <span class="ease-circle-progress-ring-percent"></span>
    <span class="ease-circle-progress-ring-label">Processing</span>
  </div>
</div>

Modifier Classes

Class Name Target Element Description
.ease-circle-progress-ring Container Block Base container. Instantiates counters and variable transitions.
.ease-circle-progress-ring--sm Container Block Shrinks dimensions to 72px and lowers font sizes.
.ease-circle-progress-ring--lg Container Block Enlarges dimensions to 180px and scales progress bar width.
.ease-circle-progress-ring--success Container Block Overrides accent progress bar color to green.
.ease-circle-progress-ring--danger Container Block Overrides accent progress bar color to red.
.ease-circle-progress-ring--warning Container Block Overrides accent progress bar color to amber.
.ease-circle-progress-ring--info Container Block Overrides accent progress bar color to blue.
.ease-circle-progress-ring--oscillate Container Block Initiates a continuous 4s infinite loading animation loop.

Customization Variables

CSS Variable Name Default Value Description
--ease-progress 0 Progress value integer (0 to 100). Drives offset and counter.
--ease-progress-accent #6366f1 Color of the animated filling progress bar.
--ease-progress-bg rgba(255,255,255,0.05) Color of the static background track circle.
--ease-progress-size 120px Height/width coordinate dimensions of the ring.
--ease-progress-width 3.5px Border thickness of the progress bar path.
