# Button Hover Shine

A lightweight CSS utility that adds a smooth shine animation to buttons when hovered.

## Features

- Smooth shine hover animation
- Pure HTML & CSS
- Lightweight and reusable
- Easy to customize
- Modern UI effect

## Usage

```html
<button class="shine-button">
  Get Started
</button>
```
