# Fluid Morph UI Kit (`.ease-fluid-*-xyz`)

An organic, water-inspired UI kit built entirely with pure CSS. It focuses on smooth, continuous morphing animations, liquid hover effects, and fluid loaders, utilizing custom `border-radius` keyframes and organic easing curves.

## Features

- 🫧 **Morphing Background Blobs**: Continuous, organic shape-shifting using complex `border-radius` keyframes.
- 💧 **Liquid Fill Cards**: Cards that "fill up" with a smooth, morphing liquid effect on hover.
- 🌊 **Organic Easing**: Custom `cubic-bezier` curves that mimic the natural physics of water.
- 🔄 **Fluid Loaders**: Three variations of morphing, stretching, and waving CSS loaders.
- 🎨 **Soft Aesthetic**: A calming, deep-ocean color palette with `mix-blend-mode` effects.

## Usage

Copy the HTML structure and the CSS classes into your project. The background blobs (`.ease-fluid-bg-xyz`) are fixed and blurred, creating a dynamic, living background.

```html
<!-- Example: Liquid Fill Card -->
<div class="ease-fluid-card-xyz">
    <div class="ease-fluid-card-content-xyz">
        <div class="ease-fluid-card-icon-xyz">💧</div>
        <h3>Liquid Fill</h3>
        <p>Hover to see the effect.</p>
    </div>
</div>

<!-- Example: Morphing Button -->
<button class="ease-fluid-btn-xyz ease-fluid-btn-primary-xyz">
    <span>Get Started</span>
</button>
