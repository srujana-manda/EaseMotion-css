# Atmospheric Carbon Capture Grid Analytics Showcase

A responsive, interactive dashboard UI for a carbon capture grid monitoring system. Built with **EaseMotion CSS** utility classes and minimal custom CSS. Zero JavaScript.

## Sections
- **Header**
- **KPI Row**
- **Capture Grid Map**
- **Control Panel**
- **Activity Log**

## EaseMotion Classes Showcased
- **Layout:** `ease-container`, `ease-flex`, `ease-grid`, `ease-gap-*`, `ease-justify-between`, `ease-items-center`, `ease-min-h-screen`, `ease-py-*`, `ease-px-*`, `ease-mb-*`, `ease-mt-*`, `ease-h-64`, `ease-lg-col-span-2`
- **Background:** `ease-bg-gray-900`, `ease-bg-gray-800`, `ease-bg-green-900`
- **Typography:** `ease-text-*`, `ease-font-bold`, `ease-font-semibold`, `ease-text-gray-*`, `ease-text-green-*`, `ease-text-blue-*`, `ease-text-purple-*`, `ease-text-yellow-*`, `ease-text-white`
- **Components:** `ease-card`, `ease-btn`, `ease-btn-primary`, `ease-badge`
- **Hover Effects:** `ease-hover-scale-105`, `ease-hover-text-green-300`
- **Animations:** `ease-fade-in`, `ease-delay-*`, `ease-transition`
- **Utilities:** `ease-rounded-lg`, `ease-w-full`, `ease-block`, `ease-uppercase`, `ease-space-y-4`

## How to Use
1. Open `demo.html` in any modern browser.
2. Adjust the sliders and observe the dashboard.
