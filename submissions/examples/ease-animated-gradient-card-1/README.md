# Animated Gradient Card

A pure CSS animated gradient border component designed to integrate seamlessly with **EaseMotion CSS**. 

## ✨ Features
- 🌈 Smooth, continuous, performant CSS-only gradient animation.
- 🚀 Leverages native EaseMotion utility classes (`ease-fade-in`, `ease-slide-up`, `ease-hover-grow`) for entrance and interaction animations.
- ♿ Fully respects `prefers-reduced-motion` for accessibility compliance.
- 📦 Zero JavaScript dependencies.
