# ease-badge-pulse

A notification badge (red dot) with a pulsing ring effect. The ring expands outward with fading opacity, drawing attention to the badge.

## Features
- Red dot badge positioned at top-right of icon
- Pulsing ring animation that expands outward
- Ring fades out as it scales
- Dark theme with icon container

## Usage
Set `--pulse` to `1` on the `.bdp-container` element via `style.setProperty()` to start the pulse animation. Set to `0` to stop.
