# Breadcrumb Current Highlight Component

A CSS breadcrumb component with an animated current-page highlight for the **EaseMotion CSS** examples collection. Click any breadcrumb item to make it active — the active item pulses with a glowing underline animation.

## Features

- Interactive click-to-set-active behavior
- Smooth underline hover transition on all items
- Pulse glow animation on the active item using `@keyframes`
- Google Fonts Inter integration
- Light and dark mode support
- `prefers-reduced-motion` respect

## Folder Structure

```text
submissions/examples/ease-breadcrumb-current-highlight-ij/

```

## Usage

Open `demo.html` in a browser. Click any breadcrumb item to highlight it as the current page. The active item animates with a pulsing underline glow effect.

## Technologies

- HTML5
- CSS3 (Flexbox, custom properties, `@keyframes`, media queries)
- Google Fonts (Inter)
- Vanilla JavaScript (click event handling)
