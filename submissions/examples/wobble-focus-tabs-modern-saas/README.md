# Wobble Focus Tabs - Modern SaaS Layout

A responsive, pure CSS tabs component with a smooth wobble animation on hover and keyboard focus.

## Features

- Pure HTML & CSS
- Responsive layout
- Keyboard accessible
- Customizable with CSS variables
- No JavaScript required

## Usage

```html
<div class="tabs">
    <button class="tab active">Dashboard</button>
    <button class="tab">Analytics</button>
    <button class="tab">Projects</button>
    <button class="tab">Settings</button>
</div>
```

## Customization

Modify the CSS variables:

```css
:root{
  --primary:#2563eb;
  --duration:.45s;
  --radius:10px;
}
```
