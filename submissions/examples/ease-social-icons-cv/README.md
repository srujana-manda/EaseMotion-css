# Animated Social Media Icons (`ease-social-icons-cv`)

A lightweight animated social media icon component built using **HTML5** and **Pure CSS**.

## Features

- Floating animation
- Hover lift effect
- Shimmer hover animation
- Multiple social icon styles
- Responsive layout
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-social-icons-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

Works in all modern browsers.
