# Pulse Active Tabs - Neumorphic Soft Layouts


A pure CSS animated Pulse-Active Tabs component
designed for modern neumorphic soft UI designs.


## Features

- Pure CSS animation
- No JavaScript required
- Neumorphic styling
- Responsive design
- Keyboard focus support
- Custom CSS variables
- Smooth pulse active effect


## Usage


```html
<div class="tabs">

<button class="tab active">
Dashboard
</button>

<button class="tab">
Profile
</button>

</div>
