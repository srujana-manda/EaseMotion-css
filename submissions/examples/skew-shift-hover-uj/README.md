# Skew Shift Hover

## What does this do?

It creates a hover effect that skews, rotates, and translates an element slightly upwards while projecting a deeper shadow, conveying a strong sense of tactile depth.

## How is it used?

Apply the `skew-shift-card-uj` class to your card, list item, or grid element:

```html
<div class="skew-shift-card-uj">
  <h3>Card Title</h3>
  <p>Card content description.</p>
</div>
```

## Why is this useful?

This effect offers a premium hover micro-interaction that enhances visual focus on clicked or inspected elements. It functions without JavaScript, avoids page reflows by utilizing compositor-friendly CSS properties, and disables motion automatically when `prefers-reduced-motion` is active.
