# Ease API Latency Distribution Panel

## Overview

A responsive HTML and CSS component for visualizing API latency percentiles, endpoint performance, and response health.

## Features

- P50, P95 and P99 latency metrics
- Endpoint monitoring
- Latency spike detection
- Status badges
- Responsive layout
- Pure HTML & CSS
- Lightweight animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

Provides a reusable UI component for API observability dashboards, performance monitoring platforms, and cloud-native applications.
