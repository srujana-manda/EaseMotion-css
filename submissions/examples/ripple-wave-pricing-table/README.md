# Ripple-Wave Pricing Table

A pure CSS animated pricing table with a smooth ripple-wave hover effect.

## Features

- Pure HTML and CSS
- Ripple wave animation on hover
- Responsive pricing layout
- Lightweight and framework free
- Accessible reduced-motion support

## Usage

Add pricing cards using:

```html
<div class="pricing-card">
    <h2>Plan Name</h2>
    <h3>$Price</h3>
    <p>Description</p>
</div>
