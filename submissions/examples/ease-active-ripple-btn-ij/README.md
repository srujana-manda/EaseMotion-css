# ease-active-ripple-btn

A button with a click ripple effect.

## Features
- Ripple originates at click position
- JavaScript sets `--ripple-x` and `--ripple-y` custom properties
- CSS keyframe animation drives the ripple expansion and fade

## CSS Custom Property Control
Set `--ripple-x` and `--ripple-y` (percentage values) on the button to control the ripple origin.
