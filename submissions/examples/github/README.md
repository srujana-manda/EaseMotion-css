
# 🌐 EaseMotion Glass Navbar

A modern, responsive, glassmorphism-inspired navigation bar built using HTML and CSS for the EaseMotion CSS component library.

## ✨ Features

- 🎨 Modern glassmorphism design
- 🌙 Dark theme inspired by the EaseMotion demo
- 📱 Fully responsive layout
- 🔗 GitHub repository button
- 📖 Documentation link
- 🌟 Smooth hover animations
- 🟣 Animated underline effect
- ☀️ Theme toggle button UI
- ⚡ Lightweight and easy to customize

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/saptarshi-coder/easemotion-css.git
```

2. Navigate to the project folder.

3. Open `demo.html` in your browser or use Live Server.
