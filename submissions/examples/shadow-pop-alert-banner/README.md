# Shadow Pop Alert Banner

A responsive **Shadow Pop Alert Banner** built with pure CSS for the EaseMotion CSS library.

## Features

- Pure CSS pop-in animation
- Soft shadow effect
- Responsive layout
- Accessible structure
- No JavaScript required

## Usage

Include the stylesheet and use the following HTML:

```html
<div class="alert-banner ease-pop">
  <span class="icon">⚠️</span>
  <div class="content">
    <h3>System Update</h3>
    <p>Your analytics dashboard has been successfully updated.</p>
  </div>
</div>
```

## Why EaseMotion CSS?

This component demonstrates reusable, animation-first UI using CSS variables, keyframes, and smooth transitions while remaining lightweight and easy to customize.
