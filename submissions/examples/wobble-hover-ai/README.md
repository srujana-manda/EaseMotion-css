# Wobble Hover Effect

A simple CSS animation that makes a button "wobble" left and right when hovered over.

### Usage
Apply the class `.wobble-hover-btn` to any button element.

```html
<button class="wobble-hover-btn">Hover Me</button>
```

### Why is this useful?
This adds an eye-catching micro-interaction to call-to-action buttons, grabbing the user's attention and encouraging them to click, fitting perfectly into EaseMotion's philosophy of simple yet engaging animations.
