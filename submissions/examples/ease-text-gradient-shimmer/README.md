# Text Gradient Shimmer

A gradient text shimmer animation — pure CSS.

## Features
- Animated gradient sweep on text
- Configurable gradient colors via `--shimmer-gradient`
- Configurable speed via `--shimmer-speed`
- Uses `background-clip: text` for text-only gradient

## Usage
```html
<h2 class="shimmer-text">Shimmer Heading</h2>
```

## Browser Support
- Chrome 4+, Firefox 5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
