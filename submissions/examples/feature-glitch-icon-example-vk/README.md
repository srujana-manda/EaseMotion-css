# Glitch Icon Example

A lightweight CSS-only glitch icon animation that creates a digital distortion effect when the icon is hovered or focused.

## Features

- Pure HTML & CSS
- Smooth glitch animation
- Keyboard accessible
- No JavaScript required
- Supports `prefers-reduced-motion`

## Files

```
demo.html
style.css
README.md
```

## Usage

Open `demo.html` in a browser.

Hover over or focus on the icon to see the glitch effect.

## Accessibility

- Uses a semantic `<button>` element.
- Includes an `aria-label`.
- Respects the user's reduced-motion preference.

## Use Cases

- Buttons
- Interactive dashboards
- Status indicators
- Gaming interfaces
- Modern landing pages
