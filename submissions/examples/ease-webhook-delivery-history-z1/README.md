# Ease Webhook Delivery History

## Overview
A CSS-only component for displaying webhook delivery logs with response status and retry information.

## Features
- Delivery history cards
- HTTP status badges
- Responsive design
- CSS variables
- Pure HTML & CSS

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` in any modern browser.

## Browser Support
Chrome, Firefox, Edge, Safari

## Responsive
Optimized for desktop, tablet, and mobile devices.

## Why EaseMotion CSS?
Provides a reusable animation-first component for webhook monitoring, automation platforms, backend dashboards, and API management systems.
