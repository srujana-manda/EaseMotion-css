```md
# Office E-Commerce Landing Page

## Overview

A productivity-focused office e-commerce landing page built entirely with EaseMotion CSS classes and responsive layouts.

## Features

- Split-screen hero with clean desk setup imagery
- Product categories section
- Work-from-home setup guides
- Bundle deal promotional banner
- Customer reviews
- Business accounts CTA
- Responsive footer

## Tech Stack

- HTML5
- CSS3
- EaseMotion CSS

## Usage

Open `demo.html` directly in a browser. No build process or dependencies are required.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## EaseMotion Classes Showcased

- `ease-fade-in`
- `ease-fade-in-up`
- `ease-slide-down`
- `ease-slide-left`
- `ease-slide-right`
- `ease-hover-lift`
- `ease-hover-grow`
```
