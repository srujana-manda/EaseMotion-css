# Fusion Energy Reactor Telemetry Monitor — Phase #858

A high-energy fusion reactor control dashboard. Built entirely with **HTML5**, **CSS3**, and **vanilla JavaScript**. Part of the EaseMotion CSS design system showcase.

---

## Features

- Animated plasma core with spinning rings
- Hexagonal grid overlay toggle
- Live status indicator
- Circular vital gauges with animated fills
- Real-time waveform canvas animation
- Subsystem health cards with status colors
- Interactive reactor controls (Pulse, SCRAM)
- Fuel mix gauge with animated segments
- Full-screen SCRAM overlay with shake
- Responsive layout
- Reduced motion support
- Zero external dependencies
