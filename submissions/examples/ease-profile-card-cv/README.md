# Animated Profile Card (`ease-profile-card-cv`)

A responsive profile card component built using **HTML5** and **Pure CSS**.

## Features

- Floating avatar animation
- Hover lift effect
- Shimmer hover animation
- Profile statistics section
- Animated action buttons
- Responsive layout
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-profile-card-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

Works in all modern browsers.
