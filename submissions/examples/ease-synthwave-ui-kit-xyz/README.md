# Synthwave UI Kit (`.ease-synth-*-xyz`)

A retro-futuristic, Cyberpunk/Synthwave inspired UI kit built entirely with pure CSS. It features a moving perspective grid, a classic sliced neon sun, glitch text effects, and glowing interactive elements.

## Features

- 🌅 **Sliced Neon Sun**: A classic synthwave sun created using layered CSS gradients.
- 🌐 **Moving Perspective Grid**: An infinite scrolling grid using `perspective` and `transform`.
- 📺 **Glitch Text Effect**: A pure CSS glitch animation using `text-shadow` and `transform`.
- ✨ **Glowing Buttons**: Neon borders with smooth, bouncy hover transitions.
- 📐 **Angled Cards**: Cards using `clip-path` for a futuristic, cut-corner aesthetic.
- 🎢 **Premium Easing**: Uses custom `cubic-bezier` curves for natural, snappy motion.

## Usage

Copy the HTML structure and the CSS classes into your project. The background elements (`.ease-synth-bg-xyz`) are fixed, so they will cover the entire viewport.

```html
<!-- Example: Glowing Button -->
<button class="ease-synth-btn-xyz ease-synth-btn-primary-xyz">Enter the Grid</button>

<!-- Example: Glitch Text -->
<h1 class="ease-synth-glitch-xyz" data-text="YOUR TEXT">YOUR TEXT</h1>
