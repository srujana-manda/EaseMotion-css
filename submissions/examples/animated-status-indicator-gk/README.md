# Animated Status Indicator System

## Overview

A modern collection of animated status indicators built using pure HTML and CSS.

The component demonstrates reusable status animations for dashboards, chat applications, admin panels, and monitoring systems.

---

## Features

- Online Pulse Animation
- Away Breathing Animation
- Busy Ripple Animation
- Syncing Spinner
- Offline State
- Responsive Layout
- Pure HTML & CSS
