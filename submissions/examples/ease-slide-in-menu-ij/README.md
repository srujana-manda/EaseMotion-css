# Slide In Menu

Menu slides in from off-screen with easing. Items stagger into view with individual delays.

## Features

- Off-screen slide-in animation
- Staggered item reveal via `--sim-i`
- Toggle via `.sim-open` class
- Smooth cubic-bezier easing

## Usage

Toggle `.sim-open` class on `.sim-menu`. JS handles click toggling. CSS handles transitions.
