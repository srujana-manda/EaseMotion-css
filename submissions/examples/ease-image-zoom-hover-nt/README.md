# Ease Image Zoom Hover (NT)

## What does this do?

Creates an image that smoothly zooms when hovered while displaying a centered zoom icon overlay.

## How is it used?

```html
<div class="zoom-card">
    <img src="image.jpg" alt="Image">
    <div class="overlay">
        <i class="fa-solid fa-magnifying-glass-plus"></i>
    </div>
</div>
```

Link the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

## Why is it useful?

This component provides a clean and interactive hover effect for image galleries, portfolios, product cards, and blog thumbnails. It enhances the user experience while remaining lightweight and implemented entirely with CSS.
