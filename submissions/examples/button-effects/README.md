# Button Hover & Pulse Effects

यह कलेक्शन बटनों के लिए विशेष है, जिनमें hover effects और continuous animations शामिल हैं।

## Effects

| क्लास | टाइप | विवरण |
|--------|-------|--------|
| `.btn-gradient` | Hover | बैकग्राउंड gradient बदलेगा और स्केल बढ़ेगा। |
| `.btn-shadow` | Hover | शैडो बढ़ेगा और बटन ऊपर आएगा। |
| `.btn-pulse` | निरंतर | बटन में पल्सिंग एनिमेशन चलता रहेगा। |

## उपयोग कैसे करें
अपने HTML में इन क्लासेस को बटनों पर लगाएं। 
उदाहरण:
```html
<button class="btn-gradient">Click Me</button>
