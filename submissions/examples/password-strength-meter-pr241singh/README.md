# Password Strength Meter Component

A modern **Password Strength Meter Component** built using **HTML5** and **CSS3** for the EaseMotion CSS examples collection.

---

## ✨ Overview

This component provides a visually appealing password strength meter with animated progress indicators, validation rules, multiple design variants, and responsive layouts. It demonstrates how CSS can enhance authentication interfaces with smooth animations and modern styling.

---

## 🚀 Features

- 🔐 Modern password input field
- 📊 Animated password strength meter
- 🟢 Weak, Medium, and Strong strength states
- ✅ Password requirement checklist
- 🎨 Filled variant
- 🖋️ Outline variant
- 🪟 Glassmorphism variant
- ✨ Hover and focus animations
- 💡 Smooth progress bar transitions
- 📱 Fully responsive layout
- 🌙 Dark mode support
- ♿ Reduced motion support
- 🎯 Clean, reusable component structure
