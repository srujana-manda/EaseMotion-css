# Interactive Bento Grid Feature Showcase

This submission showcases a modern, trendy **"Bento Grid" layout** (popularized by Apple and modern SaaS landing pages). It demonstrates how to use **EaseMotion CSS** to create beautiful, staggered entrance animations and smooth, tactile hover interactions on an asymmetric grid of feature cards.

## ✨ Features
- **Asymmetric Grid Layout**: Uses CSS Grid to create visually appealing cards of varying sizes (large, wide, standard).
- **Staggered Entrances**: Cards fade and slide up one by one on page load, creating a premium, cinematic feel.
- **Interactive Elements**: 
  - The "Lightning Fast" card features a mini bar-chart that highlights on hover.
  - The "Zero Config" card features a sleek code snippet UI.
- **EaseMotion Integration**: 
  - `ease-fade-in` & `ease-slide-up`: Base entrance animations for all cards.
  - `ease-delay-100` to `ease-delay-500`: Creates the staggered, waterfall loading effect.
  - `ease-hover-lift` & `ease-duration-fast`: Provides smooth, snappy, and tactile feedback when hovering over any card.

## 🚀 How to Use
1. Open `demo.html` in any modern web browser.
2. Watch the cards elegantly stagger into view from the bottom.
3. Hover over any card to see the smooth `ease-hover-lift` effect and subtle border color transitions.
4. Hover over the large "Lightning Fast" card to see the mini bar-chart animate and highlight.

## 📂 Files
- `demo.html`: Standalone, responsive HTML structure.
- `style.css`: Custom CSS Grid layout, dark theme styling, and interactive inner elements.
