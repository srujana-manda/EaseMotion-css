# Wiggle Hover Icon (#31842)

An icon that playfully wiggles/shakes when hovered, using CSS `:hover` with small rotation keyframes.

## Features
- Pure CSS – no JavaScript required
- Rotational wiggle with multi-step keyframes
- Smooth `ease-in-out` timing on hover

## Usage
Add the `wiggle` class to any icon element. Hover to trigger the animation.
