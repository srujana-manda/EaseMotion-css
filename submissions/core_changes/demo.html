Replace Hardcoded outline-offset with CSS Variable

Issue #12453 — components/buttons.css hardcodes outline-offset: 2px instead of using a CSS variable.

Demo

Tab through the buttons to see focus outlines controlled by --ease-outline-offset.

Override Demo

The section below uses --ease-outline-offset: 8px:

How It Works

:root { --ease-outline-offset: 2px; }

.ease-btn:focus-visible {
  outline-offset: var(--ease-outline-offset, 2px);
}

Note: No core files modified. This submission proposes the change.
