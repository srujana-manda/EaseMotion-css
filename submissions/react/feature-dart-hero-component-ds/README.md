# Dart Hero Component

A reusable React hero component that applies a smooth dart animation to attract user attention.

## Features

- Smooth dart entrance effect
- Beginner-friendly React component
- Lightweight and reusable
- Supports reduced-motion preferences
- Suitable for banners and landing pages

## Usage

```jsx
import DartHero from "./DartHero";

<DartHero>
  Welcome to EaseMotion CSS
</DartHero>;
```

## Accessibility

The component respects:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations when requested by users.

## Use Cases

- Landing page hero sections
- Promotional banners
- Attention-grabbing announcements
- Call-to-action sections
