# Avatar Border Glow
Hover the avatars to trigger a border glow effect with a tooltip. Pure CSS, no JavaScript.
## Features
- Dark theme with `--dp-bg` custom property for color
- Scale transform on hover with glow box-shadow
- Tooltip with arrow indicator via `::before`
- `prefers-reduced-motion` support
## Usage
Set `--dp-bg` on each `.dp-item` for the avatar and glow color. Hover triggers both the glow and tooltip.
