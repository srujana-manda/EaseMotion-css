# Auto Repair Shop Landing Page — Precision Auto Care

**Issue:** [#21811 — Auto Repair Shop Landing Page](https://github.com/SAPTARSHI-coder/EaseMotion-css/issues/21811)

A professional landing page for a family-owned auto repair shop. Features a dark automotive-themed hero, ASE-certified technician highlights, transparent pricing with specials, customer reviews, and online booking with location info.

## Structure

```
submissions/core_changes-issue-21811/

└── README.md     # This file
```

## Sections

- **Navbar** — Glass effect with sticky positioning, service links, and CTA button
- **Hero** — High-impact dark gradient with headline, badge, and dual CTAs
- **Services** — 6 service cards with icons (oil change, brakes, engine, AC, tires, diagnostics)
- **Why Us** — Stats grid (35+ years, 15k+ cars, 4.9 rating, 12 techs) + trust-building cards
- **Pricing** — 3 pricing cards with Oil Change Special ($29.95), Brake Service ($149), Tune-Up ($199)
- **Reviews** — Customer testimonials with vehicle badges
- **Booking** — Appointment form with vehicle/service/date fields + location sidebar (address, hours, contact, shuttle info)
- **Footer** — 4-column footer with services, quick links, payment options

## Theme

- **Primary color:** `#dc2626` (automotive red)
- **Background:** Dark gradient (hero) / clean light (content)
- **Typography:** Inter (Google Fonts)
- **Animations:** `ease-slide-up`, `ease-hover-glow`, `ease-pulse`, `ease-card-lift`
