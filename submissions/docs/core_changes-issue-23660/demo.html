Screen Reader Accessibility Utilities

Classes to support WCAG compliance by hiding items visually while remaining parsed by screen-readers, plus skip-link and layout override helpers.

1. Skip Link (Focus-Triggered Visibility)

Skip links are essential for keyboard-only users to bypass large navigation menus. This skip-link is hidden by default but becomes visible when you focus it via the keyboard.

2. Accessible Icon Buttons

Icon-only buttons have no visual text, which makes them unreadable by screen readers unless text is wrapped in .sr-only.

This button is visually compact, but a screen reader will read "Delete item, button" instead of just "button".
