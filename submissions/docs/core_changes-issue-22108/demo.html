Reference

All Utility Classes

Position

ClassValueDescription
.ease-staticstaticDefault flow positioning
.ease-relativerelativePositioned relative to self
.ease-absoluteabsolutePositioned relative to nearest positioned ancestor
.ease-fixedfixedPositioned relative to viewport
.ease-stickystickyHybrid relative/fixed based on scroll

Inset (Edge Positioning)

ClassValueDescription
.ease-inset-0inset: 0Stretch to all four edges
.ease-top-0top: 0Anchor to top edge
.ease-right-0right: 0Anchor to right edge
.ease-bottom-0bottom: 0Anchor to bottom edge
.ease-left-0left: 0Anchor to left edge

Overflow

ClassValueDescription
.ease-overflow-autoautoScroll only when needed
.ease-overflow-hiddenhiddenClip overflow, no scroll
.ease-overflow-visiblevisibleAllow overflow (default)
.ease-overflow-scrollscrollAlways show scrollbar
.ease-overflow-clipclipClip overflow like hidden but no scroll container
.ease-overflow-x-autooverflow-x: autoHorizontal scroll only
.ease-overflow-x-hiddenoverflow-x: hiddenClip horizontal overflow
.ease-overflow-x-scrolloverflow-x: scrollAlways show horizontal scrollbar
.ease-overflow-y-autooverflow-y: autoVertical scroll only
.ease-overflow-y-hiddenoverflow-y: hiddenClip vertical overflow
.ease-overflow-y-scrolloverflow-y: scrollAlways show vertical scrollbar
