ease-toggle

CSS-only checkbox-based toggle switch for boolean settings and preferences.

Default

Checkbox-based toggle with smooth knob animation and label.

.ease-toggle

Sizes

.ease-toggle-sm / .ease-toggle-lg

Colors

.ease-toggle-success / .ease-toggle-danger

Disabled State

Toggles with disabled attribute show reduced opacity and no pointer events.

Use Cases

Usage

<label class="ease-toggle">
  <input type="checkbox" checked />
  <span class="ease-toggle-track">
    <span class="ease-toggle-knob"></span>
  </span>
  <span class="ease-toggle-label">Dark Mode</span>
</label>

<!-- Sizes: .ease-toggle-sm, .ease-toggle-lg -->
<!-- Colors: .ease-toggle-success, .ease-toggle-danger -->
<!-- Disabled: add disabled attribute to <input> -->
