ease-rating

A CSS-only star rating component with interactive hover preview, half-star precision, read-only mode, and size/color variants.

Interactive

Click to Rate

Click a star to select a rating. Hover to preview — stars fill up to the hovered position.

.ease-rating
Sizes

Small, Default & Large

.ease-rating-sm / .ease-rating / .ease-rating-lg
Colors

Semantic Color Variants

.ease-rating-warning / -success / -danger / -info
Read-only

Display Only

Use .ease-rating-readonly for non-interactive display. Filled stars use the .ease-rating-filled class.

.ease-rating.ease-rating-readonly
With Value

Rating + Count

Pair with .ease-rating-value for a numeric display.

3.0 (42 reviews)
Usage

HTML Structure

<!-- Interactive (radio group, reversed order) -->
<div class="ease-rating">
  <input type="radio" name="rating" id="s5" value="5" />
  <label for="s5"></label>
  <input type="radio" name="rating" id="s4" value="4" />
  <label for="s4"></label>
  <input type="radio" name="rating" id="s3" value="3" />
  <label for="s3"></label>
  <input type="radio" name="rating" id="s2" value="2" />
  <label for="s2"></label>
  <input type="radio" name="rating" id="s1" value="1" checked />
  <label for="s1"></label>
</div>

<!-- Read-only -->
<div class="ease-rating ease-rating-readonly">
  <label class="ease-rating-filled"></label>
  <label class="ease-rating-filled"></label>
  <label class="ease-rating-filled"></label>
  <label></label>
  <label></label>
</div>

<!-- Sizes & colors -->
<div class="ease-rating ease-rating-sm ease-rating-warning">...</div>
<div class="ease-rating ease-rating-lg ease-rating-success">...</div>
