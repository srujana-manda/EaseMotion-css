Token Variables

Duration, easing, colour, and spacing tokens mapped to SCSS variables and CSS custom properties.

$ease-duration-base: 300ms;
$ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1);
_mixins.scss

ease-fade-up Mixin

One-line fade-up animation with configurable duration and spring delay for any element.

@include m.ease-fade-up(
  $duration: 500ms,
  $delay: 200ms
);
_mixins.scss

ease-stagger Mixin

Automatically staggers nth-child animations with a configurable base delay and child count.

@include m.ease-stagger(
  $base-delay: 100ms,
  $count: 6
);
Layout.astro

Global SCSS in Astro

Load global styles with is:global to bypass Astro component scoping.

<style is:global lang="scss">
  @use '../styles/global.scss';
</style>
astro.config.mjs

Auto-Import Variables

Use additionalData in Vite config to auto-import variables in every SCSS file.

additionalData: `
  @use 'src/styles/variables' as v;
`

ease-safe-motion

Wraps animations in a prefers-reduced-motion media query automatically.

@include m.ease-safe-motion {
  @include m.ease-fade-up;
}
