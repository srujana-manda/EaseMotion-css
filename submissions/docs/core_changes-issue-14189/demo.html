Click Ripple

Material-style ripple effect on button click — Issue #14189

Variants
Sizes
Without ripple (comparison)
How it works: Add .ease-btn-ripple to any button. On click, a <span class="ease-ripple"> element is injected at the click position via JS, scales from 0 to 4x with easeRipple keyframe (0.6s ease-out), and is removed after animation completes.

Proposed core changes: Add .ease-btn-ripple modifier to components/buttons.css and @keyframes easeRipple to core/animations.css.
