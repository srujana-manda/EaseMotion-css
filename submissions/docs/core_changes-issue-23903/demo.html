ease-alert

CSS-only notification banner component with color variants, sizes, styles, dismiss, icon, and animated transitions.

Features

6 Color Variants

Info, success, warning, danger, neutral, accent.

3 Styles

Filled (default), outline, subtle (no border).

4 Accent Borders

Left, top accent border + bordered edges.

2 Sizes

Default, sm, lg. Custom via CSS properties.

Dismissible

Close button with absolute positioning.

Animated

Slide-in/out, progress bar, auto-dismiss ready.

Usage

<!-- Basic alert -->
<div class="ease-alert ease-alert--info">
  <span class="ease-alert__icon">&#9432;</span>
  <div class="ease-alert__content">
    <div class="ease-alert__title">Title</div>
    <div class="ease-alert__message">Message text.</div>
  </div>
</div>

<!-- Color variants -->
<div class="ease-alert ease-alert--success">...</div>
<div class="ease-alert ease-alert--warning">...</div>
<div class="ease-alert ease-alert--danger">...</div>

<!-- Styles -->
<div class="ease-alert ease-alert--info ease-alert--outline">...</div>
<div class="ease-alert ease-alert--info ease-alert--subtle">...</div>

<!-- Sizes -->
<div class="ease-alert ease-alert--sm">...</div>
<div class="ease-alert ease-alert--lg">...</div>

<!-- Dismissible -->
<div class="ease-alert ease-alert--dismissible">
  ...
  <button class="ease-alert__close">&times;</button>
</div>
