# ease-stepper — Stepper Progress Bar with Animated Pulse Nodes

## What does this do?

A React component for a multi-step progress indicator with completed/active/pending states and a pulsing animation on the current step node.

## How is it used?

```jsx
import StepperProgress from './StepperProgress';

<StepperProgress current={2} />
```

### Props

| Prop | Type | Default |
|---|---|---|
| current | number | 1 |
