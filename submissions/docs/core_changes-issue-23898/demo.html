No Scrolling

Scrolling is completely disabled via overflow: hidden.

.ease-scrollbar-none

This content is clipped and cannot be scrolled into view. Overflow is hidden entirely, so any excess content is simply not visible to the user of the page.

Thin Scrollbar

Compact 6px scrollbar with rounded thumb.

.ease-scrollbar-thin

Scrollbar is thinner (6px) with a rounded thumb in the neutral color scheme of the theme.

Hover over the scrollbar to see the thumb highlight effect on supported browsers.

This is ideal for compact UIs where a full-width scrollbar would be intrusive.

Custom Scrollbar

Fully themed scrollbar using CSS custom properties. Style the track, thumb, and hover states.

.ease-scrollbar-custom

Custom scrollbar with themed track and thumb colors. Uses CSS custom properties for easy theming.

--ease-scrollbar-track: background of the track area. Defaults to dark slate.

--ease-scrollbar-thumb: color of the draggable thumb. Defaults to medium gray.

--ease-scrollbar-thumb-hover: color on hover. Brightens for visibility.

The thumb has a 2px transparent border with background-clip for a clean look.

Scroll down to see more content that demonstrates the custom scrollbar styling.

Auto (Reset)

.ease-scrollbar-auto restores the default browser scrollbar. Useful to override a hide class.

.ease-scrollbar-auto

Applied via .ease-hide-scrollbar.ease-scrollbar-auto on the same element to restore default scrollbar.

Usage

<!-- Hide scrollbar, keep scrolling -->
<div class="ease-hide-scrollbar">Scrollable content...</div>

<!-- Hide horizontal scrollbar only -->
<div class="ease-hide-scrollbar-x">Wide content...</div>

<!-- Thin scrollbar -->
<div class="ease-scrollbar-thin">Content with thin scrollbar...</div>

<!-- Custom themed scrollbar -->
<div class="ease-scrollbar-custom">Custom scrollbar...</div>

<!-- Responsive: hide on desktop only -->
<div class="ease-lg:hide-scrollbar">Content...</div>

<!-- Override/reset -->
<div class="ease-hide-scrollbar ease-scrollbar-auto">
  Default scrollbar restored.
</div>
