Alert Component

Inline alert banners for user feedback — success, info, warning, and danger variants.

Color Variants

Info
This is an informational alert. Use it for neutral updates.
Success
Your changes have been saved successfully.
Warning
Your session will expire in 5 minutes.
Error
Something went wrong. Please try again later.

Without Title

This is an info alert without a title.
Operation completed successfully.
Please review your input before continuing.
Unable to process your request.

Dismissible Alerts

Click the × button to dismiss this alert.
Dismissible Success
Your data has been saved. Click × to dismiss.
This warning can be dismissed.
Critical Error
This error alert can be dismissed.

Usage

<!-- Basic alert -->
<div class="ease-alert ease-alert-success">
  <div class="ease-alert-content">
    <div class="ease-alert-title">Success</div>
    Your changes have been saved.
  </div>
</div>

<!-- Dismissible alert -->
<div class="ease-alert ease-alert-warning ease-alert-dismissible">
  <div class="ease-alert-content">Warning message.</div>
  <button class="ease-alert-close" onclick="this.parentElement.remove()">&times;</button>
</div>
