# ease-pricing — Interactive Pricing Tier Switcher (Monthly/Annual)

## What does this do?

A React component for an interactive pricing tier switcher with monthly/yearly toggle and animated card layout.

## How is it used?

```jsx
import PricingSwitcher from './PricingSwitcher';

function App() {
  return <PricingSwitcher />;
}
```

### Props

None — self-contained demo component.
