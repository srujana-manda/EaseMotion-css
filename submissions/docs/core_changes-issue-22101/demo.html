Text Animation Utilities

Typewriter, text-reveal, and text-scramble effects — lightweight CSS and JS for expressive typography.

Typewriter

ease-typewriter

Character-by-character reveal with a blinking cursor. Uses CSS steps() function for smooth typing animation and a blinking right border as the cursor.

Hello, I am a typewriter effect!
3s
.ease-typewriter { --ease-typewriter-chars: N; --ease-typewriter-duration: T; }
Text Reveal

ease-text-reveal

Words animate in sequentially with a upward fade. The JS wraps each word in a <span> with staggered animation delays.

Each word appears one after another

.ease-text-reveal + JS wraps words in .ease-word spans
Shimmer

ease-text-shimmer

An animated gradient sweeps across the text, creating a sleek metallic or glassmorphic shine — perfect for hero headings and brand names.

Shimmer Effect

Scramble

ease-text-scramble

Characters rapidly cycle through random letters before settling on the final text. Hover over the text below to trigger the scramble.

Reference

All Text Animation Classes

ClassTypeDescription
.ease-typewriterCSS + JSStepped character reveal with blinking cursor. Set --ease-typewriter-chars and --ease-typewriter-duration.
.ease-text-revealCSS + JSWord-by-word reveal with stagger. JS wraps words in .ease-word spans with sequential delays.
.ease-text-shimmerCSS onlyAnimated gradient sweep across text (background-clip: text). Add your own gradient colors.
.ease-text-scrambleJS onlyCharacter cycling animation. JS iterates through random chars before settling on final text.

AttributeUsed ByDescription
