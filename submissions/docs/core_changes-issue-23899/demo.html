ease-mix-blend & ease-backdrop

Utility classes for CSS blend modes, background blend modes, isolation, backdrop filters, and preset glass/frost effects.

Features

18 Mix Blend Modes

All standard CSS mix-blend-mode values plus plus-darker/lighter.

16 Background Blend Modes

Background-blend-mode values for multi-background layering.

8 Blur Levels

none, sm, default, md, lg, xl, 2xl, 3xl.

Filter Functions

Brightness, contrast, grayscale, hue-rotate, invert, saturate, sepia, opacity, drop-shadow.

Preset Effects

Frost, darken, warm, cool, vintage, dreamy, dramatic, soft-focus.

Isolation

Isolation utility for controlling blend scope.

Usage

<!-- Mix blend modes -->
<div class="ease-mix-blend-multiply">...</div>
<div class="ease-mix-blend-screen">...</div>

<!-- Backdrop blur -->
<div class="ease-backdrop-blur">...</div>
<div class="ease-backdrop-blur-lg">...</div>

<!-- Backdrop filters -->
<div class="ease-backdrop-brightness-150">...</div>
<div class="ease-backdrop-grayscale">...</div>
<div class="ease-backdrop-hue-rotate-90">...</div>

<!-- Preset effects -->
<div class="ease-backdrop-frost">Glass morphism effect</div>
<div class="ease-backdrop-vintage">Retro photo filter</div>

<!-- Background blend -->
<div class="ease-bg-blend-overlay" style="background: url(img.jpg), linear-gradient(...)">...</div>

<!-- Isolation -->
<div class="ease-isolation-isolate">...</div>
