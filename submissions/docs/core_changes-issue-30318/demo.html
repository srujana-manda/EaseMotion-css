Text Alignment

RTL-aware alignment using .ease-rtl\:ease-text-right and .ease-rtl\:ease-text-left.

Class LTR RTL
.ease-rtl\:ease-text-right Right-aligned يمين
.ease-rtl\:ease-text-left Left-aligned يسار
.ease-text-start Logical start بداية منطقية
.ease-text-end Logical end نهاية منطقية

RTL-Aware Spacing

Inside .ease-rtl, .ease-ml-* / .ease-mr-* use logical properties so spacing flips automatically. Logical property classes (.ease-ms-*, .ease-me-*) work anywhere.
