# ease-scrollbar — Scrollbar Style Utility Classes

## What does this do?

Styles the browser scrollbar with `::-webkit-scrollbar` pseudo-elements. Supports thin/auto width, hidden scrollbar, and themed thumb/track colors.

## How is it used?

```html
<div class="ease-scrollbar-thin ease-scrollbar-thumb-primary">
  ...long content...
</div>
```

### Classes

| Class | Effect |
|---|---|
| `.ease-scrollbar-thin` | Thin scrollbar (6px) |
| `.ease-scrollbar-none` | Hide scrollbar (still scrollable) |
| `.ease-scrollbar-thumb-primary` | Primary color thumb |
| `.ease-scrollbar-thumb-muted` | Muted color thumb |
| `.ease-scrollbar-track-muted` | Muted color track |
| `.ease-scrollbar-rounded` | Rounded scrollbar thumb |
