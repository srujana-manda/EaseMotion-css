ease-autofill

CSS utility classes for styling browser autofill on form inputs

Demo: Autofill Styling

Fill out the form and use browser autofill to see the different styles. The inputs below demonstrate each autofill utility.

Browser applies its default yellow autofill background.
Autofill background is suppressed — looks seamless with the dark theme.
Custom autofill colors via --ease-autofill-bg and --ease-autofill-color.
A different color scheme showing the flexibility of CSS variables.

Usage

<!-- Remove autofill styling --> <input class="ease-autofill-none" autocomplete="email"> <!-- Apply themed autofill colors --> <input class="ease-autofill-theme" style="--ease-autofill-bg: #1e3a5f; --ease-autofill-color: #93c5fd;"> /* Customize globally via CSS */ :root { --ease-autofill-bg: #1e293b; --ease-autofill-color: #e2e8f0; }
