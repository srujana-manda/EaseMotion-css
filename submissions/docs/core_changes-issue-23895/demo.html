ease-select-none

CSS user-select and user-drag utility classes for controlling text selection, copy behavior, and element dragging.

Features

5 User-Select Values

none, text, all, auto, contain with webkit prefixes for Safari.

Inheritance Control

Child-targeting variants (select-none-children, select-text-children) for scoped selection on containers.

Drag Control

ease-drag-none, ease-drag-auto, ease-drag-element for native drag behavior.

Aliases

ease-no-select, ease-no-copy, ease-copy-none, ease-copy-all for intuitive naming.

Usage

<!-- Prevent text selection on UI elements -->
<button class="ease-select-none">Click me</button>

<!-- Allow copying code blocks -->
<pre class="ease-select-all">Select all code easily</pre>

<!-- Disable selection on drag handles -->
<div class="ease-drag-none" draggable="true">
  Drag handle (no text selection)
</div>

<!-- Disable children selection from container -->
<div class="ease-select-none-children">
  <p>Can't select this</p>
  <span>Or this</span>
</div>
