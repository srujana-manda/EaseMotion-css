Aspect Ratio Utilities

Maintain consistent proportions in images, videos, and embeds with a single class — no padding-bottom hacks needed.

Ratios

Available Aspect Ratios

Each box has a different aspect-ratio applied but the same width. The height is determined automatically by the ratio.
