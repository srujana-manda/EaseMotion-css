# What does it do ?
It solves the bug of docs-h3 class elements having poor visiblity during light mode.
# How it is used?
Simply change the css property in docs.css on line 1043 of color to black.
<!-- 
.docs-h3{
    color:black;
}
 -->
 # Why it is useful?
 It improves the visibility of the docs during light mode.
