Core Fix Proposal for Issue #20377

.github/workflows/test.yml defines the full validation suite but only triggers on workflow_dispatch — it never runs automatically on push or pull request.

Before (Manual Only)

on: workflow_dispatch:
👤 Developer pushes code
↓
ci.yml runs (partial lint)
↓
❌ test.yml skipped — manual only
↓
Merge without full validation

Full test suite never runs automatically

After (Auto + Manual)

on: push: pull_request: workflow_dispatch:
👤 Developer pushes code
↓
ci.yml runs (partial lint)
↓
✅ test.yml runs automatically
↓
Block merge if validation fails

Full suite runs on every relevant change
