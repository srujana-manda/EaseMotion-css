# Aurelia Spa — Luxurious Wellness Retreat Landing Page

1. **What does this do?**  
   A luxurious, tranquil full-page business landing page for a spa and wellness retreat, built entirely with EaseMotion CSS utility classes and component classes — no custom keyframes.

2. **How is it used?**  
   Open `demo.html` directly in any browser. The page links to EaseMotion CSS via CDN and uses only existing `ease-*` classes for all animations, layout, and interactions.

3. **Why is it useful?**  
   Demonstrates how EaseMotion CSS can build complete, production-quality business landing pages with zero custom animations and no build step — just utility classes and components. This wellness theme uses warm amber/gold tones and organic imagery.

## Sections

| Section | Description |
|---------|-------------|
| Hero | Full-viewport hero with warm gradient, shimmering float animation (`ease-float`), entrance animations (`ease-slide-up`, `ease-delay-*`) |
| Treatments | 6 treatment menu cards with icons, descriptions, and `ease-card-lift` hover |
| Packages | 3-tier pricing packages with "Best Value" featured plan, `ease-btn-primary`, `ease-hover-glow` |
| Gallery | 8 facility gallery tiles with gradient backgrounds and hover scale effect |
| Gift Card CTA | Full-width amber gradient call-to-action with `ease-pulse` animation on primary button |
| Testimonials | 3 guest review cards with avatar initials and star ratings |
| Booking | 2-column booking form with form inputs, date/time pickers, treatment dropdown, and amenities list |
| Footer | 4-column footer with brand info, location, hours, and social links |
