What is EaseMotion CSS?
EaseMotion CSS is a zero-dependency, animation-first CSS framework for building polished interfaces with readable class names.
How do I install it?
Install via npm: npm install easemotion-css, or use the CDN: <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/easemotion-css">
Does it have any dependencies?
No dependencies at all. It's pure CSS with optional vanilla JavaScript for components that need interactivity.
