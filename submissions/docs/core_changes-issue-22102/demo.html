Stagger Animation Utility

Sequenced child element animations — a single parent class creates ripple-like reveals without manual delay calculations.

Fade In

ease-stagger-fade-in

Each child fades in sequentially with a 100ms delay between items. The stagger starts when the element is scrolled into view.
