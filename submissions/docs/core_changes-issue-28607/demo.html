How it works

The container establishes a 3D perspective (perspective: 1px). Each layer is positioned absolutely and translated along the Z-axis. Elements further back (translateZ(-2px)) move slower; elements closer (translateZ(1px)) move faster. The scale() compensation keeps layers at their intended visual size.

Uses backface-visibility: hidden for Safari rendering consistency. Reduced motion queries disable all transforms.
