_variables.scss

Token Variables

Duration, easing, colour, and spacing tokens as SCSS variables and CSS custom properties.

$ease-primary: #6366f1;
$ease-spring: cubic-bezier(0.34,1.56,0.64,1);
_mixins.scss

ease-fade-up

Spring-eased entrance animation with configurable duration and delay per component.

@include m.ease-fade-up(
  $duration: 500ms,
  $delay: 200ms
);
_mixins.scss

ease-stagger

Staggers nth-child animations. Works with Svelte each blocks and static lists.

@include m.ease-stagger(
  $base-delay: 120ms,
  $count: 6
);
+layout.svelte

Global Import

Import global.scss once in the root layout — styles apply to every route automatically.

import '$lib/styles/global.scss';
vite.config.js

Auto-Import Variables

Use additionalData so every component can access variables and mixins without @use.

additionalData: `
  @use 'src/lib/styles/variables' as v;
`
Component.svelte

Scoped + Safe Motion

Wrap all animations in ease-safe-motion to respect prefers-reduced-motion automatically.

@include m.ease-safe-motion {
  @include m.ease-fade-up;
}
