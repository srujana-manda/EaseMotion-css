# SkillForge — Online Course Platform Landing Page

1. **What does this do?**  
   A vibrant, modern full-page landing page for an EdTech / online course SaaS platform, built entirely with EaseMotion CSS utility classes and component classes — no custom keyframes.

2. **How is it used?**  
   Open `demo.html` directly in any browser. The page links to EaseMotion CSS via CDN and uses only existing `ease-*` classes for all animations, layout, and interactions.

3. **Why is it useful?**  
   Demonstrates how EaseMotion CSS can build complete, production-quality EdTech landing pages with zero custom animations and no build step — just utility classes and components. This course-marketplace theme uses a vibrant purple/indigo color palette.

## Sections

| Section | Description |
|---------|-------------|
| Hero | Full-viewport hero with deep indigo gradient, floating shimmer, gradient highlight text, entrance animations (`ease-float`, `ease-slide-up`, `ease-delay-*`) |
| Featured Courses | 6 course cards with thumbnails, badges, ratings, pricing, instructor, and duration |
| Categories | 8-category navigation grid with icon, label, and course count |
| Instructor Spotlight | 3 instructor profile cards with avatar initials, company, stats, and `ease-card-lift` |
| Testimonials | 3 student success cards with job placement results and star ratings |
| Pricing | 3-tier subscription (Free, Pro, Team) with "Most Popular" featured plan |
| Newsletter | Full-width indigo CTA with email input and subscribe button |
| Footer | 4-column footer with platform links, company info, legal, and social |
