ease-accordion

A CSS-only accordion component built with native <details> / <summary> elements — zero JavaScript required.

Basic

FAQ Accordion

Click any item to expand its content. The chevron rotates to indicate open state.

.ease-accordion
What is EaseMotion CSS?

EaseMotion CSS is a zero-dependency, animation-first CSS framework. It provides human-readable utility classes for animations, hover effects, components, and layouts — all with built-in dark mode and accessibility support.

Do I need JavaScript?

No. The accordion uses native <details> / <summary> elements and works without any JavaScript. All animations are pure CSS transitions.

How do I customise the colours?

Override the --ease-accordion-* CSS custom properties on the .ease-accordion container or use EaseMotion design tokens like --ease-color-primary, --ease-color-surface, etc.

Pre-opened

Default Open Item

Add the open attribute to any <details> to start expanded.

Getting Started

Include the stylesheet and add the markup. That is it — no build step, no JavaScript, no configuration.

Browser Support

<details> / <summary> is supported in all modern browsers. The accordion degrades gracefully in older browsers — content remains readable without transitions.

Usage

HTML Structure

The accordion uses semantic HTML — no role, aria-expanded, or JS required.

<div class="ease-accordion">

  <details class="ease-accordion-item">
    <summary class="ease-accordion-trigger">
      <span>Title</span>
      <span class="ease-accordion-chevron"></span>
    </summary>
    <div class="ease-accordion-content">
      <p>Collapsible content.</p>
    </div>
  </details>

  <!-- Add open attribute to expand by default -->
  <details class="ease-accordion-item" open>
    <summary class="ease-accordion-trigger">
      <span>Pre-opened Item</span>
      <span class="ease-accordion-chevron"></span>
    </summary>
    <div class="ease-accordion-content">
      <p>This panel starts open.</p>
    </div>
  </details>

</div>
