# Animated Warehouse Logistics Dashboard

1. **What does this do?**  
   An animated logistics dashboard for warehouse operations — shows overview stats, shipment progress, warehouse capacity, delivery timeline, activity feed, notifications, and quick actions.

2. **How is it used?**  
   Open `demo.html` directly in a browser. The dashboard renders with CSS animations — stat cards fade up, progress bars fill in, and the logo icon bounces. The layout adapts responsively on mobile.

3. **Why is it useful?**  
   Demonstrates how EaseMotion CSS can power real-world UI dashboards with smooth animations, dark theme styling, and responsive grid layouts — all with zero JavaScript and no build step.

## Sections

| Section | Content |
|---------|---------|
| **Overview Cards** | Active Shipments (248), Inventory Items (48,250), Dispatches Today (186), Revenue MTD ($1.2M) |
| **Shipment Progress** | Progress bars for 4 active orders with percentage tracking |
| **Warehouse Capacity** | Zone-level capacity utilization (A: 85%, B: 62%, C: 78%, D: 91%) |
| **Delivery Timeline** | 5-step timeline showing stages from pickup to delivery |
| **Logistics Activity** | Real-time activity feed with colored status indicators |
| **Notifications Panel** | Urgent, info, and success notification cards |
| **Quick Actions** | Dispatch Shipment, Add Inventory, Export Report buttons |
