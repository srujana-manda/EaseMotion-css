ease-breakout

Break elements out of their parent container to full viewport width. Uses the 100vw + negative margin technique or CSS Grid container approach.

Features

↔️
Full Breakout
◀️
Left Only
▶️
Right Only
📏
Sizes (sm/lg/xl)
📐
Grid Container
📱
Responsive

Full Breakout (Both Sides)

Breaks out of the container on both sides to span the full viewport width.

Container (600px simulated)

This is the parent container. Content below breaks out of it.

.ease-breakout — Full viewport width

Back inside the container after the breakout.

Left-Only Breakout

Breaks out to the left side only. Content stays aligned to the container on the right.

.ease-breakout-left

This content is inside the container.

.ease-breakout-left — Extends to the left edge

Back in the container.

Right-Only Breakout

Breaks out to the right side only.

.ease-breakout-right
.ease-breakout-right — Extends to the right edge

Breakout Sizes

Small (less width), large, and extra-large breakout variants.

.ease-breakout-sm (slightly beyond container)
.ease-breakout-sm — Slightly wider than container
.ease-breakout-lg (wider breakout)
.ease-breakout-lg — Wider breakout
.ease-breakout-xl (maximum breakout)
.ease-breakout-xl — Maximum width breakout

Grid Container Approach

Alternative method using CSS Grid. Items span columns to break out.

.ease-breakout-container + .ease-breakout-item
Normal content in column 2
.ease-breakout-item spans all 3 columns
Back to normal content

Overlay Approach

Uses absolute positioning + translateX to break out.

.ease-breakout-overlay
.ease-breakout-overlay — Position-based breakout

Usage

<!-- Full breakout -->
<section>
  <div class="container">
    <p>Normal content</p>
    <div class="ease-breakout">
      This spans the full viewport width.
    </div>
  </div>
</section>

<!-- Grid container -->
<div class="ease-breakout-container">
  <div>Normal width</div>
  <div class="ease-breakout-item">Full width</div>
  <div>Normal width</div>
</div>
