ease-z

Z-index scale utility classes with numeric steps, semantic names, negative values, and responsive breakpoint variants.

Features

20+ Numeric Steps

0 through 1000, plus max (2147483647), 9999, and negative values.

10 Semantic Names

Dropdown, sticky, fixed, offcanvas, modal-backdrop, modal, popover, tooltip, toast, notification.

Responsive Prefixes

sm, md, lg, xl, 2xl breakpoint variants for responsive stacking.

Negative Values

Negative z-index values for layering behind parent backgrounds.

Usage

<!-- Numeric -->
<div class="ease-z-50">...</div>
<div class="ease-z-100">...</div>

<!-- Semantic -->
<div class="ease-z-modal">Modal dialog</div>
<div class="ease-z-tooltip">Tooltip</div>
<div class="ease-z-dropdown">Dropdown menu</div>
<div class="ease-z-notification">Notification toast</div>

<!-- Negative -->
<div class="ease-z-negative">Behind parent</div>

<!-- Responsive -->
<div class="ease-z-10 ease-z-md-50">z-10 on mobile, z-50 on md+</div>
