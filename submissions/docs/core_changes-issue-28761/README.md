# ease-magnetic — Magnetic Dock Navigation

## What does this do?

Creates a CSS-only magnetic dock navigation bar where items attract/repel based on hover proximity, simulating a magnetic field effect — no JavaScript needed.

## How is it used?

```html
<nav class="ease-magnetic-dock">
  <a href="#" class="ease-magnetic-item">Home</a>
  <a href="#" class="ease-magnetic-item">Search</a>
</nav>
```

### Classes

| Class | Effect |
|---|---|
| `.ease-magnetic-dock` | Container that establishes the magnetic dock |
| `.ease-magnetic-item` | Individual dock item with magnetic hover effect |
