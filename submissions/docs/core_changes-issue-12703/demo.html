Generated Classes

All classes can be regenerated via the build script:

bash submissions/core_changes-issue-12703/generate.sh > style.css

Usage

<!-- Padding -->
<div class="ease-p-4">Uniform padding</div>
<div class="ease-pt-4 ease-pb-2">Top and bottom padding</div>
<div class="ease-px-6">Horizontal padding only</div>

<!-- Margin -->
<div class="ease-mb-4">Bottom margin</div>
<div class="ease-mx-auto">Centered via auto margin (custom)</div>
