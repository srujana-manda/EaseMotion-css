# Animated Image Gallery & Lightbox

1. What does this do? Provides a responsive CSS Grid image gallery with hover zoom, caption overlay, and a radio-button-controlled fullscreen lightbox with smooth enter animation.
2. How is it used? Place `.gallery-item` divs containing a hidden radio input, a `.gallery-label` trigger, and a `.lightbox` overlay. Clicking an image opens the lightbox; clicking the close button dismisses it.
3. Why is it useful? Image galleries with lightboxes are essential for portfolios, product pages, and photography sites. This component follows EaseMotion CSS's zero-JavaScript philosophy with smooth cubic-bezier animations, reduced-motion support, and dark mode compatibility.
