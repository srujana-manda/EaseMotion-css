ease-progress

CSS-only progress bar component with animated fill, color variants, sizes, stripes, gradients, indeterminate state, vertical orientation, stacked segments, and wrapper with label.

Features

9 Color Variants

Primary, secondary, success, danger, warning, info, accent, pink, orange.

5 Sizes

sm, md (default), lg, xl, 2xl.

Styles

Solid, striped, animated stripes, gradient, indeterminate.

Layouts

Horizontal, vertical, stacked (segmented), wrapper with label.

Label

Overlay percentage label, or side text via wrapper.

Accessible

CSS custom property for value, aria-ready, reduced-motion.

Basic

Set the width via --ease-progress-value CSS custom property.

Usage

<!-- Basic progress bar -->
<div class="ease-progress" style="--ease-progress-value: 50%;">
  <div class="ease-progress__bar"></div>
</div>

<!-- With overlay label -->
<div class="ease-progress" style="--ease-progress-value: 65%;">
  <div class="ease-progress__bar">
    <span class="ease-progress__label">65%</span>
  </div>
</div>

<!-- Color variants -->
<div class="ease-progress ease-progress--success" style="--ease-progress-value: 80%;">
  <div class="ease-progress__bar"></div>
</div>

<!-- Sizes -->
<div class="ease-progress ease-progress--sm" style="--ease-progress-value: 40%;">
  <div class="ease-progress__bar"></div>
</div>
<div class="ease-progress ease-progress--xl" style="--ease-progress-value: 60%;">
  <div class="ease-progress__bar"></div>
</div>

<!-- Styles -->
<div class="ease-progress ease-progress--striped" style="--ease-progress-value: 50%;">
  <div class="ease-progress__bar"></div>
</div>
<div class="ease-progress ease-progress--animated" style="--ease-progress-value: 50%;">
  <div class="ease-progress__bar"></div>
</div>
<div class="ease-progress ease-progress--indeterminate">
  <div class="ease-progress__bar"></div>
</div>

<!-- With wrapper label -->
<div class="ease-progress__wrapper">
  <div class="ease-progress" style="--ease-progress-value: 73%;">
    <div class="ease-progress__bar"></div>
  </div>
  <span class="ease-progress__text">73%</span>
</div>

<!-- Vertical -->
<div class="ease-progress ease-progress--vertical" style="--ease-progress-value: 60%;height:120px;">
  <div class="ease-progress__bar"></div>
</div>
