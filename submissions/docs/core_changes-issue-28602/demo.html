Content Preview

Cards and text adapt automatically when the theme changes.

Sample Card

This card's background, text, and border colors update smoothly when you toggle the theme. The transition is powered by CSS custom properties and the data-theme attribute on the root element.

Feature: localStorage Persistence

Your theme preference is saved to localStorage. Refresh the page and the theme will persist. On first visit, the component respects your system's prefers-color-scheme setting.

Integration

Add the toggle alongside the JS to any page for instant dark mode support. The script automatically initializes all .ease-theme-switcher buttons on the page.

<button class="ease-theme-switcher" aria-label="Toggle theme">
  <span class="ease-theme-icon">
    <svg class="ease-sun-icon">...</svg>
    <svg class="ease-moon-icon">...</svg>
  </span>
</button>
<script src="theme-switcher.js"></script>
