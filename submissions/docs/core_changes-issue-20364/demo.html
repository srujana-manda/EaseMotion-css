Core Fix Proposal for Issue #20364

npm run release:check fails because scripts/validate-package.mjs does a case-sensitive comparison of the repository URL, but the actual URL in package.json uses lowercase.

Bug

manifest.repository?.url?.includes("SAPTARSHI-coder/EaseMotion-css")

package.json has:

saptarshi-coder/easemotion-css

MISMATCH → validation fails

Fix

manifest.repository?.url?.toLowerCase().includes("saptarshi-coder/easemotion-css")

Both lowercased:

saptarshi-coder/easemotion-css ← match → saptarshi-coder/easemotion-css

MATCH → validation passes

See README.md for the proposed code change to scripts/validate-package.mjs.
