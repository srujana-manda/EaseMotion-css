# ease-will-change — Will-Change Utility Classes

## What does this do?

Hints to the browser about upcoming changes via `will-change`, enabling GPU acceleration and optimized rendering for animations and transitions.

## How is it used?

```html
<div class="ease-will-change-transform">Animating transform</div>
<div class="ease-will-change-opacity">Animating opacity</div>
```

### Classes

| Class | Value |
|---|---|
| `.ease-will-change-auto` | `will-change: auto` |
| `.ease-will-change-transform` | `will-change: transform` |
| `.ease-will-change-opacity` | `will-change: opacity` |
| `.ease-will-change-scroll` | `will-change: scroll-position` |
| `.ease-will-change-top` | `will-change: top, left` |
| `.ease-will-change-contents` | `will-change: contents` |
