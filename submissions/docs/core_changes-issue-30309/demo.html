How It Works

Interactive mode: The <input type="range"> sits invisibly over the entire container. On input, it updates --ease-compare-pos which drives the clip-path on .ease-compare-after and the inset-inline-start of .ease-compare-handle.

Structure: .ease-compare > .ease-compare-before + .ease-compare-after + .ease-compare-handle + input.ease-compare-range

Customizing: Set --ease-compare-pos on the container (default 50%). Use .ease-compare-vertical for top/bottom split.
