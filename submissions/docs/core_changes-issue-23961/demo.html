Custom Properties

All drop cap styles respond to these CSS variables:

:root { --ease-drop-cap-size: 3em; --ease-drop-cap-color: #60a5fa; --ease-drop-cap-font: 'Georgia', serif; --ease-drop-cap-weight: 700; }

Usage

<!-- Apply drop cap to a paragraph container --> <div class="ease-drop-cap"> <p>Your text here...</p> </div> <!-- Customize per instance --> <div class="ease-drop-cap-circle" style="--ease-drop-cap-color: #34d399; --ease-drop-cap-size: 4em;"> <p>Circular drop cap with custom color...</p> </div>
