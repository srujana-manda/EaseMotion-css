ease-anchor

CSS utility classes for Anchor Positioning

Visual Demo: Anchor + Positioned Tooltips

The centered box is the anchor. Tooltips positioned around it use the anchor positioning utilities.

Anchor Element (.ease-anchor)
ease-anchor-top Positioned above
ease-anchor-bottom Positioned below
ease-anchor-left Positioned left
ease-anchor-right Positioned right
ease-anchor-center Centered on anchor

How It Works

1. Define the anchor

Add .ease-anchor to the reference element. This sets anchor-name: --ease-anchor.

2. Position relative to anchor

Add .ease-anchor-top, .ease-anchor-bottom, .ease-anchor-left, .ease-anchor-right, or .ease-anchor-center to the positioned element.

3. Adjust offset

Use the --ease-anchor-offset variable to control spacing from the anchor edge (default: 8px).

Usage

<!-- Step 1: Mark the anchor --> <div class="ease-anchor" id="ref">Anchor</div> <!-- Step 2: Position a tooltip relative to it --> <div class="ease-anchor-top">Tooltip above</div> /* Optional: override the offset */ .my-tooltip { --ease-anchor-offset: 16px; }
Browser Support Note: CSS Anchor Positioning is an emerging feature (Chrome 125+). It allows absolutely-positioned elements to attach themselves to anchor elements declaratively, without JavaScript. Check caniuse for the latest support data. The classes provided use the standard anchor-name and position-anchor / anchor() syntax.
