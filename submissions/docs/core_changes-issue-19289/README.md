# Shimmer Skeleton Loading Screen

1. What does this do? Provides pure-CSS shimmer-effect skeleton loading placeholders in three variants — card grid, list items with avatar, and article layout — using animated gradient backgrounds for the shimmer effect.
2. How is it used? Apply skeleton classes like `.s-card`, `.s-list-item`, or `.s-article` with nested `.s-line`, `.s-img`, `.s-avatar` elements. The shimmer animation runs automatically via CSS keyframes.
3. Why is it useful? Skeleton screens improve perceived performance by showing content placeholders while data loads. This component follows EaseMotion CSS's zero-JavaScript philosophy with reduced-motion support and dark mode compatibility.
