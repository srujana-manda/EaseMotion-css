ease-accordion

Pure CSS collapsible panel component using native <details> / <summary> elements. Smooth chevron rotation, hover states, focus-visible outlines, and three variants.

Default

Standard accordion with rounded items, chevron indicator, and hover states.

What is EaseMotion CSS?

EaseMotion CSS is a lightweight, utility-first CSS framework focused on animations, transitions, and motion design. It provides pre-built components, animation utilities, and SCSS mixins for rapid UI development.

How does the accordion work?

The accordion uses native HTML <details> and <summary> elements — no JavaScript required. The chevron rotates 90 degrees when the item is open. Focus-visible outlines ensure keyboard accessibility.

Is it accessible?

Yes. The native <details> element is keyboard accessible by default — users can Tab to focus and press Enter/Space to toggle. The component also includes focus-visible outlines and supports prefers-reduced-motion.

How do I install EaseMotion?

Install via npm:

npm install easemotion-css

Or download the CDN build from the releases page.

Can I use it with React?

Yes. EaseMotion CSS is framework-agnostic. There are also React-specific wrappers available for components like accordion, modal, tabs, and toast.

Does it support dark mode?

Yes. The framework includes prefers-color-scheme support across all components, including the accordion. The component inherits --ease-color-* tokens for seamless theme switching.

Nested Content

Accordion bodies support any HTML — text, lists, code blocks, images, and even nested components.

Feature Overview

Key features of the accordion component:

  • Pure CSS — zero JavaScript required
  • Three size variants: default, small, bordered
  • Animated chevron indicator on open/close
  • Keyboard accessible with focus-visible outlines
  • Respects prefers-reduced-motion
  • Uses --ease-* design tokens throughout
