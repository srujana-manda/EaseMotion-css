ease-image

CSS utility classes for image loading strategies — content-visibility hints

Demo: Image Loading Strategies

These cards demonstrate CSS hints for controlling how images render. Combine these classes with the native loading HTML attribute for best results.

Best Practice: Use these CSS classes together with the HTML loading="lazy" or loading="eager" attribute on <img> tags. The CSS hints (content-visibility) control rendering of the containing element, while the HTML attribute controls when the browser fetches the image source.
.ease-image-lazy

Lazy Image Card

Uses content-visibility: auto with an intrinsic size hint. The browser defers rendering until the card is close to the viewport.

ease-image-lazy

CSS + HTML Lazy

Combined with loading="lazy" on the <img> element for deferred fetching and deferred rendering.

.ease-image-eager

Eager Image Card

Uses content-visibility: visible. The browser renders the element immediately — useful for above-the-fold images.

ease-image-eager

CSS + HTML Eager

Combined with loading="eager" on the <img> for immediate fetching and rendering.

Usage

<!-- Lazy image (deferred rendering + deferred fetch) --> <div class="ease-image-lazy"> <img src="photo.jpg" loading="lazy" alt="Description"> </div> <!-- Eager image (immediate rendering + immediate fetch) --> <div class="ease-image-eager"> <img src="hero.jpg" loading="eager" alt="Hero"> </div> /* Custom intrinsic size for lazy images */ .ease-image-lazy { --ease-image-lazy-size: 400px; }

Recommended HTML Attributes

HTML loading attribute: Use loading="lazy" to defer image fetching until the image is near the viewport. Use loading="eager" (or omit it) to fetch the image immediately.

HTML decoding attribute: Use decoding="async" for off-thread image decoding (improves performance).

HTML fetchpriority attribute: Use fetchpriority="high" for high-priority images (e.g., LCP candidates).

These CSS classes are hints to the browser about rendering. They work best when combined with the appropriate HTML attributes for a complete loading strategy.
