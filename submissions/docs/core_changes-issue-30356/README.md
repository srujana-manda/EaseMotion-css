# improved UX with search and filters

## What does this do?

Demonstrates improved UX with search and filters for a student job board. Part of the core_changes-issue series adding search and filter capabilities.

## How is it used?

Open  in a browser. Interact with the filter controls to see real-time job listing filtering powered by EaseMotion CSS animations.

| Class | Purpose |
|---|---|
|  | Header entrance animation |

## Features

- Focused demo for: improved UX with search and filters
- Real-time job filtering
- Animated card transitions
- Dark mode support
