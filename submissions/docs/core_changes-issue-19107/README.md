# Interactive Docs Setup with Tailwind + EaseMotion

1. What does this do? Provides a minimal, self-contained interactive documentation template using plain HTML, Tailwind CSS (via CDN) for utility-first styling, and EaseMotion CSS animation classes for smooth micro-interactions — with theme toggle, code copy, and expandable sections.

2. How is it used? Open `demo.html` in a browser. No build step or server required. The page demonstrates interactive docs features: theme switching, animated copy-to-clipboard feedback, collapsible sections, and scroll-triggered reveal animations.

3. Why is it useful? Many documentation sites over-engineer with heavy frameworks. This approach keeps everything static, lightweight, and instantly editable while still providing smooth animations and interactivity through EaseMotion CSS classes and small vanilla JS snippets. It aligns with EaseMotion CSS's philosophy of animation-first, composable, and dependency-free design.
