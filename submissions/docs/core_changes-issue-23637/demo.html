ease-stagger

Utility class for sequential child animation delays. Add .ease-stagger to a parent — each direct child gets an incremental animation-delay via nth-child. Variants: 50ms, 100ms (default), 200ms. Also supports custom --ease-stagger-interval.

Usage

<!-- Default stagger (100ms intervals) -->
<div class="ease-stagger">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
</div>

<!-- Named variants -->
<div class="ease-stagger ease-stagger-50">...</div>   <!-- 50ms -->
<div class="ease-stagger ease-stagger-200">...</div>  <!-- 200ms -->

<!-- Custom interval -->
<div class="ease-stagger" style="--ease-stagger-interval: 150ms;">...</div>

How it works

The .ease-stagger class sets --ease-stagger-interval: 100ms on all direct children, then uses nth-child(1) through nth-child(20) selectors to multiply the interval. Works with any animation class applied to children — stagger only controls animation-delay. The !important flag ensures stagger delays override any existing animation-delay set on children.
