# Animated Floating Particles Background

1. What does this do? Creates a decorative CSS-only animated particle background with 20 floating elements that drift upward with organic translateX/Y motion, size and opacity variation, and geometric shape diversity.
2. How is it used? Add a `.particles-bg` container with `.particle` divs using CSS custom properties (`--p-size`, `--p-x`, `--p-dur`, `--p-del`, `--p-color`, `--p-opacity`) for per-particle uniqueness. Content sits above with `z-index: 1`.
3. Why is it useful? Floating particle backgrounds add visual depth to hero sections, landing pages, and login screens without any JavaScript. This follows EaseMotion CSS's philosophy of zero-JS, accessible, declarative motion design.
