# ease-grid-align — Grid Alignment Utility Classes

## What does this do?

Controls alignment of grid items within their cells — `justify-self`, `align-self`, `justify-items`, and `align-items` for grid and flexbox layouts.

## How is it used?

```html
<div class="ease-grid ease-grid-cols-3">
  <div class="ease-justify-self-center">Centered</div>
  <div class="ease-align-self-stretch">Stretched</div>
</div>
```
