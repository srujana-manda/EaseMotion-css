# ease-scrollbar

## What does this do?
Custom scrollbar styling utilities: thin scrollbar, hidden scrollbar (scrollable but invisible), and themeable scrollbar with custom thumb/track colors via `--ease-scrollbar-thumb` and `--ease-scrollbar-track` variables.

## How is it used?
```html
<div class="ease-scrollbar-thin">...</div>
<div class="ease-scrollbar-hide">...</div>
<div class="ease-scrollbar-custom" style="--ease-scrollbar-thumb: #6c63ff;">...</div>
```

## Why is it useful?
Provides consistent scrollbar styling across browsers without repetitive custom CSS.
