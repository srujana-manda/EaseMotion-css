Small

How it works

  • Input collapses to 40px circle; expands to 280px on focus via width transition
  • Transition timing uses --ease-speed-medium (300ms) and --ease-ease-out easing
  • Search icon fades in on focus from opacity: 0.4 to 1 and shifts to brand blue
  • Placeholder text fades in with a 100ms delay after expansion starts
  • Focus ring appears via box-shadow on the input
  • Size variants: .ease-search-wrapper-sm (34px / 220px) and .ease-search-wrapper-lg (48px / 340px)
  • Dark and light mode via [data-theme] CSS custom properties
  • Respects prefers-reduced-motion: reduce — disables all transitions
  • Keyboard accessible: native :focus and :focus-within for expansion
