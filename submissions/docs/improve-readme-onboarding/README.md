# 10-Second Quick Start

Get started with EaseMotion CSS in just a few lines.

```html
<link rel="stylesheet" href="easemotion.min.css">

<button class="ease-btn ease-btn-primary ease-hover-grow">
    Get Started
</button>
```

# Troubleshooting

### Animations are not working

- Make sure the EaseMotion CSS file is properly linked.
- Verify that the class names are spelled correctly.

### CDN is not loading

- Check your internet connection.
- Verify that the CDN URL is correct.

### Scroll animations do not trigger

Some scroll-based animations require **reveal.js**. Ensure it is included and initialized properly.
