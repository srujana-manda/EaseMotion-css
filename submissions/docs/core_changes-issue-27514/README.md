# ease-badge — Badge Notification Pulse Indicator

## What does this do?

A React component for a notification badge with a pulsing ring indicator. Supports danger, success, warning, info variants and sm/md/lg sizes.

## How is it used?

```jsx
import BadgePulse from './BadgePulse';

<BadgePulse count={5} variant="danger" size="md" />
```
