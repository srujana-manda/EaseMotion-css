Dropdown Menu #30305

Hover and click-triggered dropdown menus with smooth animations, RTL support, and accessibility.

Hover Variant

Use .ease-dropdown-hover — opens on hover with :focus-within fallback for keyboard users.

Click Variant

Uses a hidden checkbox (:checked) for click-toggle — no JS required.

Position Variants

Add .ease-dropdown-menu-end to align right, or .ease-dropdown-menu-top to open upward.

Rich Items

Dropdown items support icons, text labels, and keyboard shortcut hints.

Usage Notes

Hover variant: Wrap in .ease-dropdown.ease-dropdown-hover. The trigger must have .ease-dropdown-trigger and the menu must have .ease-dropdown-menu. Keyboard accessible via :focus-within.

Click variant: Use .ease-dropdown.ease-dropdown-click with an <input type="checkbox" class="ease-dropdown-checkbox"> as the first child, and a <label for="..."> as the trigger. Toggle the checkbox to open/close.

Positioning: Add .ease-dropdown-menu-end for right alignment, .ease-dropdown-menu-top to open upward. Default is bottom-left. RTL-aware via inset-inline-* properties.
