Dark Mode Theme Toggle

Add .ease-dark to <html> to enable dark mode. All components inherit the dark palette automatically via CSS custom properties. Uses --ease-color-* design tokens for seamless theming.

Design Tokens

Background

--ease-color-bg
--ease-color-surface

Text

--ease-color-text
--ease-color-muted

Border

--ease-color-border

Component Preview

Buttons

Badges

Primary

Success

Danger

Navigation

Active Default

Usage

<!-- Toggle via class -->
<html class="ease-dark">

<!-- Toggle via JavaScript -->
document.documentElement.classList.toggle('ease-dark');

<!-- Auto dark mode via OS preference (already built into core/variables.css) -->
@media (prefers-color-scheme: dark) { ... }
