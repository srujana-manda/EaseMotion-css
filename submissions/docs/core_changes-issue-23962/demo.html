ease-text-underline

CSS utility classes for text-underline-offset, text-decoration-thickness, and text-decoration-style

Underline Offset

Control the distance of the underline from the text baseline.

Offset 1px .ease-underline-offset-1
Offset 2px .ease-underline-offset-2
Offset 3px .ease-underline-offset-3
Offset 4px .ease-underline-offset-4
Offset auto .ease-underline-offset-auto

Underline Thickness

Control how thick the underline line is.

Thin underline .ease-underline-thin
Thick underline .ease-underline-thick

Underline Style

Different visual styles for the underline.

Wavy underline .ease-underline-wavy
Dotted underline .ease-underline-dotted
Dashed underline .ease-underline-dashed

Combined Example

Classes can be combined for full control.
