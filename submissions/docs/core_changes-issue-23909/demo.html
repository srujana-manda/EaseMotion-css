ease-breadcrumb

Responsive breadcrumb component with auto-separator, four separator styles, and mobile collapse. Built on the existing ease-breadcrumb base.

Default (slash separator)

Auto-separator via ::before pseudo-element — no need to manually insert separators.

Responsive Collapse

Add .ease-breadcrumb-collapsible — on mobile (<640px), intermediate items collapse to "...". Resize the browser to see it in action.

Usage

<!-- Default with auto-separator -->
<nav aria-label="Breadcrumb">
  <ol class="ease-breadcrumb-list">
    <li class="ease-breadcrumb-item">
      <a href="#" class="ease-breadcrumb-link">Home</a>
    </li>
    <li class="ease-breadcrumb-item">
      <span class="ease-breadcrumb-current">Current</span>
    </li>
  </ol>
</nav>

<!-- With chevron separator + collapse -->
<nav class="ease-breadcrumb-chevron ease-breadcrumb-collapsible"
     aria-label="Breadcrumb">
  ...
</nav>
