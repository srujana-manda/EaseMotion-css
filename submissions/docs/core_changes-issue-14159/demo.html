How it works: Each accordion item uses an <input type="radio"> hidden behind a <label> header. When checked, :checked + .ease-accordion-header + .ease-accordion-content transitions max-height from 0 to 500px and opacity from 0 to 1. The radio name attribute ensures only one item is open at a time (exclusive accordion). Arrow icon rotates 180° on open.
