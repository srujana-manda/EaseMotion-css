ease-highlight

CSS utility classes for ::selection and <mark> styling

Selection Highlights

Select text in each paragraph below to see the different highlight colors.

.ease-highlight-primary

This paragraph uses the primary highlight color. Select this text to see a blue selection background with white text. The ::selection pseudo-element is styled using the --ease-highlight-primary-bg and --ease-highlight-primary-color custom properties.

.ease-highlight-warm

This paragraph uses the warm highlight color. Select this text to see an amber selection background with dark text. The --ease-highlight-warm-bg and --ease-highlight-warm-color variables control this appearance.

.ease-highlight-none

This paragraph disables the selection highlight entirely. When you select text here, no background color is applied — the selection is invisible. This can be useful for UI elements where selection should not be visually indicated.

Mark Tag Styling

The <mark> element is styled to stand out against the dark background.

Styled <mark> tag

This is a paragraph with highlighted text using the native <mark> element. The ease-highlight utility also provides a .ease-mark class for custom highlighting when you cannot use the <mark> tag itself.

You can also combine selection styles with marks: this marked text has no selection highlight.

Usage

<!-- Apply selection highlight to a section --> <div class="ease-highlight-primary"> Select this text to see blue highlight </div> <!-- Styled mark tag --> <p> This is <mark>highlighted</mark> text. </p> /* Customize via CSS variables */ :root { --ease-highlight-primary-bg: #8b5cf6; --ease-highlight-primary-color: #ffffff; }
