ease-text-overflow

Text overflow, break, and whitespace utilities

Truncation

.ease-truncate
This is a very long text that should be truncated with an ellipsis because it overflows the container.
.ease-text-clip
This is a very long text that will be clipped abruptly with no ellipsis when it overflows.

Word Break

.ease-break-word
Supercalifragilisticexpialidocious breaks gracefully.
.ease-break-all
Supercalifragilisticexpialidocious breaks at any character.

Whitespace

.ease-whitespace-nowrap
Never breaks to a new line no matter how long
.ease-whitespace-pre
Preserves spaces and line breaks.
.ease-whitespace-pre-wrap
Preserves spaces and wraps when needed.
.ease-whitespace-normal
Collapses spaces and wraps normally.
