# ease-flip-card — User Profile Card with Flip Animation

## What does this do?

A React component for a user profile card with a 3D flip animation on hover — front shows avatar/name/role, back shows bio and social links.

## How is it used?

```jsx
import ProfileFlipCard from './ProfileFlipCard';

<ProfileFlipCard name="Alex" role="Designer" bio="Creative problem solver." />
```
