# Shimmer Skeleton Loader

## What does it do?

The Shimmer Skeleton Loader shows animated placeholders that move smoothly to give a shimmering effect.It helps users see that content is loading by showing a visual of what's coming, so they know something is happening.


### This demo includes multiple skeleton variants:

- Avatar Skeleton
- Text Skeleton
- Card Skeleton
- Button Skeleton

## How is it used?

Add the appropriate skeleton classes to placeholder elements while content is loading.

Example:

<div class="skeleton skeleton-line"></div>


Once the actual content is available, replace the skeleton placeholders with the real content.

## Why is it useful?

Skeleton loaders help make the user experience better by making the waiting time feel shorter and showing something while the system is working in the background.

Benefits include:

- Smooth shimmer animation using CSS
- No JavaScript dependencies
- Responsive design
- Dark mode support
- Multiple loading placeholder variants which are great for modern dashboards, cards, forms, profiles, and websites with a lot of content.
