# Animated Toggle Switch

1. What does this do? Provides a pure-CSS animated toggle switch with smooth knob sliding, color transitions, multiple size variants, and optional icon indicators.
2. How is it used? Apply `.toggle` to a `<label>` wrapping a hidden `<input type="checkbox">`, a `.track` with a `.knob` child, and an optional `.label-text` span. Use `.toggle-sm` or `.toggle-lg` for size variants.
3. Why is it useful? Toggle switches are a common UI pattern for settings, feature flags, and preference toggles. This component follows EaseMotion CSS's utility-first, accessibility-conscious philosophy — zero JavaScript, focus-visible outlines, reduced-motion support, and dark mode compatibility built in.
