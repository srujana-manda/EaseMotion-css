ease-aspect-*

Aspect ratio utility classes — maintain consistent media proportions with a single class. Includes responsive variants.

Built-in Aspect Ratios

Each box below is set to width: 100% with its aspect ratio class. The height is derived automatically.

Square1:1
.ease-aspect-square
Video16:9
.ease-aspect-video
Portrait3:4
.ease-aspect-portrait
Photo4:3
.ease-aspect-photo
Wide16:10
.ease-aspect-wide
Ultrawide21:9
.ease-aspect-ultrawide

Auto (Natural)

The .ease-aspect-auto class resets to the element's natural aspect ratio.

Auto200x100 (no constraint)
.ease-aspect-auto

Custom Aspect Ratio

Use .ease-aspect-custom with the --ease-aspect-ratio variable for arbitrary values.

2:1--ease-aspect-ratio: 2/1
.ease-aspect-custom
5:3--ease-aspect-ratio: 5/3
.ease-aspect-custom
1:2--ease-aspect-ratio: 1/2
.ease-aspect-custom
