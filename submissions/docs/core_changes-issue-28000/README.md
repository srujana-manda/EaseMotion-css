# ease-caret — Caret & Cursor Utility Classes

## What does this do?

Controls the text input caret (`caret-color`) and mouse cursor (`cursor`) appearance.

## How is it used?

```html
<input class="ease-caret-primary" placeholder="Colored caret" />
<div class="ease-cursor-pointer">Clickable</div>
<div class="ease-cursor-wait">Loading...</div>
```
