How the mobile menu works: The hidden .ease-navbar-checkbox sits between the brand/toggle and the links. When checked (via the toggle label), :checked ~ .ease-navbar-links switches from display: none to display: flex with an easeSlideDown animation. The hamburger icon animates to an X using rotate() transforms on the three spans. On desktop (>768px), the toggle is hidden and links are shown inline.
