ease-search-bar

An animated search bar component with smooth expand-on-focus, 5 style variants, clear button, and responsive behavior.

Default

ease-search-bar (basic)

The default variant. Hover or focus to expand. Type to reveal the clear button.

.ease-search-bar
Variants

Filled, Outline, Rounded & Glass

Five predefined styles via modifier classes.

.ease-search-filled
.ease-search-outline
.ease-search-rounded
.ease-search-glass
Usage

HTML Structure

<div class="ease-search-bar">
  <span class="ease-search-icon">
    <!-- SVG magnifier icon -->
  </span>
  <input class="ease-search-input"
         type="text"
         placeholder="Search..." />
  <button class="ease-search-clear">&times;</button>
</div>

<!-- Variants -->
<div class="ease-search-bar ease-search-filled">...</div>
<div class="ease-search-bar ease-search-outline">...</div>
<div class="ease-search-bar ease-search-rounded">...</div>
<div class="ease-search-bar ease-search-glass">...</div>
