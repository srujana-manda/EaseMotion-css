Tooltip Viewport Overflow Fix

Tooltips no longer overflow the viewport on any edge. max-width, overflow-wrap, and edge-aware positioning keep tooltips visible even on the far right or left.

What was fixed

  • Added max-width: min(300px, 90vw) to prevent overflow beyond viewport
  • Added overflow-wrap: break-word + word-break: break-word for long text
  • Right-positioned tooltips near right edge constrain to calc(100vw - 100% - 2rem)
  • Left-positioned tooltips near left edge constrain to calc(100% - 2rem)
  • Mobile viewport uses calc(100vw - 2rem) for maximum space
