Modal Focus Trap

Accessibility enhancements for the EaseMotion modal component — keyboard focus trapping, aria-modal, role="dialog", and Escape-to-close.

Accessibility Features

🎯
Focus Trap
Tab/Shift+Tab cycles inside modal
⌨️
Escape Key
Closes modal, returns focus
🏷️
ARIA Attributes
aria-modal="true" + role="dialog"
🔙
Focus Return
Returns to trigger element on close

Demo

Click the buttons below to open modals. While a modal is open, Tab and Shift+Tab cycle through focusable elements inside the modal only — focus never escapes behind the overlay. Press Escape or click the backdrop to close.

Open Modal 1 Open Modal 2 Modal with Form

How to test

Open a modal above, then press Tab repeatedly. Focus will wrap through all focusable elements inside the modal. Press Shift + Tab to go backwards. Press Escape to close. After closing, focus returns to the button that opened the modal.

Keyboard Shortcuts

×

The following keyboard shortcuts are available when a modal is open:

  • Tab — Move to next focusable element
  • Shift + Tab — Move to previous focusable element
  • Escape — Close the modal
