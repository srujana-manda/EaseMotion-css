ease-clock-loader

A minimalist analog clock loader with rotating hands to indicate waiting or processing.

Sizes

Small, Default & Large

Three predefined sizes via modifier classes — .ease-clock-loader-sm, default, and .ease-clock-loader-lg.

.ease-clock-loader-sm
.ease-clock-loader
.ease-clock-loader-lg
Controls

Pause & Resume

Use the buttons below to pause or resume the clock animation — useful for demonstrating reduced-motion fallback behaviour.

.ease-clock-loader (animating)
Usage

Markup

Wrap three hand elements inside .ease-clock-loader. The loader works anywhere — buttons, loading overlays, inline indicators.

<div class="ease-clock-loader">
  <div class="ease-clock-hand ease-clock-second"></div>
  <div class="ease-clock-hand ease-clock-minute"></div>
  <div class="ease-clock-hand ease-clock-hour"></div>
</div>

Accessibility

prefers-reduced-motion

The clock loader respects the prefers-reduced-motion media query — hands stop rotating when the user requests reduced motion. Wrap the loader in a container with aria-busy="true" and add aria-label="Loading" for screen-reader users.
