Custom Offset Demo

Use --ease-sticky-offset to change when the element becomes sticky. The header below sticks 20px from the top.

OFFSET Sticky with 20px offset (--ease-sticky-offset: 20px)
Notice the gap above this header when it sticks.
Use --ease-sticky-offset to control the sticky position.
Also works with ease-sticky-bottom.

Usage

<!-- Sticky top header --> <div class="ease-sticky-top"> Sticky Header </div> /* Customize offset and z-index */ .my-sticky { --ease-sticky-offset: 1rem; --ease-sticky-z: 100; }
