fade-slide-combo
Page Entrance Mixin

A single SCSS mixin that pairs opacity fade with directional slide for smooth, accessible page entrance animations. Supports all four directions, distance control, spring easing, and staggered children.

@include fade-slide-combo(up) View SCSS Source

Usage

SCSS Mixin API

_fade-slide-combo.scss

// 1. Basic usage
.hero-title { @include fade-slide-combo(up); }

// 2. All parameters
.section-heading {
  @include fade-slide-combo(
    $direction: left,
    $distance: 32px,
    $duration: 500ms,
    $delay: 150ms,
    $scale: true
  );
}

// 3. Shortcut mixins
.card { @include fsc-up($delay: 200ms); }
.sidebar { @include fsc-right($duration: 600ms); }

// 4. Stagger children
.list > li {
  @include fsc-up;
  @include fsc-stagger($count: 6, $step: 100ms);
}
