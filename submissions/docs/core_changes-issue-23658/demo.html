Button Loading / Spinner State

Add .ease-btn-loading to any .ease-btn to show a CSS-only spinner, hide the label, and disable interaction — works with all variants and sizes.

All Variants

Each variant supports the .ease-btn-loading modifier.

.ease-btn-loading

Sizes

Works with .ease-btn-sm, default, .ease-btn-lg, and .ease-btn-xl.

.ease-btn-sm .ease-btn-lg .ease-btn-xl

Pill & Block

Loading state is fully compatible with .ease-btn-pill and .ease-btn-block.

.ease-btn-pill .ease-btn-block

Click a button to simulate an async action — it shows the loading state for 2 seconds, then returns to normal.
