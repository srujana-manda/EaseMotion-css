Usage

Add the animation class and .ease-animated for standard duration. Options include .ease-animated-fast, .ease-animated-slow, and delay classes.

<!-- Basic usage -->
<h1 class="ease-animated ease-bounce-in">Hello World</h1>

<!-- Custom duration & delay -->
<div class="ease-animated ease-animated-slow ease-light-speed-in ease-animated-delay-2">
  Staggered entrance
</div>

<!-- Multiple elements with cascade -->
<div class="ease-animated ease-bounce-in-up ease-animated-delay-1">Item 1</div>
<div class="ease-animated ease-bounce-in-up ease-animated-delay-2">Item 2</div>
<div class="ease-animated ease-bounce-in-up ease-animated-delay-3">Item 3</div>
