Quick Install & Usage

Three import styles — full, partial, or utilities-only

// 1. Full package
@use 'easemotion-scss' as em;
.hero { @include em.ease-fade-up($delay: 200ms); }

// 2. Tokens only (zero CSS output)
@use 'easemotion-scss/variables' as v;
.btn { transition: background v.$ease-duration-base v.$ease-snap; }

// 3. Utility classes only
@use 'easemotion-scss/utilities';
