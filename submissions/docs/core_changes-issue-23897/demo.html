ease-resize

Element resize utility classes. Control how elements can be resized by users with both, horizontal, vertical, and none variants. Includes custom resize handle styling and responsive breakpoints.

Both Directions

Drag the bottom-right corner to resize in any direction.

.ease-resize
This element can be resized in both directions. Grab the handle at the bottom-right corner and drag.

Horizontal Only

Resize width only. Height stays fixed.

.ease-resize-x
Drag horizontally to change the width of this element. The height remains fixed.

Vertical Only

Resize height only. Width stays fixed.

.ease-resize-y
Drag vertically to change the height of this element. The width remains fixed.

All Variants

Explicit both and logical property variants.

.ease-resize-both
Explicit resize: both. Same as .ease-resize.
.ease-resize-inline
Logical inline direction resize.
.ease-resize-block
Logical block direction resize.

None (No Resize)

Resize is disabled. Useful to override a parent resize class.

.ease-resize-none
This element cannot be resized. The handle is hidden.

Custom Handle

Styled resizer handle with ::-webkit-resizer or a CSS ::after pseudo-element.

.ease-resize-handle
Custom styled resizer via ::-webkit-resizer with neutral colors.
.ease-resize-handle-custom
Pseudo-element handle with a corner bracket indicator using ::after.

Usage

<!-- Resize both directions -->
<div class="ease-resize" style="width:200px;height:100px;">
  Resizable content...
</div>

<!-- Resize horizontal only -->
<div class="ease-resize-x" style="width:200px;height:100px;">
  Width adjustable...
</div>

<!-- Disable resize -->
<div class="ease-resize-none">
  Fixed size...
</div>

<!-- Custom handle -->
<div class="ease-resize-handle-custom" style="width:200px;height:100px;">
  Custom styled resizer...
</div>
