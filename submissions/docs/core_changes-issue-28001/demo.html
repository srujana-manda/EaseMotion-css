EaseMotion CSS is a utility-first CSS framework designed for motion and interaction. It provides low-level utility classes for building custom designs with minimal CSS.

Built with modern CSS features like container queries, cascade layers, and CSS custom properties. It's lightweight, flexible, and extensible.

Each utility class does one thing well and can be composed to create complex designs. Perfect for rapid prototyping and production use.
