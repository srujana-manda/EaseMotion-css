Component

Accordion

CSS-only collapsible content sections using <details> / <summary>. Zero JavaScript required.

Default

Basic accordion with .ease-accordion and .ease-accordion-item.

What is EaseMotion CSS? ▼
EaseMotion CSS is a lightweight, animation-first CSS framework with reusable UI components, modern effects, and zero dependencies. It is designed to be beginner-friendly and open-source.
How do I install it? ▼
You can install via npm (npm install easemotion-css) or use the CDN: https://cdn.jsdelivr.net/npm/easemotion-css@latest/easemotion.min.css
Is it accessible? ▼
Yes! Since it uses semantic HTML (<details> and <summary>), it is natively keyboard-accessible and works with screen readers out of the box. 
<div class="ease-accordion">
  <details class="ease-accordion-item" open>
    <summary class="ease-accordion-trigger">
      Question
      <span class="ease-accordion-icon">▼</span>
    </summary>
    <div class="ease-accordion-content">
      Answer text here.
    </div>
  </details>
</div>
