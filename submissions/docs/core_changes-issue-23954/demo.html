ease-text-balance

Text wrapping utilities: balance, pretty, and stable for improved typography.

.ease-text-balance — evenly distributes lines

This heading text is automatically balanced across multiple lines for a more even appearance.

.ease-text-pretty — avoids orphans

This paragraph uses text-wrap: pretty to prevent single words from appearing on their own line at the end of a paragraph. It's particularly useful for body text and articles where orphans look awkward and disrupt the reading flow.

.ease-text-stable — no reflow on dynamic changes

Content with text-wrap: stable won't cause reflow when content changes dynamically. This is ideal for updating content without shifting the layout unexpectedly around the reader.

class="ease-text-balance"  /* headings */
class="ease-text-pretty"   /* body text */
class="ease-text-stable"   /* dynamic content */
