ease-skeleton

CSS-only loading placeholder component with shimmer animation, multiple shapes, color themes, speed control, and reduced-motion support.

Features

10+ Preset Shapes

Text lines, headings, avatars, buttons, chips, thumbnails, cards, images, circles, pills.

6 Color Themes

Default, primary, secondary, success, danger, warning, dark.

4 Animation Modes

Shimmer (default), pulse, wave, fade. Plus static/no-animation.

Speed Control

Normal, slow (3s), fast (0.8s) via CSS custom properties.

Layout Wrappers

Row, column, card, card-row layout presets.

Accessible

Respects prefers-reduced-motion, aria-hidden ready.

Usage

<!-- Basic skeleton -->
<div class="ease-skeleton ease-skeleton--text"></div>

<!-- Shape presets -->
<div class="ease-skeleton ease-skeleton--avatar"></div>
<div class="ease-skeleton ease-skeleton--button"></div>
<div class="ease-skeleton ease-skeleton--card"></div>
<div class="ease-skeleton ease-skeleton--image"></div>

<!-- Animation modes -->
<div class="ease-skeleton ease-skeleton--pulse"></div>
<div class="ease-skeleton ease-skeleton--wave"></div>
<div class="ease-skeleton ease-skeleton--fade"></div>

<!-- Speed -->
<div class="ease-skeleton ease-skeleton--fast"></div>
<div class="ease-skeleton ease-skeleton--slow"></div>

<!-- Color themes -->
<div class="ease-skeleton ease-skeleton--primary"></div>
<div class="ease-skeleton ease-skeleton--dark"></div>

<!-- Layout wrappers -->
<div class="ease-skeleton__wrapper--card">
  <div class="ease-skeleton ease-skeleton--image"></div>
  <div class="ease-skeleton ease-skeleton--heading"></div>
  <div class="ease-skeleton ease-skeleton--text"></div>
</div>
