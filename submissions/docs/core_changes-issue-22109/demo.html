Object-Fit & Object-Position

Utility classes for controlling how images and videos fill their containers. No distortion, no manual CSS — just a single class.

Object-Fit

How Each Value Behaves

All images below use the same source image in a 4:3 container. The class name tells the browser how to fit it.

.ease-object-contain
Fits inside with letterboxing
.ease-object-cover
Fills container, crops excess
.ease-object-fill
Stretches to fit (may distort)
.ease-object-none
Original size, no scaling
.ease-object-scale-down
Smallest of none vs contain
