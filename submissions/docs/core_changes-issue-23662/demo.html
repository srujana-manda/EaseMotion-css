Vitest Animation Keyframes Unit Tests

This folder contains unit tests verifying that all keyframes (e.g. ease-kf-fade-in, ease-kf-bounce, etc.) are correctly structured and defined in the final CSS sheet using Vitest and jsdom.

Tests included in keyframes.test.js:

  • ✓ should find all core keyframe rules in the stylesheet
  • ✓ should verify the correct structure of ease-kf-fade-in keyframe rule
  • ✓ should verify structure of ease-kf-bounce keyframe rule
  • ✓ should verify ease-kf-bounce-in entry structure
  • ✓ should check if element animated classes trigger the keyframe animations

To run the test suite locally, copy the test file to the tests directory and execute the test script:

cp keyframes.test.js ../../tests/keyframes.test.js
npm run test
