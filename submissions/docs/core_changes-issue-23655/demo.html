Keyboard Instructions: Use Tab to focus on the active tab header. Use the Left/Right Arrow keys (or Up/Down Arrow keys) to move focus between tabs. Press Enter or Space to activate the focused tab. Use Home to focus the first tab, and End to focus the last tab.
