ease-pointer-events

Utility classes for pointer-events, touch-action, and combined interaction control. Includes SVG pointer-events values, touch-action presets, and media query variants.

Features

14 Pointer Events Values

All CSS pointer-events values including SVG-specific (visible, painted, fill, stroke, bounding-box).

10 Touch Action Presets

Auto, none, manipulation, pan-x, pan-y, pan-left, pan-right, pan-up, pan-down, pinch-zoom.

Combined Utilities

ease-clickable, ease-unclickable, ease-no-interaction for common patterns.

Media Query Variants

Hover-aware, touch-aware, and reduced-motion-aware pointer-events classes.

Usage

<!-- Disable interaction on an overlay -->
<div class="ease-pointer-events-none">...</div>

<!-- Re-enable for child elements -->
<div class="ease-pointer-events-none">
  <button class="ease-pointer-events-auto">Clickable</button>
</div>

<!-- Touch optimization -->
<div class="ease-touch-action-manipulation">
  Touch-friendly scrollable area
</div>

<!-- Combined shortcuts -->
<div class="ease-clickable">Clicks show pointer</div>
<div class="ease-unclickable">Looks disabled</div>
