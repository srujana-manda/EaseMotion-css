ease-cursor

Cursor utility classes covering all standard CSS cursor values plus hover, touch, and mouse media query variants.

Features

36 Cursor Values

All CSS cursor keywords including directional resize cursors, zoom, grab, and aliases.

Hover Variants

Cursor changes only when hover is supported via @media (hover: hover).

Pointer Variants

Separate classes for coarse (touch) and fine (mouse) pointer devices.

Reduced Motion

Smooth cursor transition only when user hasn't requested reduced motion.

Usage

<!-- Basic cursor -->
<button class="ease-cursor-pointer">Click me</button>
<div class="ease-cursor-grab">Drag me</div>
<input class="ease-cursor-text" readonly value="Read only">

<!-- Disabled states -->
<button class="ease-cursor-not-allowed" disabled>Disabled</button>

<!-- Resize handles -->
<div class="ease-cursor-ew-resize" style="width:200px;height:20px;background:#e2e8f0;"></div>
<div class="ease-cursor-ns-resize" style="width:20px;height:100px;background:#e2e8f0;"></div>

<!-- Interactive -->
<div class="ease-cursor-zoom-in" onclick="zoom()">Zoom area</div>
<div class="ease-cursor-move">Draggable element</div>

<!-- Hide cursor -->
<div class="ease-cursor-none">No cursor here</div>
