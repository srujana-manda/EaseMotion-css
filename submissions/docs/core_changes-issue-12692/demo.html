ease-progress

CSS-only progress bar component with determinate, indeterminate, striped, animated, and color variants.

Determinate (Default)

Set width via inline style on the bar element.

Progress 45%
.ease-progress

Sizes

Small (sm)
.ease-progress-sm
Medium (md, default)
.ease-progress (md)
Large (lg)
.ease-progress-lg
Extra Large (xl)
.ease-progress-xl

Colors

.ease-progress-primary
.ease-progress-success
.ease-progress-danger
.ease-progress-warning
.ease-progress-info

Indeterminate

For unknown or loading progress.

.ease-progress-indeterminate

Striped

.ease-progress-striped
.ease-progress-striped.ease-progress-animated

Usage

<!-- Determinate -->
<div class="ease-progress">
  <div class="ease-progress-bar" style="width: 45%;"></div>
</div>

<!-- Indeterminate -->
<div class="ease-progress ease-progress-indeterminate">
  <div class="ease-progress-bar"></div>
</div>
