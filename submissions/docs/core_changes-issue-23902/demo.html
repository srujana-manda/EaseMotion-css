ease-pagination

CSS-only pagination component with page numbers, active/disabled states, ellipsis, sizes, colors, alignment, layout variations, and responsive visibility.

Features

6 Color Themes

Primary, secondary, success, danger, warning, dark mode.

3 Sizes

sm, default, lg.

4 Alignments

Start (default), centered, end, space-between.

Layout Variants

Default, compact (segmented), stretched (equal width), no-border, outline active.

Responsive

Responsive mode hides inner pages on small screens.

Accessible

Focus-visible ring, aria-label ready, nav landmark.

Basic

Default pagination with page numbers, prev/next, and ellipsis.

Usage

<nav class="ease-pagination" aria-label="Page navigation">
  <a class="ease-pagination__item ease-pagination__item--disabled" href="#">&laquo; Prev</a>
  <a class="ease-pagination__item ease-pagination__item--active" href="#">1</a>
  <a class="ease-pagination__item" href="#">2</a>
  <a class="ease-pagination__item" href="#">3</a>
  <span class="ease-pagination__item ease-pagination__item--ellipsis">&hellip;</span>
  <a class="ease-pagination__item" href="#">10</a>
  <a class="ease-pagination__item" href="#">Next &raquo;</a>
</nav>

<!-- Alignments -->
<nav class="ease-pagination ease-pagination--centered">...</nav>
<nav class="ease-pagination ease-pagination--end">...</nav>

<!-- Sizes -->
<nav class="ease-pagination ease-pagination--sm">...</nav>
<nav class="ease-pagination ease-pagination--lg">...</nav>

<!-- Colors -->
<nav class="ease-pagination ease-pagination--success">...</nav>

<!-- Layouts -->
<nav class="ease-pagination ease-pagination--compact">...</nav>
<nav class="ease-pagination ease-pagination--stretched">...</nav>
<nav class="ease-pagination ease-pagination--no-border">...</nav>
