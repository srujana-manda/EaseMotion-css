# KPI Dashboard Widget

1. What does this do? A modern KPI dashboard widget with compact statistic cards supporting icons, animated counters, trend indicators (up/down), percentage changes, glassmorphism variant, dark mode variant, and responsive grid layouts.

2. How is it used? Add the HTML markup to your page and link `style.css`. Each KPI card uses classes like `.kpi-card`, `.kpi-icon`, `.kpi-value`, `.kpi-label`, `.kpi-trend`, with optional `.glass` and `.dark` variant classes.

3. Why is it useful? KPI widgets are essential for admin panels, SaaS dashboards, analytics platforms, finance applications, and productivity tools. This component demonstrates animated counters, trend indicators, multiple visual variants, and responsive layouts that align with EaseMotion CSS's animation-first and composable design philosophy.
