Scroll-to-Top Button

Scroll down past 300px to see the Floating Action Button (FAB) appear. Click it to smoothly return to the top. Built with .ease-fab and .ease-hover-grow / .ease-hover-rotate.

Why scroll-to-top?

On long pages — documentation, articles, product listings — users often need to quickly return to the top for navigation. A scroll-to-top FAB provides this without manual scrolling, improving UX and accessibility.

How it works

The .ease-fab is a fixed-position button anchored to the bottom-right. A scroll listener toggles the .visible class when window.scrollY > 300. When clicked, window.scrollTo({ top: 0, behavior: 'smooth' }) scrolls the page up.

Variants

Base: .ease-fab — fixed position, 56×56px circle, shadow, hidden by default. Theme: .ease-fab-primary (indigo gradient) and .ease-fab-secondary (glass). Hover: .ease-hover-grow (scale 1.12) and .ease-hover-rotate (360° spin).
