ease-glass & ease-neumorph

Modern visual effect utilities — glassmorphism (frosted glass) and neumorphism (soft 3D shadows) — applied with a single class. Dark mode support built in.

Glassmorphism

.ease-glass applies a frosted glass effect with backdrop-filter: blur(). Add .ease-glass-sm or .ease-glass-lg for different blur intensities.

Default

blur(12px) — standard frosted glass

Small

blur(6px) — subtle blur

Large

blur(24px) — heavy blur

.ease-glass .ease-glass-sm .ease-glass-lg

Neumorphism

.ease-neumorph for soft raised cards, .ease-neumorph-inset for the pressed/inset variant.

Raised

.ease-neumorph — soft outer shadows creating a 3D raised effect

Inset

.ease-neumorph-inset — inner shadows simulating a pressed state

.ease-neumorph .ease-neumorph-inset

Dark Mode

The utilities include @media (prefers-color-scheme: dark) overrides for automatic dark mode support. Toggle your system theme to see the difference.

Glass

Dark background, lighter border

Neumorph

Dark-adapted shadows

Inset

Dark-adapted inset
