# ease-field-sizing

## What does this do?
CSS `field-sizing` utility classes. `ease-field-sizing-content` makes form fields auto-grow with content. `ease-field-sizing-fixed` keeps them at a fixed scrollable size.

## How is it used?
```html
<textarea class="ease-field-sizing-content" rows="1">Auto-growing textarea</textarea>
<input class="ease-field-sizing-content" value="Auto-sizing input">
```

## Why is it useful?
Eliminates the need for JavaScript-based auto-resize textareas. Pure CSS solution using the modern `field-sizing` property.
