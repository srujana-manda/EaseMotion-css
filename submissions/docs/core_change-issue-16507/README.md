## Description

### Border Draw Animation Effect on Hover

A clean and modern hover effect where borders animate and draw around the card/button when the user hovers over it. Built using only HTML and CSS.

## Features

* Smooth border drawing animation
* Pure HTML & CSS
* Lightweight and responsive
* Easy to customize
* Beginner-friendly implementation

## Preview

Hover over the card to see the border animate around the element.

## Technologies Used

* HTML5
* CSS3

## Customization

You can easily modify:

* Border color
* Animation speed
* Card dimensions
* Background colors
* Border thickness

## Learning Concepts

* CSS Pseudo Elements (`::before`, `::after`)
* CSS Transitions
* Positioning
* Hover Effects
* Transform Properties
