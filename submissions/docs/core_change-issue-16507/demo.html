            # Border Draw Animation Effect

            A clean and modern hover effect where borders animate
            and draw around the card when hovered.

            ## Features

            - Smooth border drawing animation
            - Pure HTML & CSS
            - Responsive design
            - Easy customization
            - Beginner friendly

            ## Technologies Used

            - HTML5
            - CSS3

            ## Learning Concepts

            - CSS Pseudo Elements
            - CSS Transitions
            - Positioning
            - Hover Effects
            - Transform Properties

            ## Customization

            You can modify:
            - Border color
            - Animation speed
            - Card size
            - Border thickness
