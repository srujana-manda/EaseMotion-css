# Serenity Yoga Studio Landing Page

1. **What does this do?**  
   A calm, serene full-page business landing page for a yoga studio, built entirely with EaseMotion CSS utility classes and component classes — no custom keyframes.

2. **How is it used?**  
   Open `demo.html` directly in any browser. The page links to EaseMotion CSS via CDN and uses only existing `ease-*` classes for all animations, layout, and interactions.

3. **Why is it useful?**  
   Demonstrates how EaseMotion CSS can build complete, production-quality business landing pages with zero custom animations and no build step — just utility classes and components.

## Sections

| Section | Description |
|---------|-------------|
| Hero | Full-viewport hero with gradient background, breathing circle animation (`ease-float`), entrance animations (`ease-slide-up`, `ease-delay-*`) |
| Classes | 6 class type cards with icon, description, level badge, and `ease-card-lift` hover |
| Instructors | 3 instructor profile cards with avatar, credentials, and `ease-card-lift` on hover |
| Schedule | Weekly class timetable with hover row highlighting |
| Pricing | 3-tier membership cards with "Most Popular" featured plan, `ease-btn-primary`, `ease-hover-glow` |
| Testimonials | 3 student review cards with star ratings |
| Booking CTA | Full-width gradient call-to-action with `ease-pulse` animation on primary button |
| Footer | 3-column footer with brand info, quick links, and contact details |
