# Skeleton / Placeholder Component — Issue #30378

CSS-only skeleton loading placeholders with shimmer animation.

## Classes

| Class | Purpose |
|-------|---------|
| `.ease-skeleton` | Base skeleton (requires shape modifier) |
| `.ease-skeleton-text` | Text line skeleton |
| `.ease-skeleton-text-sm` | Smaller text line |
| `.ease-skeleton-text-lg` | Larger text line |
| `.ease-skeleton-circle` | Circular skeleton |
| `.ease-skeleton-circle-sm/md/lg` | Avatar sizes (2rem/2.5rem/3.5rem) |
| `.ease-skeleton-rect` | Rectangular skeleton |
| `.ease-skeleton-rect-sm/md/lg/xl` | Rectangle heights (4rem/8rem/12rem/16rem) |
| `.ease-skeleton-inline` | Inline display variant |

## CSS Variables

- `--ease-skeleton-base` — Shimmer base color
- `--ease-skeleton-shine` — Shimmer highlight
- `--ease-skeleton-radius` — Border radius
- `--ease-skeleton-duration` — Animation speed
