Submission guide

Keep CSS demos self-contained

A standard EaseMotion CSS submission should include only the required demo, stylesheet, and README files.

Required demo.html

Shows the effect with semantic, local markup.

Required style.css

Contains the raw CSS for the animation or component idea.

Required README.md

Explains what the submission does, how to use it, and why it fits.

Only if requested script.js

Leave JavaScript out unless a maintainer explicitly asks for it.
